　　第426章
<br/>　　赵子龙抵达牵牛谷所需的时间，比程大雷事先预估的更早。事实上，在第二天早晨，赵子龙便已接近牵牛谷。
<br/>　　时间整整提前了一半。
<br/>　　并不是程大雷之前的估算出了什么问题，而是他忽略了一件事。在将山寨升级为初级城池后，蛤蟆城获得了『神行』技能，行军速度大幅度提升。
<br/>　　因此，预估黄昏才能抵达战场，却在天光微亮时已靠近牵牛谷。
<br/>　　从此地望过去，牵牛谷便在眼前。龙字军在赵子龙身后整齐停下，等待赵子龙的命令。
<br/>　　赵子龙是第一个到的，其他如秦蛮、关鱼虽然气得火起，但两条腿终究跑不过四条腿，现在还在来的路上。
<br/>　　据阿喜汇报的情况，宋伯康提了十万兵去长安，目前留守在凉州城的兵力，大概在五万左右。攻击高飞虎，自然不会五万人倾巢而动，敌人兵力大概在两到三万左右。
<br/>　　宋伯康和戴白离开后，主持军政大局的人是陆洛。程大雷当然做过关于他的背景调查，其人出身江南陆家，是个文职官员，至于领兵打仗，没听说他有这方面才能。
<br/>　　赵子龙以眼神示意程大雷，看程大雷的意思，这场仗该如何打。
<br/>　　程大雷向前伸伸手：“您请。”
<br/>　　既然将龙字军交给赵子龙，程大雷便准备好放手不管，总不能自己一直亲力亲为，扶着他们走路。且也借着这次事情看看，赵子龙的指挥能力如何。
<br/>　　赵子龙轻轻点头，示意自己明白。然后，他抬起手遥指牵牛谷方向，口中吐出两个字。
<br/>　　“进攻！”
<br/>　　三千龙字军同时发动，向山谷进发，他们安静时如石柱，但在赵子龙下达命令后，又同时像强弓射出的弩箭。
<br/>　　这一幕，正合程大雷的心意。
<br/>　　天下武功，唯快不破。这句话放在战场上也是一样，以有备攻无备，以有算打无算，本就是越快越好。龙字军在『神行』技能的作用下，极短时间赶到战场，本就是发动突袭的最好时机，稍一耽搁便可能贻误战机。
<br/>　　程大雷这次只负责厮杀，其他事情不管。当赵子龙下令后，他便和樊梨花同时冲杀了出去。
<br/>　　他，赵子龙和樊梨花算是冲在队伍最前方的三人，也算是最早和凉州兵接触的三人。
<br/>　　程大雷一斧炸开营寨前的木栅栏，领着大兵杀了进去。
<br/>　　黎明是人每天中最疲惫的时刻，凉州兵正在帐篷里睡得踏实，忽然就听到外面的喊杀声，等他们冲出来时，看到微薄的天光中，一匹黑牛横冲直撞，在营寨里大杀特杀。
<br/>　　如果不是天快要亮了，诸人能看清楚牛背上还托着一个人，怕还真会把程大雷当做某种从山里窜出来的怪物。
<br/>　　高飞虎被堵在山谷中，被堵得分外憋屈。愁眉紧皱，心中七上八下。虽然心中相信程大雷不会看着自己出事不管，但中间毕竟隔着三四年时间，物是人非，谁知道有些事情是不是确定的。
<br/>　　这个时候，山谷外喊杀声突起。有喽啰向高飞虎汇报，高飞虎本也没有睡着，大步走出帐篷，看到山谷外厮杀声一片。
<br/>　　“什么情况，这是什么情况！”
<br/>　　“老大，好像有人来劫凉州兵的大营，只是不知道是什么人。”齐少建。
<br/>　　高飞虎心中一动，忽然大叫道：“还能是什么人，肯定是大当家，大当家来救我们了。”
<br/>　　高飞虎立刻停止了腰板，愁云散去，整个人扬眉吐气。
<br/>　　“快集结兄弟！让我们配合大当家打出去！”
<br/>　　凉州兵大营，陆洛此刻也是一头乱麻。夜里的确安排了哨岗，但主要是防备高飞虎在夜里突围。陆洛千算万算，没算到程大雷的兵会杀过来。关键是，他们赶来的速度确实出乎陆洛预料。
<br/>　　三千人闯入大营，杀人放火，凉州兵被打了个措手不及，一时无法组织起防御力量，三千人如入无人之境。
<br/>　　陆洛急得焦头烂额，抓紧时间阻止力量反击，宋伯康肯将他留下来守城，此人便不会是酒囊饭袋。而凉州兵的反应也是不慢，在摸不清楚敌人底细的情况下，竟然组织起一股不小的力量防御。
<br/>　　赵子龙令手下人鸣锣，下达撤退命令。
<br/>　　如此一来，陆洛便见识到这支部队的不凡之处。在乱战的情况下，赵子龙一声令下，所有人便开始有条不紊的撤退。有人趟开道路，有人掩护，有人负责断后，三千人的撤退并没有浪费太多时间。
<br/>　　山谷内，高飞虎刚集结好人手，准备带队冲杀出来，却发现山谷外的战斗已经结束了。
<br/>　　他的人愣在哪里，却不知该如何是好，目光同时望向高飞虎。
<br/>　　“老大，他们不会发觉打不过逃了吧，现在咱们该怎么办？”齐少建。
<br/>　　“闭上你的鸟嘴。”
<br/>　　高飞虎气不打一处来，生气的主要原因，是也害怕齐少建说得是真的。毕竟，凉州兵不弱，人数又多，大当家打不过的情况下，选择弃卒保车也不是没有可能。
<br/>　　“咱们，先等等，看接下来怎么样？”高飞虎下达命令。
<br/>　　接下来陆洛主动派人出击，对方杀过来时令他心惊胆战，可一旦撤退时就露了底细。原来对方只有三千人，陆洛立刻派出人马追击。
<br/>　　两支骑兵队伍在荒原上展开追逐战，令陆洛意想不到的是，对方的逃跑速度很快。极短时间内，便甩了己方一大段路程，然后，对方突然转向，回过头杀了一个回马枪。
<br/>　　己方被一举攻破，溃不成军，然后丢盔弃甲的逃了回来。
<br/>　　陆洛恍然大悟，对方根本就是诱兵出击，然后一举击溃。之后，陆洛领略到敌军的狡诈卑劣。当机派出重兵出击，想要与对方展开一场硬碰硬的遭遇战时，对方立刻远遁，自己连敌人的马屁股都摸不到。
<br/>　　然而，当己方选择按兵不动时，他们又会冲过来骚扰，占些便宜就跑。
<br/>　　如此，这场战斗一直持续到黄昏，在黄昏时分，秦蛮、关鱼、张肥、高飞豹的步兵队伍同时到了。在兵力上，已能与敌人持平。
<br/>(本章完)
