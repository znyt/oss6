　　萧逸大手一挥，清除了周遭痕迹与火焰气息。
<br/>　　数天前，在六龙地域时。
<br/>　　虽然吸收了毒煞、血煞，还借了雷煞之力淬炼肉体。
<br/>　　但毒煞、血煞，仅仅是增强了他体内的血丹和毒丹，故他的修为并未增长分毫。
<br/>　　而淬炼肉体，虽让肉体达到绝世之境，但那只是等同普通体修的绝世之境，也算不了什么。
<br/>　　修罗战体，如今仅仅是修炼至圣皇境八重层次。
<br/>　　日后，得修罗战体也修炼至绝世层次时，肉体力量双重增幅下，才算是一次极大的的提升。
<br/>　　反倒是现今，修为臻至圣皇境巅峰。
<br/>　　体内气泉元力已满，甚至溢出些许至小世界。
<br/>　　这代表着，他将能爆发更多的元力，凝聚火焰时，威力也更加强横。
<br/>　　还有，便是离开六龙地域前，吸收的那十万邪修的武魂力量，让得控火兽的层次，直追冰鸾剑。
<br/>　　那时起，他的控火能力，又是一次飙升。
<br/>　　控火能力、元力，二者的增加，方才是他控火一道上实力的大幅度增加。
<br/>　　现在起，若他再遇到毒袍，将极其轻松便能将其斩杀。
<br/>　　他早便说过，毒袍没有机会了。
<br/>　　自毒袍屡败之后，毒袍便再无机会追上他，也将再也不是他的对手。
<br/>　　“呼。”萧逸深呼吸一口气，一阵神清气爽。
<br/>　　实力的提升，才是他更加自信，也更加底气十足的来源。
<br/>　　事实上，阻碍他修为提升的，仅仅是因为他那庞大得恐怖的小世界，所需的修炼资源太过恐怖。
<br/>　　拿这次的三枚乾坤戒奖励来说。
<br/>　　若换了别的天骄妖孽，怕是足以从圣皇境八重，一路修炼至绝世之境了。
<br/>　　当然，前提是这个天骄拥有足够的天赋，不必被掌控天地武道而阻拦。
<br/>　　而即便如此，这个天骄，起码在修炼至绝世层次时，都不必再为修炼资源而担心了。
<br/>　　难怪总殿主会说，那份奖励，足够让人眼馋。
<br/>　　可这份奖励，放到萧逸身上，仅仅是能让他提升两重修为。
<br/>　　而且气泉满了后，元力溢出小世界，也仅仅是溢出些许，并不太明显。
<br/>　　他的小世界和气泉之庞大，可见一斑。
<br/>　　不过，虽如此，萧逸丝毫不在乎。
<br/>　　他的修为提升是慢了，可他的实力，可是单凭元力就能远超同境界武者，甚至是轻松碾压之。
<br/>　　这是别的手段远远无法比及的。
<br/>　　而且，他的小世界与气泉足够庞大，他才能拥有足够多的元力去施展自己的手段。
<br/>　　拿他的九阳轮回诀来说，凝聚火焰圆环；还有，兽手、兽脚等增幅，还有破晓无极等等。
<br/>　　哪一项不是赋予了他越级战斗的能力，又哪一项不是需要耗费大量的元力？
<br/>　　若他的小世界不是足够庞大，元力足够多的话，根本支撑不了这些手段。
<br/>　　若他的元力不够多，恐怕在兽手、兽脚等增幅下，进入绝世以上的层次战斗时，战个几分钟便元力耗尽了。
<br/>　　反倒是小世界足够庞大下，元力足够多，方才让他能一直越级酣畅战斗。
<br/>　　故，小世界和气泉庞大，虽大大拖缓了他的修为提升，但这绝非坏事。
<br/>　　萧逸笑笑，敛下眼中喜意，化作平淡后，缓步走出了闭关室。
<br/>　　刚出闭关室。
<br/>　　闭关室外，一位殿内人员，已迎了上来。
<br/>　　“易霄副殿主，这么快就出关了？”殿内人员，脸上尽是惊色。
<br/>　　萧逸进入闭关室，怕是还不到数个时辰。
<br/>　　“有问题吗？”萧逸淡漠问道，“有事？”
<br/>　　“额，是。”殿内人员愣了愣，随后点点头，道，“三位总殿主有令，你出关后，便唤你去一趟。”
<br/>　　“走吧。”萧逸点点头。
<br/>　　在殿内人员的带领下，萧逸再次来到那间殿宇之外。
<br/>　　推门而入，三位总殿主，仍旧在殿宇之内商量着什么。
<br/>　　“嗯？”三位总殿主，看着萧逸的到来，脸上同时闪过一丝惊色。
<br/>　　“你不是去闭关了？”
<br/>　　“这么快便出关，可是有什么事？”
<br/>　　猎妖总殿主，看向萧逸，问道。
<br/>　　一旁殿内人员回答道，“易霄副殿主，已经出关了。”
<br/>　　“按三位总殿主之令，带易霄副殿主前来。”
<br/>　　“出关了？”猎妖殿总殿主一愣。
<br/>　　“嗯。”萧逸点点头。
<br/>　　“老夫给你的那些天材地宝，都炼制光了？”药尊殿总殿主，也愣了愣。
<br/>　　“炼光了。”萧逸再次点点头。
<br/>　　“老夫给你的那些天火玄精、火精脉，也吸收光了？”炎殿总殿主，急声问道。
<br/>　　“吸光了。”萧逸仍旧点点头。
<br/>　　“这…”三人一惊，随后对视了一眼。
<br/>　　“我去一趟。”猎妖殿总殿主皱眉说了一声，身影一闪，消失在原地。
<br/>　　殿宇之内，药尊殿总殿主对那殿内人员摆摆手，道，“你先下去吧。”
<br/>　　“是。”殿内人员行了一礼，转身离去。
<br/>　　殿宇之内，只剩两位总殿主，以及萧逸。
<br/>　　萧逸不语，眼眸淡漠地站着。
<br/>　　两位总殿主，同时凝视着萧逸。
<br/>　　“小子，如果让老夫知晓你说谎，呵呵。”炎殿总殿主冷笑一声，话虽未说完，但语气中明显带着不善的气息。
<br/>　　“唉。”药尊殿总殿主，看了眼炎殿总殿主，叹了口气。
<br/>　　目光，随后也看回萧逸。
<br/>　　“小子，你何必说这等谎话，你虽天赋惊人，但也不必急于求成，不必如此好大喜功。”
<br/>　　“我和猎妖殿总殿主，本就对你青睐有加，亦赞赏有加，你何必…唉。”
<br/>　　药尊殿总殿主，再次叹了口气。
<br/>　　炎殿总殿主，倒是冷笑一声，“那么大堆材料，修炼之物，没个数天时间，区区数个时辰便炼光了，吸光了？骗鬼去吧。”
<br/>　　“你看到了，这小子谎话连篇，这般心性叫好？”
<br/>　　“看来，之前也是这般大话连连，别有用心。”
<br/>　　“这般急功近利，邀功讨喜，啧啧。”
<br/>　　萧逸皱了皱眉，“在下不知道两位总殿主在说些什么。”
<br/>　　萧逸哪里知晓三位总殿主之前商量过什么。
<br/>　　自然的，现在听起来，莫名其妙。
<br/>　　而炎殿总殿主，显然给萧逸贴上了他之前便怀疑的‘谎话连篇’‘别有用心’的标签。
<br/>　　“想狡辩？”炎殿总殿主冷笑一声。
<br/>　　药尊殿总殿主摇了摇头，“小子，你太小看我们三位总殿主的能耐了。”
<br/>　　“即便不必搜你身，我们要知你是否说谎，其实很简单。”
<br/>　　萧逸皱了皱眉。
<br/>　　半晌，猎妖殿总殿主，闪身而回。
<br/>　　只是，脸色，一片凝重。
<br/>　　“如何了？”炎殿总殿主，问了一声。
<br/>　　药尊殿总殿主不语，但看向萧逸的目光，显然闪过一丝怜悯。
<br/>　　似乎在说，罪证确凿。
<br/>　　“你们自己看吧。”猎妖殿总殿主大手一挥，一个光幻画面，凭空而现。
<br/>　　画面之内，是猎妖殿总殿主，出现于闭关室中。
<br/>　　只见猎妖殿总殿主随意挥了挥手。
<br/>　　闭关室内，空间忽然扭曲，一个个画面，开始回溯。
<br/>　　“时间回溯？”萧逸大吃一惊。
<br/>　　他早便听闻，传奇以上强者，有回溯时间之手段。
<br/>　　且实力越强，手段越厉害，能回溯的时间便越多。
<br/>　　观猎妖殿总殿那般轻松的随意挥手，怕是他掌控此手段早已达到骇人听闻的地步。
<br/>　　短短数秒时间，画面，已回到数个时辰之前。
<br/>　　药尊殿总殿主、炎殿总殿主，明显注意到萧逸的惊色。
<br/>　　炎殿总殿主，再次冷笑一声。
<br/>　　药尊殿总殿主，则叹息地摇了摇头。
<br/>　　只是，待得那一个个画面快速涌动，两位总殿主，脸色，却渐渐惊骇了起来。
<br/>　　画面之内，萧逸从一开始炼丹，再到炼制出一座丹药小山，再到最后的吸收大堆火属性修炼资源…
<br/>　　“这小子没有说谎。”猎妖殿总殿主，凝重地说了一声。
<br/>　　“好快的炼药速度，这小子，几乎是在随意扔掷材料，瞬间成丹。”药尊殿总殿主，咽了口口水。
<br/>　　“这小子是变态吗？那么多的精纯火属性力量，随意吸摄，天地武道降临，顷刻掌控？”炎殿总殿主，瞪大了眼睛。
