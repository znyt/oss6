        这个男人的五官俊美凛冽，眉宇间总是带着一丝浑然天成的霸气，他的手骨节分明，既修长又好看。
<br/>
<br/>        落以柔有些迷恋地看着这个外表几近完美、内在气宇非凡的男人，突然发现自己好像对他多了一丝隐隐的怦然心动。
<br/>
<br/>        她轻轻地拿条被子覆在他的身上，才走进浴室梳洗，步至小阳台烹调早餐。
<br/>
<br/>        长这么大，她还是第一次为一个男人下厨。
<br/>
<br/>        不知道今后，她会不会有幸为他做一辈子的早餐。
<br/>
<br/>        啊啊啊！
<br/>
<br/>        她想哪去了！
<br/>
<br/>        落以柔连连摇头，回头看了一眼那个还在沉睡中的帅男人后，赧然一笑，然后继续做着手里的早餐。
<br/>
<br/>        一阵食物的香气惊醒沉睡中的傅夜沉，伟岸的身子轻轻挪动，被子滑落，他错愕地抬起，淡淡的香气盈满鼻间，那是一股柔性的气息……
<br/>
<br/>        目光飘向单人床，空无一人，听见小阳台传来的炒菜声，颀长的身子立刻往小阳台靠近，不怎么苟同地注视忙碌的身影。
<br/>
<br/>        “呃……大哥，你起来了？”落以柔感受到灼热的目光，讪笑地回过头来，不怎么习惯有旁人存在。“昨天晚上麻烦你了，谢谢。再等一下就可以吃早餐了。”
<br/>
<br/>        一抹清新的笑意漾在她略显苍白的脸庞。
<br/>
<br/>        “不用了，我先走了！”傅夜沉那冷漠的语气抛出。
<br/>
<br/>        落以柔一边打开锅盖，一边慌忙挽留：“不好意思，我这里很简陋，希望你不要嫌弃，浴室里我放了毛巾和牙刷，你梳洗过后，还是吃了早餐再走吧！早餐很快就做好了！”
<br/>
<br/>        有一种令他熟悉得食物的香气，再次扑鼻而来。
<br/>
<br/>        傅夜沉停顿了好几秒，然后照落以柔的意思往浴室里走去。
<br/>
<br/>        十分钟后，两碗猪肝粥上了桌，还配有两个心形荷包蛋。
<br/>
<br/>        上次他失血过多的事情，她心里一直惦记着。
<br/>
<br/>        傅夜沉看着桌上那晚猪肝粥，与筱颖第一次相遇的画面忽然间历历在目。
<br/>
<br/>        那晚，他问：“这是什么汤？”
<br/>
<br/>        筱颖回答：“猪肝红枣枸杞羹，专门补血的，很适合你。”
<br/>
<br/>        “这是你特意为我做的？”他反问。
<br/>
<br/>        筱颖碎碎念：“不是！这是我晚餐的时候煮多了，剩锅底的，你放心，这汤很干净，我没动过！”
<br/>
<br/>        他平生最讨厌吃猪肝，而那一次，筱颖的手艺让他欣然接受了吃猪肝。
<br/>
<br/>        从什么时候起，傅夜沉忽然恍然大悟，筱颖对于他来说，真的只剩下回忆了。
<br/>
<br/>        两人沉默地用餐，没有任何的对话。
<br/>
<br/>        用餐完毕，落以柔收拾碗盘的当口，傅夜沉冷冷地抛下一句话：“今天你在家好好休息。”
<br/>
<br/>        落以柔纤细的身子微微怔了一下，认真地回道：“不用放假，我可以的，反正只是坐在椅子上，又……不用动脚，没事的。”
<br/>
<br/>        傅夜沉的目光再次移到她的脸上，良久过后，她以为他会斥责她不知好歹时，但他只是用冷冷的语气说：“随便你！”
<br/>
<br/>        “大哥，我叫落以柔。”就在傅夜沉离开她家门口前，落以柔嘴角噙着一抹甜甜的微笑，大声自我介绍道。
<br/>
<br/>        她笃定他听到了，只是他未回头。
<br/>
<br/>        傅夜沉毫不留恋地离开落以柔的家后，驱车直接去了公司。
<br/>
<br/>        澳洲，悉尼大学附近的联排别墅区。
<br/>
<br/>        老公出轨，小三带着私生子找上门来了，该怎么办？
<br/>
<br/>        童筱颖在笔记本电脑浏览器的搜索栏里输入了这句话后，得出来的结果有很多。
<br/>
<br/>        第一种，装作什么都不知道，擦干眼泪继续陪老公过下半辈子；
<br/>
<br/>        第二种，与老公对峙，让老公做决定，老公若选择回归家庭，便原谅老公，与老公继续婚姻。老公若选择小三，离婚时让老公净身出户，或平分家产后，向老公索要补偿费；
<br/>
<br/>        第三种，与老公对峙，坚决不离婚，两人就这样吵吵闹闹至死方休；
<br/>
<br/>        第四种，果断与老公离婚，要么老公净身出户，要么平分家产，并向老公索要各种补偿损失费；
<br/>
<br/>        第五种，先不要打草惊蛇，买监听器监听老公一切举动，暗中收集证据，用于打离婚官司为自己争取最大的利益……
<br/>
<br/>        看完这些，童筱颖的心里更加迷茫了起来。
<br/>
<br/>        其实，她潜意识里，已经选择了最作最犯贱的一种，那就是装作什么都不知道，擦干眼泪继续陪着小卫过下半辈子。
<br/>
<br/>        童筱颖隐约想起，曾经她的前男友何明旭也劈腿的时候，她可以做到非常爽快地分道扬镳，甚至能在最短的时间内，化悲伤为动力，继续自己的生活直到邂逅了傅昼景。
