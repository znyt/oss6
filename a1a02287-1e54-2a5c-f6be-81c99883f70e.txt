　　“成功了？真的成功了！万劫魔尊证道混元了。”
<br/>　　“这怎么可能？如此大劫，竟然都被他熬过来了，真是不可思议！”
<br/>　　“混元大罗金仙，一个全新的境界，堪比圣人，就这样成功了？真是厉害！”
<br/>　　“万劫魔尊，天地间又多出以为巨头，洪荒六圣，轮回后土，再加上万劫魔尊，洪荒八大强者。”
<br/>　　“劫魔殿，成为洪荒顶级势力，万劫魔尊的崛起，已经势不可挡了。”
<br/>　　“如此毁天灭地的大劫，连圣人都不敢轻易触碰，万劫魔尊，正是不可思议。”
<br/>　　“……”
<br/>　　在这一刻，洪荒变得彻底沸腾了，今日他们再次见证了一位混元大罗金仙的崛起，洪荒变得风起云涌。
<br/>　　万劫魔尊完全靠自己的实力去证道混元，没有如洪荒六圣那般借助大道之基鸿蒙紫气，更没有如后土那般的无尽的功德之力，靠的是自己实实在在的力量，成为混元强者，这让无数处在准圣巅峰的强者们，看到了突破的希望。
<br/>　　“万劫魔尊既然可以，那么，我也可以？”
<br/>　　万寿山五庄观，镇元子看着虚空静立的万劫魔尊，眼神之中闪过一丝艳羡，但更多的却是坚定，他可以，自己为何不可以？自己又不是比别人差。
<br/>　　“万劫魔尊，哼！让你先了一步，我也会成功的。”
<br/>　　幽冥血海，冥河望着虚空中那仿佛魔神般的万劫魔尊，眼中闪过一丝狠辣，心中杀道法则沸腾，坚定道。
<br/>　　北冥宫、人族武宫、东海海底……这一刻，洪荒各地距离混元的众多大能们，纷纷神色坚定，好似看到了未来的道路，不至于那般绝望。
<br/>　　昆仑山，元始看着虚空中的万劫魔尊，脸色淡然，丝毫没有因为万劫魔尊渡过他降下的天劫而愤怒，就好似那成为混元大罗金仙的万劫魔尊与他毫无关系一般，在他的脸上，看不到丝毫的情绪。
<br/>　　若是有道行更高的在此，便可发现，如今的元始，全身上下就仿佛一个凡人一般，没有丝毫的气息，但却能够让人感觉到深不可测，在他的眼眸深处，闪过七道彩色光芒。显然，元始如今的实力，已经是步入了混元后期，混元大罗金仙七重天境界。
<br/>　　要知道，其他人如今大多都还在混元初期奋斗着，就如准提圣人，如今还只是圣人一重天，但元始却是到了混元后期，真是恐怖。洪荒之中，除了鸿钧道祖那个圣人九重天圆满的，就没有人能够比得上他。
<br/>　　八景宫中，太上那一向淡然的脸上，此刻闪过一丝错愕，不过，很快，他便恢复了淡然，继续炼制手中的仙丹，若是仔细看，便会发现，在他的瞳孔深处，还残留着深深的震惊，显然，他的内心，并不如表面那般淡然。
<br/>　　上清宫中，通天教主手持青萍剑，看着虚空中的万劫魔尊，眼中闪过一丝战意，手中青萍剑微微颤抖，散发出通天剑意，仿佛知道主人心中所想，想要为主人斩灭一切。
<br/>　　“有意思，有意思，这下子看西方那两位要怎么办？哈哈哈！”
<br/>　　西方虚空中，女娲看着渡过天劫成为混元之境的万劫魔尊，眼中闪过一丝震惊，感受着对方身体内那强悍的能量，不比她弱，甚至更强。她眼中异彩连连，喃喃道：
<br/>　　“你这人，还真是神秘？”
<br/>　　而接引圣人与准提圣人，此刻却是除了震惊之外，更多的是愤怒，昔日他们看为蝼蚁一般的人物，如今却是成为了与他们同等级的人物，他们心情能好才怪，更何况眼前这人还是他们的敌人。
<br/>　　不知过了多久，也许是一个世纪，也许是一瞬，万劫魔尊缓缓睁开眼睛，眼中爆射出无比强烈的光芒，洞穿虚空，带着无比强烈的毁灭之力，向着准提圣人爆射而去。
<br/>　　虚空破碎，乾坤动荡，风云变色。紧接着，一道冷冷的声音，传遍洪荒，久久不消：
<br/>　　“准提，可敢一战？”
<br/>　　准提圣人挥手破灭射向他的两道毁灭之光，眼神之中，除了愤怒之外，更多的却是凝重，他知道，这一刻的万劫魔尊，更强大了。之前还没有成为混元，万劫魔尊便能够与他大战，如今实力大涨，更是成为第三步混元大罗金仙，他还能否与之一战，这是一个未知数。
<br/>　　可，他是谁？他可是圣人，天地间最伟大的圣人，怎么能够退缩？他深吸一口气，道：
<br/>　　“战！”
<br/>　　“哈哈，好，本尊今日就来领教圣人高招！杀！”万劫魔尊嚣张大笑道。
<br/>　　大战爆发，两者一上来就动用了禁忌神术，也许需要数百上万招才能分出胜负，但他们希望在瞬间之内便有结果。
<br/>　　准提圣人万法合一，菩提法相大涨，光辉散遍天地，然后法相与大道融合，合而为一，竟然出现一尊恐怖的神魔。
<br/>　　他双手合十，佛光大涨，身高数十万丈，法天象地，大梦千秋，万法合一。
<br/>　　万劫魔尊周身异象纷呈，执掌天地万劫，最后合而为一，化为一尊从虚无之中走出来的恐怖虚影。
<br/>　　这虚影，全身大涨，高大数十万丈，背后密密麻麻，足足有三千只手，而他的身上，密密麻麻的布满了眼睛，每只眼睛之中，都放射出不同的光芒，有贪婪，有淡定，有凶狠，有慈祥，有嗜血，有怨毒，有欢喜，有威严……三千种不同的情绪，集中于一身。
<br/>　　三千魔手，三千魔眼，混沌魔神，魔道法相！
<br/>　　万劫魔尊三千只魔手爆发，划开时空，击穿岁月，与那佛道魔神发生剧烈大触碰。
<br/>　　“噗嗤！”
<br/>　　鲜血飞溅，万劫魔尊的肩头被洞穿，冒出一串的血滴，但他头也不回冲了过去。
<br/>　　前方，鲜血更多，准提圣人整条手臂连着半个肩膀被撕裂，几乎坠落下来，只连着一层皮肉，差点被斩断。
<br/>　　更为可怕的是，万劫魔尊的魔道混沌魔神发相，睥睨天下，藐视圣人，将那尊佛道神魔压在了下方。
<br/>　　他生生压着神魔跪伏下去，口中冷喝一声，道：
<br/>　　“可服？”
<br/>　　而与此同时，万劫魔尊的真身也一击击飞准提圣人，他冷漠无情的看着准提圣人的真身，道：
<br/>　　“圣人，不过如此！”
<br/>　　“啊！万劫魔尊，欺人太甚！”
<br/>　　准提心中怒火中烧，他身为圣人，一直高高在上，何时受过如此羞辱，今日被万劫魔尊如此对待，可想而知，今后在洪荒之中，他准提圣人的脸面何存？
<br/>　　“杀！”
