　　“看来是场激烈的战斗。”林劲眼睛一亮。
<br/>　　“听声音，距离这里不过数百米。”铁牛嗡声说道。
<br/>　　“过去凑凑热闹。”柳嫣然温柔如水的眼眸中，蕴含着一丝期待。
<br/>　　萧逸不语，心里却暗道，“这三个家伙，都不是安分的主儿。”
<br/>　　.....
<br/>　　一行四人循着战斗声响的方向走去。
<br/>　　路上，铁牛问道，“萧逸，你只有先天三重修为，战力却这么强，武魂一定很厉害吧。”
<br/>　　萧逸淡淡道，“还行吧。”
<br/>　　柳嫣然则忽然一副心不在焉的样子，自语道，“来裂天剑派参加考核的，都是全郡各地的天才，尽皆是心高气傲之辈。”
<br/>　　“怎么却肯放下尊严为别人卖命，如此不择手段呢。”
<br/>　　萧逸轻笑道，“很正常，有人的地方，便有江湖，有江湖的地方，便必定有争斗。”
<br/>　　“天才也好，庸才也罢，只要能达成目的，不择手段又如何？”
<br/>　　事实上，萧逸看得很清。
<br/>　　考核如此，这整个世界也是如此。利益二字，驱使着人联合或敌对。
<br/>　　早在前几天，当剑派内的执事说完规则，他就知道会有武者们联合起来的事发生。
<br/>　　只不过，他不在乎，这次的考核，他很轻松就能通过。
<br/>　　选择和林劲、铁牛、柳嫣然一起，并非是怕危险，而是对他们三人产生了兴趣。
<br/>　　这三人，确实是昨天才互相认识的。
<br/>　　但他们的实力，却绝不像表面上那么简单。
<br/>　　萧逸可以确定，就算刚才自己不出手，他们三人中的随便一人，都可以轻松打败胡海以及那数十敌人。
<br/>　　只不过不知出于什么原因，这三人似乎都不想出全力。
<br/>　　他们看起来，更像是在胡闹。
<br/>　　萧逸确定，自己不会看错。
<br/>　　那林劲，虽是先天七重修为，但眸子中蕴含的坚毅，显然是个久经战斗的武者。
<br/>　　还有玄铁重刀，本身就是一种珍贵而奇特的武器。
<br/>　　玄铁，是一种非常稀罕的材料，比之紫云铁珍贵百倍不止。
<br/>　　哪怕只有一丁点儿，也重若千斤。
<br/>　　用它来打造的兵器，怕是比一座小山还要重。
<br/>　　而作为武魂，会让武者每每挥刀前，自身都承受着极大的压力，甚至是刀都提不起来。
<br/>　　就算提起来了，也大大影响速度。
<br/>　　就是因为这个弊端，玄铁重刀武魂才会只是绿色武魂。
<br/>　　但玄铁打造的武器，还有一个功效，便是能无限加入，无限增加其重量。
<br/>　　当然，玄铁珍贵而稀有，无限加入不可能，也不现实。
<br/>　　但作为武魂，武者注入真气，就等同于变相地注入玄铁。也就是说，武者修为越高，凝聚出的玄铁重刀武魂，会越来越重。
<br/>　　虽说对自身的压力越来越重，但威力也是越来越恐怖。
<br/>　　从威力这个层次讲，玄铁重刀武魂，绝对是超越绿色武魂的存在。
<br/>　　且观林劲，战斗时如臂指使，丝毫没有压力。
<br/>　　证明他的真正实力，绝对远超先天七重，战力惊人。
<br/>　　还有铁牛的撼地神牛武魂。
<br/>　　撼地神牛，乃是六级巅峰妖兽。
<br/>　　这种妖兽，身躯高如山岳，防御和力量都高得吓人。
<br/>　　但它非常温顺，几乎不会主动攻击人，终年都在沉睡，而且反应非常迟钝。
<br/>　　只有被人打成重伤，才会反应过来要还击。
<br/>　　因此，觉醒出这种武魂的人，简单来说，都非常憨厚，甚至是蠢。
<br/>　　也是这个原因，这种武魂才沦落到绿色，否则同样是超越绿色武魂的存在。
<br/>　　当然，撼地神牛还比不上紫炎狮王。
<br/>　　紫炎狮王属于霸主级别的妖兽，它的本命天赋，紫晶灵炎太过恐怖了。那焚毁万物的力量，没有东西能抵挡住。
<br/>　　萧逸至今也想不明白，紫炎狮王这种起码需要天元境武者才能打败的妖兽，当年狂血玄君是如何击杀的。
<br/>　　言归正传。
<br/>　　至于烟雨碧柳，萧逸倒是知之甚少，但肯定也非普通武魂。
<br/>　　这三人年纪轻轻，武魂又如此稀有，论天赋，绝对不比北山榜上排名前二十的天才差，甚至更强。
<br/>　　但性格，却如此胡闹而有趣。
<br/>　　也不知这三人昨日是怎么碰到一起的。
<br/>　　还有今日，自己竟然遇到这三个奇葩。
<br/>　　这时，萧逸忽然问道，“铁牛，你真的叫铁牛吗？”
<br/>　　铁牛点点头，嗡声道，“是阿，我爹姓铁，我小名叫牛子，他是个懒人，便随便给我起了个铁牛的名字。”
<br/>　　“额。”萧逸无语了。
<br/>　　不多时，四人来到战斗现场，借着树木藏身在一边。
<br/>　　前方，一个和他们年龄相仿的少年被百多武者团团围住。
<br/>　　可是，少年丝毫没有惊慌。
<br/>　　反倒是那百多个武者，满脸凝重之色。
<br/>　　“一群傻瓜，追了老子那么久才追上，没用的东西。”少年轻蔑地瞥了周围的武者一眼。
<br/>　　“哼，少嚣张了。如今你已被我们包围，有什么资格这么得意。”那些武者满脸怒色。
<br/>　　“哦？包围了我？我说你们是傻瓜吧。”少年得意一笑，而后脚步一跃，在半空中翻了个帅气的身姿。
<br/>　　而后竟化作一道流光，快速飞离。
<br/>　　地面，百多个武者目瞪口呆，却无能为力。
<br/>　　“哈哈哈哈，一群笨蛋，有本事继续追来，你们追到死也伤不到本大爷一根毫毛。”
<br/>　　空中只留下一道得意至极的声音，少年的人影，早就没了踪影。
<br/>　　在一边藏身的四人微微一惊。
<br/>　　柳嫣然柔声问道，“那家伙应该还未到洞玄境吧，怎么御空飞行了。”
<br/>　　林劲低声道，“那家伙是稀有的飞禽武魂，凡境时就能飞行了。”
<br/>　　飞禽武魂，很是稀少。武者拥有这种武魂，只要体内有真气，便能飞行。
<br/>　　不过，这种武魂擅长飞行和速度，在战斗方面却差了一点儿。
<br/>　　当然，也不至于差很多。
<br/>　　铁牛嗡声道，“看刚才那道流光内的样子，好像是金翼雕。”
<br/>　　“金翼雕？”林劲微微一惊，道“看来，刚才那少年，应该就是秦飞扬，北山榜排名16，先天八重修为，他的武魂正是金翼鵰。”
<br/>　　这时，萧逸苦笑道，“先别说了，那个家伙飞走了，那百多个武者却朝我们来了。”
<br/>　　“嗯？”柳嫣然三人一惊。
<br/>　　那百多个武者，竟快速朝他们围拢而来。
<br/>　　“哼，跑了个秦飞扬，倒逮到了四个不知死活的家伙。”百多个武者中，一个为首的年轻男子狞笑着。
<br/>　　这群人中，大多是先天四重，有数十个是先天五重。
<br/>　　林劲等人脸色一凛，显得有些忧心忡忡。
<br/>　　“这群家伙，看着不像是长风公子的手下，应该是另一个团伙的。”林劲咬牙说道。
<br/>　　“人数这么多，我们要跑路吗？”铁牛嗡声说到。
<br/>　　柳嫣然双眸间闪过一道战意，但很快就掠去，而后点点头，表示赞同。
<br/>　　萧逸翻了个白眼，暗道，三个混蛋，还在装蒜。
<br/>　　“这百多号人，身上的分数可不少。”萧逸说到道。
<br/>　　林劲摆摆手，道，“打不过，没办法。”
<br/>　　萧逸撇撇嘴，看向铁牛，道，“你不想得到他们的分数？”
<br/>　　“想。”铁牛毫不犹豫地点点头。
<br/>　　这里头，最好骗的就是铁牛，但他下一秒反应了过来，又摇摇头，道，“打不过。”
<br/>　　萧逸气急，道，“这里的一半家伙我包了，剩下的一半交给你们如何？”
<br/>　　林劲三人立刻点了点头，爽快地说道，“好。”
<br/>　　萧逸顿时恶狠狠地想道，三个混蛋，你们最好是因为有特殊原因才隐藏实力；要是让我知道你们只是想偷懒不出力，老子跟你们没完。
<br/>　　想罢，萧逸率先出手。
<br/>　　“嗯？不过四人联手，竟妄图对付我们百多人的围攻，真真找死。”为首的年轻男子冷声道，“宰了他们四个。”
<br/>　　百多号武者蜂拥而上。
<br/>　　只是，结果可想而知。
<br/>　　战斗仅仅持续了不足半个时辰，百多号人，纷纷被打成重伤，只能跪地求饶。
<br/>　　萧逸也没有赶尽杀绝，他们乖乖交出分数点后，便放了他们离开。
<br/>　　这一战，每人都收获了数十点分数，萧逸收获了近百点。
<br/>　　之前已经说好了，分数点靠自己实力，谁打败的敌人，分数点便归谁。
<br/>　　这百多号人，看来已经抢夺了不少其他武者的分数，但现在却全便宜了萧逸他们。
<br/>　　.....
<br/>　　数天后。
<br/>　　萧逸一行四人遭遇了数次的围攻和战斗。
<br/>　　结果同样如此，林劲三人嘴上说逃跑，最后是萧逸包揽一半的敌人，而后将围攻他们的团伙反过来抢了个清光。
<br/>　　萧逸如今的分数点，足有四百多，林劲三人，也有两百多点。
<br/>　　倒是林劲，还说着风凉话，拍了拍萧逸的肩膀，道，“萧逸兄弟，这几天，辛苦你了。”
<br/>　　萧逸白了他一眼，没好气道，“无所谓，反正我得的分数点比你们多，多对付些敌人也是应该。”
<br/>　　林劲摆摆手，贱笑道，“我也无所谓，反正我又没打算得到前十的奖励，能通过考核就好了。”
<br/>　　铁牛嗡声笑道，“呵呵，要是之后也能这么轻松，直到通过考核就好了。”
<br/>　　萧逸越来越觉得，这三人根本就只是想偷懒。
<br/>　　刚想反驳几句，忽然，不远处传来了激烈的战斗声。
<br/>　　三个不安分的八卦家伙，立马拉着萧逸过去看热闹。
<br/>　　不料，却遇上了熟人。
<br/>　　数百个武者，在一个年轻男子的带领下，将秦飞扬团团围住。
<br/>　　今日的秦飞扬，似乎显得很狼狈，身上带伤。
<br/>　　“嗯？又是这个秦飞扬。”铁牛说道。
<br/>　　“那家伙似乎有危险阿，要不要出去帮他？”林劲说道。
<br/>　　可是，这两人说话的同时，都看向了萧逸，眼神中都写着同样的意思，你先上，打头阵。
<br/>　　萧逸剐了他们一眼，不为所动。
