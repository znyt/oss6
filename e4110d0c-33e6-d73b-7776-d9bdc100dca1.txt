        昏迷之中，连星儿觉得自己好像又听到了明司寒的声音。
<br/>
<br/>        难道是幻觉？
<br/>
<br/>        就在她想到明司寒的时候，明司寒一个电话打了过来，约她出去见面。
<br/>
<br/>        三个室友都在睡觉，连星儿出去时，关门的动作都很轻。
<br/>
<br/>        其实她也有些头晕，只不过明司寒说有个重要的事情要跟她说，不然她也不会离开宿舍区见他。
<br/>
<br/>        宿舍楼下前坪是校道，明司寒站在香樟树下，双手抱臂等着连星儿出来。
<br/>
<br/>        连星儿走到明司寒的跟前，直接开门见山地问道：“你找我到底是有什么重要的事情要说？”
<br/>
<br/>        明司寒这才淡定地从裤兜里掏出手机，点开了一段视频。
<br/>
<br/>        视频里，朱小溪哭丧着脸，被几个女生围着，把为首的女生吐在地上的口香糖给捡了起来，塞到她自己的寝室门锁孔里，然后又把她自己晾在走廊上的衣服洒落在地上，用力地在上面踩踏。
<br/>
<br/>        连星儿此刻却看得一头雾水。
<br/>
<br/>        明司寒接着说道：“整蛊你的女生就是她，我派人‘以牙还牙’地替你报仇了，你不高兴吗？”
<br/>
<br/>        “真幼稚。”连星儿不以为然地转过身去。
<br/>
<br/>        只是，她不明白，朱小溪为什么要这么对她？
<br/>
<br/>        明司寒见连星儿要走，连忙伸手抓住了她的手腕，信誓旦旦地说道：“我保证，以后不会再发生这种事情。有我在，我不会再让她们欺负你。”
<br/>
<br/>        连星儿随之转过身来，看着明司寒，突然间恍然大悟地说道：“噢！我知道了！朱小溪一定是喜欢你，而你我最近在闹绯闻，她整蛊我，十有八九是因为你。”
<br/>
<br/>        “我们不提别人的事情了，好吗？我还有一样东西，要送给你。”明司寒连忙从另一个裤兜里掏出一个粉色的小礼盒。
<br/>
<br/>        他一边递给连星儿，一边微笑着说道：“这是我亲手做的礼物，希望你能喜欢。”
<br/>
<br/>        “？！”连星儿没接，反而不明所以然地问道，“你没事又送我礼物做什么？”
<br/>
<br/>        “你上次不是说，我送给你的订婚见面礼很没诚意吗？所以，这次是我亲手做的，够诚意了吧？”明司寒嘴角微扬地说。
<br/>
<br/>        连星儿仍旧不解地看着明司寒：“我说你是太悠闲了，还是闲着没事做？”
<br/>
<br/>        “我……”明司寒欲言又止，看着连星儿那张质疑自己的漂亮脸蛋，瞬间无话可说了。
<br/>
<br/>        他找了一家珠宝首饰店工作坊的师傅，特意拜师学艺，只为了亲手给她打造一条既有意义又有价值的手链，到头来被她说成是闲着没事干？
<br/>
<br/>        “你这先后频繁送礼，是不是对我有所图啊？”连星儿甩开了明司寒的手，双手抱臂，扬起下巴，质问道。
<br/>
<br/>        顿了顿后，连星儿又接着说道：“你难道是想通过我，找我爹地投资？”
<br/>
<br/>        “……”明司寒彻底无语了。
<br/>
<br/>        “这事简单啊！你没必要给我送这送那的。你是想做电竞游戏项目，缺少资金对吧？没关系，我替你跟我爹地打个招呼就是。”连星儿落落大方地说道。
<br/>
<br/>        他明司寒的个人资料，那天妈咪已经给她看过了。
<br/>
<br/>        明司寒家里确实有钱，但是他家里给的创业资金是限数的，所以，他若是要继续自己创业，就得找其他大老板投资入股。
<br/>
<br/>        至于他上次的那个项目，已经被傅夜沉的团队否决了，他就没想过要再去找傅夜沉了。
<br/>
<br/>        可是，他现在的所作所为，为什么在她连星儿的眼里和心里就是另有所图？
<br/>
<br/>        那么那个洛羽呢？
<br/>
<br/>        她难道不知道，真正对她另有所图的人是洛羽吗？
<br/>
<br/>        “这礼物，你爱要不要！”想到这里，明司寒心里就是一阵恼火，以至于说话时的口气很冲。
<br/>
<br/>        他收回了手，连同那个小礼物也一并被收回。
<br/>
<br/>        连星儿一脸莫名其妙地看着明司寒，只见他皱着眉头，双手抄着裤袋，转身就走。
<br/>
<br/>        他这个人，怎么就这么莫名其妙呢？
<br/>
<br/>        连星儿十分不解地看着明司寒那渐行渐远的背影。
<br/>
<br/>        哎——
<br/>
<br/>        真是个阴阳怪气的男孩子。
<br/>
<br/>        被明司寒这么一闹，连星儿彻底不头晕了，回到宿舍后，她打开了笔记本电脑，在线做完一套英语四级的模拟试卷，得了九十分后，她觉得不满意，于是又做了一套。
<br/>
<br/>        直到拿到满分试卷，连星儿才放松地撑了个懒腰。
<br/>
<br/>        突然间，她又想玩游戏了，于是打开了游戏对战平台。
<br/>
<br/>        登录游戏账号后，她收到了“余寒犹厉”给她的留言。
<br/>
<br/>        前段时间，我的账号被盗了，好友也被清空了，还好能重新加上你。——余寒犹厉。
