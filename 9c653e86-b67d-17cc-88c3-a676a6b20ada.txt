　　“混沌族的人，若是一直不见你的踪迹，定然会派人查探，若是被混沌族之人找到这里，我们都完了！”紫霄摇了摇头说道，他并不赞成帝一魂留在这里。
<br/>　　还是去幽冥族、不死族，或者是魔族才最为保险。
<br/>　　“放心，这颗星辰是一颗废弃星辰，混沌族的人不会找到这里！”
<br/>　　帝一魂微微一笑，继续道：“再者，这颗星辰极为特殊，处在空间夹层之中，若不是路过这里，神念都无法查探的到，这是绝佳的安生之所！“
<br/>　　“我只是给你一个提议，免得你阴沟里翻船！”紫霄无奈的说道，他知晓自己无法说服帝一魂，也只能无奈的放弃了。
<br/>　　“没事，我自有分寸！”帝一魂摇了摇头，他既然敢留在这里，自然有些手段，就算是混沌元王来了，都不会轻易的发现他。
<br/>　　“还有，天王宫的人，一部分都在玄荒星域，距离魔族的一支天魔族不远，他们已经在那里暂时安顿了下来！”紫霄说道。
<br/>　　“那就好！”帝一魂闻言点头，虽然目前天王宫对于他来说，作用已经不大了，但也是一股不小的力量。
<br/>　　“这些资源给你，我走了！”紫霄手掌一番，掌中出现了一枚储物戒，递向帝一魂。
<br/>　　这储物戒内，均是修炼资源，有源石、有天材地宝，还有珍稀丹药，乃是他从幽冥族的强者手中为帝一魂弄来的。
<br/>　　“你去何地？”帝一魂问道，他和紫霄之间，并未多深的交情，只不过是互相合作的关系，对于对方的打算，他并不是很清楚。
<br/>　　“去玄荒星域，找我的好师妹！”紫霄微微一笑，眸中闪过阴冷的光芒。
<br/>　　他可是清楚，绫裳曾经出声与宇宙之中，乃是上清神宫的祖师。
<br/>　　现如今，神域毁了，绫裳肯定会回上清神宫，就算不回，那上清神宫也是绫裳的软肋。
<br/>　　言罢，紫霄转身离去，离开了这颗蔚蓝色的星辰，向玄荒星域而去。
<br/>　　……
<br/>　　玄荒星域，羽化星河。
<br/>　　一颗同样不起眼的小型星辰之上，坐落着一片无垠的山脉，延绵无际。
<br/>　　山脉中山高林密，兽吼震天，充满了凶煞之气。
<br/>　　山脉深处，一座幽深的山谷之中，有这一座漆黑的山洞，山洞极深，其内非常宽敞。
<br/>　　此时此刻，山洞中，有数十人盘坐在其内。
<br/>　　这些人，正是地王宫之人，有拓跋青云，地渊宫主，地藏神王，以及一些地王宫的弟子和长老。
<br/>　　“父亲，现在地王宫的人手，全部都分散了，只有我们数十人聚在了一起，必须尽快将核心之人全部找到！”地藏神王面色凝重的望着地渊宫主。
<br/>　　目前，整个地王宫，只有他们数十人，其它人全部分散了。
<br/>　　他们之所以能聚拢在一起，还是因为有特殊的联系方法，才得以重聚。
<br/>　　“是啊宫主，若是不尽快，时间一长，地王宫就名存实亡了！”一名长老附和道。
<br/>　　若是时间拖的太久，那些人就生出了异心，加入其它势力力，或者是孤身在外，被人斩杀也有可能。
<br/>　　“除了长老人物和一些核心的弟子之外，其它人就不用管了！”地渊宫主沉声道。
<br/>　　“什么？”
<br/>　　“宫主，您的意思是，那数十万的弟子，都不要了吗？”
<br/>　　众人纷纷惊呼出声，完全没有想到，宫主居然有如此的想法，这么多的弟子，若是全部都不要了，那地王宫也就明存实亡了。
<br/>　　“那些普通的弟子，要了也是无用，只要将所有长老和数百名核心的天才弟子找到，天王宫就能传承下去！”地渊神王沉声说道，这也是无奈之举。
<br/>　　“这……！”一众地王宫之人，个个面色晦暗，他们完全没有想到，事情居然会发展到这个地步。
<br/>　　“诸位！”
<br/>　　略显嘶哑的声音响起，身形矮壮，身穿黑抛的拓跋青云长身而起，环视众人，道：“现在是非常时期，必须有所舍弃！”
<br/>　　略一停顿，拓跋青云继续道：“接下来，帝一魂绝对不会放过苏莫，幽冥族、不死族、魔族和混沌族，必然会爆发剧烈的冲突，甚至，还会有大量的势力会卷入其中；现如今苏莫势大，帝一魂更加势大，我们只能隐藏在暗中，坐山观虎斗。”
<br/>　　地渊宫主点了点头，道：“青云的意思，便是老夫的意思！”
<br/>　　众人闻言纷纷点头，目前的局势，的确是这样，他们一旦行踪暴露，将有灭顶之灾。
<br/>　　无论是帝一魂还是苏莫，一旦知晓他们在这里，必然会发出雷霆一击。
<br/>　　总体来说，拓跋青云处于绝对的弱势，只有韬光养晦才有希望获得最后的胜利。
<br/>　　“父亲、青云，既然如此，不如我们再去加一把火，让三大种族和混沌族，拼个你死我活，我们再渔翁得利！”地藏神王轻笑一声说道。
<br/>　　“说来听听！”地渊宫主道。
<br/>　　“办法很简单，挑起混沌族和三大种族，以及其它各族的矛盾，让得所有势力都更加怨恨苏莫和混沌族！”地藏神王道。
<br/>　　“不妥！”
<br/>　　地渊宫主立刻摇头，沉声道：“三大种族势大，而且，会借助苏莫和混沌族毁掉神域所引起的仇恨，拉拢到众多的势力对付苏莫和混沌族，这样下去苏莫和混沌族必败。”
<br/>　　“这倒也是！”地藏神王叹了口气，苏莫可不能这么快败给帝一魂。
<br/>　　“父亲、大哥，帝一魂和苏莫的争斗，我们无需插手，只要注意局势的发展便可！”拓跋青云开口了，声音低沉的道：“现在，最主要的是找到时空之城。”
<br/>　　“不错，时空之城是关键，以往是永恒神宫之物，现在，谁能找到就是谁的！”
<br/>　　“可是，荒界宇宙这么大，我们人手太少，如何去寻找？”
<br/>　　“是啊，四大星域，比神域大无数倍，想要在别人之前找到时空之城，可能性微乎其微。”
<br/>　　众人纷纷开口，他们自然知晓时空之城的重要性，若是他们能得到，拓跋青云就稳操胜券了。
<br/>　　“再困难也要去找！”拓跋青云面色凝重，道：“另外，还要去找永恒神宫的强者，只要拉拢到他们，也可以布置出时间阵法。”
<br/>　　“永恒神宫是不会给外人布置时间阵法的，拉拢也无用！”地藏神王道。
<br/>　　“今时不同往日，先找到永恒神宫的人再说！”拓跋青云道。
<br/>　　众人闻言沉吟，这倒也是，现在神域都没有了，永恒神宫说不定会改变态度。
<br/>　　“青云，你继续修炼，所有的事情都交给我们吧！”地渊神王拍了拍拓跋青云的肩膀，道。
<br/>　　“是，父亲！”拓跋青云点头。
<br/>　　随后，拓跋青云进入了山洞深处，前去闭关修炼。
<br/>　　而地王宫的这些人，也分头行动了起来，一边收拢天王宫散落在外的强者，一边寻找时空之城和永恒神宫的人。
