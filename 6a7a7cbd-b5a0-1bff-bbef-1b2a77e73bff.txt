　　“爸妈？那是什么东西……听起来好好吃的样子！”
<br/>　　蓝色小人发出一阵尖锐的笑声，随后身上蓝光大炽，照出了白小飞脸上还残留的笑容。
<br/>　　“铿！”
<br/>　　长刀停在变异青蛙的头顶一寸处，发出一声金铁相交的声音，仿佛那里，有一层看不见的金属墙壁阻挡一般，无论白小飞怎么用力，都无法再推进一丝。
<br/>　　“嘎嘎~上当了哦！”
<br/>　　脸上露出得意的笑容，蓝色小人双手合在胸前，与此同时，他身下的变异蛤蟆张开了嘴巴。
<br/>　　白小飞心中闪过一抹警兆，但还未来得及有任何动作，四周的墙壁却是片片崩塌。
<br/>　　与此同时，一股强烈无比的波动涌入耳朵，那强大的力量，几乎将他体内的内脏都震的要碎裂了一般。
<br/>　　可，那个青蛙明明没做什么啊！
<br/>　　没有任何声音，肉眼也看不见有任何东西，这无形无声的攻击，让他不知道应该怎么去应对。
<br/>　　次声波么……
<br/>　　刚从灰尘中走出，刘哔便感应到了这股力量，这是一种类似于声波攻击的方式，但区别是这种攻击没有声音，并且会直接攻击人体内柔弱的内脏，所以更加难防。
<br/>　　不过，从他原本的世界中，他是见识过这种手段的，只要清楚这种手段，抵御起来并不是多么困难的事情。
<br/>　　可，他用得着防御么……
<br/>　　为了隐蔽，次声波的振动幅度要比声波小了很多，故此才能达成静音的效果，但如此一来，次声波的作用，也就只能用在人体内部。
<br/>　　这样一来，身处次声波中，刘哔根本不会受到任何影响。
<br/>　　直接伤害内脏？
<br/>　　伤就好了。
<br/>　　以他的身体强度，即使是内脏，也不是区区这样的攻击能震碎的。
<br/>　　更何况，哪怕是震碎了，也没有什么关系。
<br/>　　五脏六腑，到现在刘哔都没搞明白这些物件对自己还有什么用处。
<br/>　　“去死吧，嘎嘎！”
<br/>　　蓝色小人发出一阵狂妄的笑声，白小飞脸上的表情愈加痛苦，眼看将要支撑不住的时候，一柄带着浓浓火焰的长棍，却突然出现在了蓝色小人头顶。
<br/>　　“该死，你怎么，这样都硬抗？！”
<br/>　　虽然一直紧盯着刘哔的位置，但对方爆发出来的速度，让它根本看不清楚。
<br/>　　也只有不到一个呼吸的时间，那炳棍子就出现在他头顶了。
<br/>　　“噗嗤！”
<br/>　　没有任何反抗的余地，蓝色小人跟随青蛙一起，被砸成了一地碎屑，甚至令人作呕的血肉，因为火焰棍棒的高温，瞬间燃烧了起来。
<br/>　　空中飘出一股腐肉被烧焦的恶臭，白小飞从精神恍惚的状态中缓了过来，张了张口，正欲说些什么，却软倒在了地面上。
<br/>　　皱了下眉头，刘哔伸手探了探白小飞的鼻息，却发现他只是昏迷了过去，便也不在管他。
<br/>　　“啪啪啪……”
<br/>　　鼓掌的声音响起，刘哔目光一凝，转头看去，是先前已经撤离战场的保安小哥。
<br/>　　“精彩，精彩至极。”保安小哥身边躺着小鹿，生死不知，而他本人正一脸微笑的看着刘哔：“谢谢你，给我带来了这么一出好戏。”
<br/>　　转动了一下手中的火焰长棍，苍白的手掌穿过赤炎，紧握着黑色棍体，刘哔没有说话，脸上都是戒备的神色。
<br/>　　正主……出来了。
<br/>　　“咦？看起来，你似乎不怎么担心你的同伴。”保安小哥停下手中的动作，目光中带着疑惑。
<br/>　　短暂的接触，在他看来，刘哔应该是一个很在乎同伴的人，否则的话，也不会为了帮这个小女孩冒险，而深入险地了。
<br/>　　“关我屁事。”
<br/>　　语气有些冷漠，刘哔一改先前在团队中的态度。
<br/>　　白小飞昏迷了，小鹿又生死未知，其他人不在这里，他自然用不着搞那些虚头巴脑的伪装了。
<br/>　　“这样啊……那么你来这里，又是为了什么呢？”
<br/>　　保安小哥转头看了看小鹿，继续猜测着，明显，他相信刘哔的说辞。
<br/>　　这根本不是为了营救同伴而伪装出来的不在意，而是彻底的漠不关心，他能听出，刘哔话语中的那股漠然。
<br/>　　“当然是，吃了你啊。”
<br/>　　摘下人皮面具，刘哔的嘴唇上，两根已经遮盖不住的锋利獠牙，不甘寂寞的从唇上露了出来，保安小哥心中暗惊，他没有想到，刘哔不是人类。
<br/>　　可这样子，是尸兄么，但为什么有些像吸血鬼？
<br/>　　那些蠢笨的尸兄，也没有这样高的智慧啊。
<br/>　　“你是……谁家的人？这个牙齿，该不会是……”
<br/>　　脸色阴沉了下来，保安小哥收起了先前放松的姿态，对于眼前的敌人，他也打起了十二分的戒备。
<br/>　　“关你屁事。”
<br/>　　轻舔着獠牙，刘哔看向保安小哥的眼神，如同在看一个大号肉排一般，充满了渴望。
<br/>　　“小子，说话这么冲，小心活不长啊。”
<br/>　　神秘人的脸色彻底阴沉了下去，像是将要滴出水来一般，在他看来，刘哔简直就是在挑衅自己。
<br/>　　事实上，只是他想多了，刘哔只是懒得找借口回答他而已。
<br/>　　在这个世界上，任何问题都能用关你屁事和关我屁事来回答，当然，前提是你要能接受被这样回答人的怒火。
<br/>　　而明显，刘哔大概有这样回答的实力，所以不加犹豫的，他便这样回答了。
<br/>　　“这样说啊，那你大可来试试，能不能杀掉我。”

<br/>　　自然也注意到了这样的动作，刘哔眼睛微微一眯，浑身的肌肉瞬间绷紧。
