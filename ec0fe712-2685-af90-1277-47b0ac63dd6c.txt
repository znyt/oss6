　　逛够了玻璃大棚，李二与长孙两人又参观了一下李承乾的‘兰若寺’，再一次被其奢华的布置所震惊。
<br/>　　只见‘兰若寺’中，所有的窗子全部被换成了玻璃窗，午后的阳光透过玻璃照进屋子，使房间里的光线异常明亮。
<br/>　　“啪”，李二终于是忍不住了，一把掌抽在李承乾的手脑勺上：“混小子，你这还是陋室？！啊？！就这一块玻璃，值不值一个诸葛庐！你自己说！”
<br/>　　想到东宫门口那篇该死的《陋室铭》，以及上面魏征的提字，李二就感觉这真是滑天下之大稽，也不知道李承乾到底是怎么蛊惑魏老头的，竟然会留下那样一篇东西。
<br/>　　还说什么什么‘斯是陋室，惟吾德馨’，这话就是狗屁，完全就是狗屁！
<br/>　　那个‘德馨’的会拿着小镜子把自己老子气的一夜没睡好；又有哪个‘德馨’的会守着玻璃大棚，却不给自己老子送去几筐新鲜蔬菜；再有哪个‘德馨’的会让自己老子大白天点灯批奏折，差点把眼睛累瞎，自己却躲在光线明亮的屋子里打扑克……
<br/>　　另外，还有那句‘无丝竹之乱耳，无案牍之劳形’，自己这个当爹的天天批折子都特么快要累死了，这混蛋小子，竟然还敢‘无案牍之劳形’，这简直就是在找打，不打不足以平‘朕’愤！
<br/>　　“啊！别，别打，父皇，儿臣有宝物献上，有宝物献上！”眼瞅着李二就要抓狂暴走，李承乾撒腿就跑，在一根廊柱下站定之后，急赤白脸就开始解释。
<br/>　　“宝物也不行，不就是镜子么，朕不要了，今天老子要是不亲自揍你一顿，难解心头之恨！”李二越琢磨那篇《陋室铭》越觉得生气，撸胳膊挽袖子就要去追李承乾。
<br/>　　这混蛋小子，还敢说什么‘水不在深，有龙则灵’，今天把他揍成一条虫，看他怎么个灵法。
<br/>　　“二哥，且先消消气，看看乾儿说的宝物到底是什么可好？”长孙皇后见李二光打雷不下雨，只是一个劲的在吵吵，便知道老头子不是真想惩罚混小子，于是便上前劝阻，顺便给了李二一个台阶下。
<br/>　　“是啊，父皇，儿臣献上宝物您一定喜欢，一定喜欢！”只要能不被揍，李承乾也是豁出去了。
<br/>　　上次被揍了一回，屁股都被打肿了，趴了十来天才好。
<br/>　　这次要是再被揍，指不定被打成啥样呢。
<br/>　　李承乾可没忘这几天自己都干了些啥，又是怎么和老头子别苗头的，被抓住，结局如何不用想都知道。
<br/>　　“哼！拿来给朕看看！”被老婆拉住的李二眼了李承乾一眼，面色不善的说道。
<br/>　　“喏！”李承乾答应一声，几步窜进屋子，伸手在桌上抓起一个圆筒，又极快速的从屋子里窜出来，看样子生怕被李二给堵在屋里。
<br/>　　“父皇，这个叫望远镜，就是放在眼前……”李承乾一边说，一边把圆筒放在眼前，演示了一下用法。
<br/>　　“朕不用你教！”李二劈手夺过李承乾手中的望远镜，学着儿子的样子，往眼睛前面一放。
<br/>　　顿时，眼前的一切景物瞬间远离，变的无比渺小。
<br/>　　李二被吓的激灵一个哆嗦，手一抖，望远镜直接脱手向地上摔去，好在身边的方老太监眼急手快，探手一把抓住，才没有掉到地上。
<br/>　　对面老头子怒瞪过来的眼神，李承乾缩了缩脖子，尴尬的笑笑，比了个翻转的手势：“父皇，反，反了！”
<br/>　　反了？李二老脸微微一红，再一次拿过方老太监递过来的望远镜，放到眼前。
<br/>　　这一次与刚刚的结果截然相反，远处的一切事物瞬间直逼眼前，由其是李承乾那一对漂亮的大板牙，简直就像贴在眼睛上一般。
<br/>　　这次李二有了心理准备，没有过份的惊讶，只是换了一个方向，抬起望远镜向更远的地方看去。
<br/>　　刚刚看着迷迷糊糊的景物在所谓的望远镜中变的清晰无比，学着李承乾的样子拉动套筒，更远处的景物也开始变得清晰，甚至李二可以清楚的看到他曾经住过的丽正殿檐角。
<br/>　　放下望远镜，李二面色有些严肃，指着一个侍卫说道：“你，立刻给朕爬到丽正殿屋脊上去。”
<br/>　　待侍卫离开，长孙皇后有些忐忑的问道：“二哥，怎么了！”
<br/>　　李二的脸色严肃的有些吓人，这让长孙皇后有些担心李承乾的屁股。
<br/>　　“观音婢，你来看看吧，当心些，不要被吓到！”李二趁着侍卫去爬丽正殿的时间，开始现学现卖的教老婆怎么玩望远镜。
<br/>　　果然，望远镜拿到眼前之后，长孙皇后也被吓了一跳，但几个呼吸之后，这位漂亮的过份的后宫之主就喜欢上了望远镜这东西。
<br/>　　拿在手里一会儿看看这，一会儿看看那，还会扯着李二讨论自己看到的东西。那快乐的样子，看的李二一阵心痛，好多年没有见到老婆这么开心的样子了。
<br/>　　自从当年自己领兵出征的那一天开始，这个女人就一直在自己的身后，默默的注视着自己，不断的为自己祈祷，为自己担心。
<br/>　　十多年的时间过去，似乎自己真的欠这个女人很多！
<br/>　　不过想归想，伤感的情绪来的快去的也快，作为一个皇帝，李二很懂得如何控制自己。
<br/>　　又一个圆筒被递过来，那是李承乾拿出来的第二个望远镜，老妈玩的开心，明显一时半会儿不打算还给老头子了，所以作为一个孝子，必须为老头子排忧解难。
<br/>　　丽正殿屋脊上，刚刚被派去的那个侍卫老老实实的站在上面，李二在观察了很长一段时间之后，终于满意的点点头。
<br/>　　不错，这东西的确算是宝物。
<br/>　　身为天策上将，李二几乎是在发现望远镜可以看到远处景物的瞬间，立刻就明白了这东西的用途。
<br/>　　兵法有云，知己知彼，百战不殆，能够首先发现敌人，提前知道敌军的布置情况，这对于领兵打仗的将领来说，无疑是一件绝世宝物。
<br/>　　所以李二对望远镜很满意，决定暂时先放李承乾一马，只等下次有机会再揍他也不迟。
<br/>　　反正这小子一天到晚总是在折腾，想揍他不愁找不到机会！
