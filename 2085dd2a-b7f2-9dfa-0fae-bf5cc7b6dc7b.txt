　　砰的一声，枪响了，倒下的却不是小三，而是草上飞。
<br/>　　小三身边一个少年举着手里的步枪，枪管还冒着烟，刚才那一枪直接打在了草上飞的脑袋上，在眉心处打出了一个血洞，直接毙命。
<br/>　　“二当家的说了，举手最多的人就是新任大当家，四金刚用枪指着大当家，其罪不容诛，若非我反应快，三爷可能这会儿已经被他杀了！我也是不得已。”这个少年端着枪说道。
<br/>　　小三看了这少年一眼，眼底是深深的赞许，却立刻又很狠下来斥道：“以下犯上，就算你有理由，可也不该打死了他，你们几个把他关起来，回头再处置！”
<br/>　　几个手下上去按着这少年，朝外面押了出去，这少年也不反抗，一边走一边大声喊：“弟兄们，三爷才是适合当家的，跟着他，大伙跟着三爷干啊！一起吃香的喝辣的！”
<br/>　　少年被押走之后，铁算盘站出来，折扇一打，说道：“好了，四金刚的死，也是咎由自取，大当家既然选出来了，按规矩办吧。”
<br/>　　说完自己先跪下去，参拜大当家。
<br/>　　之后土匪们陆续都跪了下去，就连草上飞的几个铁杆手下，也是没有办法了，草上飞已经死了，他们没有靠山，就算是再不情愿，也是不得不接受现实了。
<br/>　　小三推辞了一番，只好坐在了鬼见愁曾经坐的那张铺着虎皮的头把交椅上，发自肺腑地跟发表了一些鼓舞士气的话。
<br/>　　之后有一个土匪插绺子的仪式，小三封了新的四大金刚，两个是自己的手下，两个是山寨里的元老，把他们放在前面，是为了让他们心理平衡一点，这二人在土匪之间人缘最好，说话能服众，之前一直被四大金刚压着，有些郁郁，现在小三一上任就提拔他们做了四大金刚的前两位，两人心中也是十分欣喜，对小三自然是比之前对鬼见愁还要看着顺眼。
<br/>　　仪式完事之后，又重开宴席，大伙接着喝，一直喝到天亮，方才各自去歇息。
<br/>　　按照规矩，三爷应该是住在鬼见愁的那间房，不过因为昨晚才闹过僵尸，有些晦气，于是小三还是回到自己的卧房去，在回房之前，他先去看望了被押在牢房里的那个少年。
<br/>　　那少年一看他来了，立刻说道：“拜大当家。”
<br/>　　“金祖安，这里没外人，都是我们自己几个弟兄们，不必这样。”三爷在牢房里蹲下，招呼几个手下都坐，然后拿出一包卷烟，给大家分着抽。
<br/>　　这几个都是当时跟他一起上山的逃兵，吴营长的手下，吴营长被他作掉之后，成了他的手下。本来还有几个，在灭吴营长的时候，有几个都被他杀了，还有几个之前死在那女尸的手下，就剩眼前这几个了。
<br/>　　三爷吸了口烟，转头看着几个人，说道：“我小三当时要当土匪，兄弟们还犹豫，现在可曾比你们回家种地的好？”
<br/>　　几个人抽着烟，都嘿嘿笑起来。
<br/>　　“三爷抬举！”有人奉承了一句。
<br/>　　三爷摆了摆手，“我们是喝过血酒的，过命的兄弟，坦白说我不是什么好人，不过我也需要自己信得过的兄弟，咱哥几个是从死人堆里爬过来的，我小三话放在这，只要你们铁了心跟着我干，说的文一点，苟富贵，勿相忘！大白话就是，有我特么的小三一口肉吃，决不让你们喝汤！”
<br/>　　几个人互相看看，眼神中都闪烁着一种光，一起点了点头，发誓绝对忠诚。
<br/>　　三爷看了金祖安一眼道：“金祖安，你今天做的很好，三爷我绝不会忘了你，你先吃几天牢饭，等局势完全稳定，我让你出山！”
<br/>　　金祖安用力点头。
<br/>　　一根烟的工夫，三爷跟他们说了很多掏心窝子的话，又趁势鼓舞了一番，这几个本来就忠诚于他，现在这种忠诚更近了一步，对未来也是充满了展望。
<br/>　　大伙一起出了牢房，三爷吩咐他们去巡逻，今晚自己刚当上老大，怕出事，自己回房间去睡觉。
<br/>　　刚进屋，就看到铁算盘从座位上站起来，笑着拱手，道：“大当家！”
<br/>　　三爷笑起来，拍着他的肩膀坐下，自己也在对面坐下，说道：“这里也没外人，实话说，之前在树林里，多谢先生助我一臂之力。”
<br/>　　铁算盘的身份是二当家，按说该叫一声二爷，不过因为江湖人供奉的是关二爷，“二爷”这个称呼，绝对是不能用的，再加上铁算盘是书生，所以就称呼先生了。
<br/>　　铁算盘笑道：“三爷这么说就严重了，我也是明哲保身，为自己谋个后路。”
<br/>　　三爷微笑，说道：“你怎知我要夺这头把交椅？”
<br/>　　铁算盘道：“从三爷第一天上山，我就看出来了，三爷不是久居人下的人，你是一匹狼！鬼见愁死，你必然是要夺这寨主之位，我若是跟草上飞等人一路货色，现在只怕是也死了。”
<br/>　　三爷听了这话也不是生气，反问道：“你是二当家，大当家死，你这二当家的理应补上，为什么让给我？”
<br/>　　铁算盘摇头说道：“我清楚我那两下子，之前我没撒谎，我只能当个军师，掌不了舵。我也不想给草上飞那种货色卖命，不然将来也会连累我。”
<br/>　　三爷道：“你说我是狼，你给狼卖命，你不怕我吃了你？”
<br/>　　铁算盘望着他，露出笑意，“狼不吃同伴。”
<br/>　　“你怎么知道我把你当同伴？”
<br/>　　铁算盘拱了拱手，道：“恕我直言，三爷，你是枭雄的命，为了自身利益，你是可以舍弃任何的，不过……我对你只会有用，却没有威胁。我跟你是一条船上的，你掌舵，我扯帆。”
<br/>　　三爷望着他，笑着点了点头，说了四个字：“如虎添翼。”
<br/>　　两人聊了一会，铁算盘说道：“三爷，你有什么打算？”
<br/>　　“打算？”
<br/>　　“二龙山这种小地方，养不出真龙，以三爷的手段才智，不可能只想做个土匪头子，必然有更远的打算，再者这二龙山离县城太近，之前一直赶上战乱，官府腾不出手来对付我们，将来总有一天会打上来，二龙山不是长久质地。”
<br/>　　三爷点点头，道：“依你之见呢？”
<br/>　　铁算盘一笑，开始侃侃而谈，两人制定起将来的计划……
<br/>　　存了，忘了发了。。。。
