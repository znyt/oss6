　　“第二城的强者很多，有贺兰殇、龙羽、还有阎公子！”公孙长绝道。
<br/>　　“哦？”苏莫闻言，顿时眸光一闪，龙羽和阎魔鬼圣居然都在第二城之中。
<br/>　　三位公子级的人物，居然全部汇聚到了一起。
<br/>　　“其它几座城池呢？”苏莫继续问道。
<br/>　　“第四城、第五城、第六城和第七城也都打开了护城阵法，我无法潜入，情况不明！”
<br/>　　公孙长绝叹了口气，继续道：“第八城虽然也开启了护城大阵，但是，在大阵开启之前，我在其内感受到了强烈的古魔太子的气息，所以我猜测，古魔太子在第八城；至于第九城比较诡异！”
<br/>　　“第九城怎么了？”苏莫沉声问道。
<br/>　　“第九城是空城，一个人都没有！”公孙长绝面色凝重的说道。
<br/>　　“什么？”苏莫闻言一怔，其他人亦是满脸诧异，一个人都没有，怎么可能？
<br/>　　参加千选战的九百四十人，随即被分配到九座城池之中，第九座城池怎么可能会没人？
<br/>　　“这不可能吧！”霸天族的力闫皱眉说道，他有些不太相信公孙长绝之言，但是，貌似对方不至于用这么拙劣的骗局。
<br/>　　“我没敢进入城内，只在城墙上空停留了一瞬，一个人也没有见到，也没有任何的气息！”公孙长绝道。
<br/>　　众人闻言，纷纷沉默了下来，这的确是诡异无比。
<br/>　　“苏莫，你接下来准备怎么行动？”公孙长绝沉声问道，现在，他自己也有些搞不清楚局势，不知该如何行动。
<br/>　　最主要的是，不清楚每个城池的情况，不能贸然行动。
<br/>　　“可以舍弃城池吗？就是所有人都离开城池？”苏莫疑惑的问道，这一点龙腾好像并没有告诉他。
<br/>　　“舍弃可以，但是，城池若是被其他人占领和摧毁，我们就算是败了！”公孙长绝说道。
<br/>　　“既然如此，那么第九城之中，肯定有人！”苏莫眸中精光一闪，心中暗自沉吟。
<br/>　　目前，第一、第四、第五、第六、第七、第八城，全部都开启了护城大阵，想必这几座城内的局势都稳定了。
<br/>　　要么是有人独领风骚，要么是众人合作，反正这几座城是稳定了内部。
<br/>　　第九城表面无人，情况不明，不好行动。
<br/>　　现在，唯独是第二城在内战，是一个比较好的突破口。
<br/>　　“公孙兄，你查看第二城之时，城中战况如何？”苏莫再次问道。
<br/>　　“在我去查看之时，第二城中的贺兰殇、龙羽、阎公子，还有另外几人都在争夺第二城的统领之权，不过，由于第二城之中强者比较多，还没有稳定局势。“公孙长绝道。
<br/>　　苏莫闻言沉默不语，他看了龙双双一眼，对方对他微微摇了摇头。
<br/>　　适才，他曾暗示对方，试着能不能给龙腾、龙羽等人传讯。
<br/>　　现在对方摇头，看来在这里，并不能传讯。
<br/>　　“诸位，你们觉得，我们现在该如何行动？”苏莫环视众人，朗声问道。
<br/>　　现在局势不明，他心中也没有太好的计策，不知道该如何行动？
<br/>　　“我觉得，我们应该主动出击，趁着第二城混乱，击败第二城！”霸天族的力闫说道。
<br/>　　“不妥，我们若是去攻击第二城，必然有人来袭我们第三城！”公孙长绝摇了摇头道。
<br/>　　“公孙兄，你有什么好办法？”一名人王宫的弟子问道。
<br/>　　“九座城池，任何一座城池，想要凭自己的实力击败其它八城，都没有任何的可能性！”公孙长绝面色郑重的说道。
<br/>　　“你的意思是，合纵连横，寻找盟友？”苏莫微眯着眼眸问道，他心中也是这个想法，自身绝对不能独自出击。
<br/>　　最主要的是，他不清楚那些厉害人物的实力，如霸云公子、古魔太子等人，都是公子级天才，本身的修为又高达八品、九品，即便修为被压制到了一品低级，战力也绝对恐怖。
<br/>　　所以，苏莫不敢贸然行动，若是他能确定自己的实力能横扫这些人，哪里还会这么谨慎，直接便带人杀出去了。
<br/>　　“不错，寻找盟友！”公孙长绝点了点头，这是最稳妥的办法。
<br/>　　苏莫闻言，心中在思量，除了第二城和第九城之外，所有的城池都开启了护城大阵，其内是哪些人他并不清楚。
<br/>　　只有第八城能大致确定，应该有古魔太子在其中。
<br/>　　但是，他却是并不想与古魔太子联合，此人实力太强，而且，也让他极度不信任。
<br/>　　“现在，最好的合作对象就是我族太子！”魔野开口说道，若是能和太子联合，那是最好不过了。
<br/>　　“还是与第二城联合最好！”龙双双开口说道，龙羽在第二城，她自然希望于第二城联盟。
<br/>　　“不错，第二城最合适！”苏莫点了点头，第二城之中不仅有龙羽，还有阎魔鬼圣，正是他联盟的对象。
<br/>　　只不过，第二城之中还有贺兰殇，此人当初在时空之城，不知为何对他出手。
<br/>　　魔野闻言，眉头微皱，不过，他也没有多言，他心中清楚，自己争不过苏莫，且先看看情况再说。
<br/>　　公孙长绝没有什么意见，相比于不太明了的第八城，第二城的确是更加的合适。
<br/>　　苏莫心中沉吟，神念透过护城大阵，向第二城探查而去，想要看看能不能看出什么端倪。
<br/>　　每一座城池，都有编号，苏莫能轻易的找到第二城。
<br/>　　不过，当他的神念到达第二城之时，发现第二城的护城大阵已经开启，说明其内的局势已经暂时稳定了。
<br/>　　“龙双双，你去一趟第二城！”苏莫转头看向龙双双道。
<br/>　　“我？”龙双双闻言一怔，让公孙长绝出去，不是更合适吗？
<br/>　　“对，由你前去，代表我们第三城和他们第二城联盟，但你记住，除了龙羽之外，不能让其他人知晓我在第三城，就说第三城是有公孙兄统领！”苏莫沉声说道，他并不信任公孙长绝，此事还是由龙双双前去最好。
<br/>　　“可是，我不擅长隐匿，若是前去第二城，肯定会被其他城池的人知晓！”龙双双面色凝重的说道，她并不是不想前去，只是他不擅长隐匿行踪。
<br/>　　“无妨，我会注意着你的情况！”苏莫道，虽然他们第三城之中，除了公孙长绝之外，肯定有善于隐匿之人，但还是龙双双最让他信任。
<br/>　　“好吧！”龙双双点了点头，随即身形一闪，破空而去，这城池上的护城大阵，从内向外并无阻拦，可以轻易的出去。
<br/>　　龙双双的速度很快，如同一缕闪电，向第二城疾驰而去。
<br/>　　不过，她的行踪，的确是暴露了，暴露给了所有城池之人。
<br/>　　因为，每个人的身上，都有所在城池的编号，这编号是由第三城的阵法之力凝聚，在头顶之上形成了一个大大的‘三’字。
<br/>　　若是善于隐匿之人，自然不会被别发现，但是龙双双不善于隐匿，一下子就暴露了出来，让所有城池都知晓，第三城有人出来了。
<br/>　　龙双双速度极快，加上第三城距离第二城本就极近，几乎是两个呼吸的时间，便接近了第二城。
<br/>　　苏莫的神念，牢牢的锁定着龙双双，暗中监察情况，城中其他人亦是如此。
<br/>　　但就在此刻，异变陡生！
<br/>　　一道璀璨的刀芒，从另一个方向出现，如同一道绚丽的极光，向龙双双暴斩而去。
