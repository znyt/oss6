　　“之后大当家的去洞房，我们提前出来，为的就是想要找机会等这僵尸现形之后，立刻动手灭了他，不想我们还是晚了一步，让大当家惨遭毒手……”
<br/>　　说到这里，小三叹了口气，做出悲伤的表情。
<br/>　　寨子里的那些之前躲藏着的土匪，听说僵尸死了之后，都聚涌了过来，差不多全在这了，听了小三这一席话，惊得说不出话来，再回想之前的情况，的确跟他说的差不多。
<br/>　　小三的一个手下说道：“是这样的，刚才在这里，是三爷跟他表哥一起对付的僵尸，才把这僵尸杀了，不然你们想想，山寨里怎么会这么碰巧来了个法师，不是三爷请的还能有谁？”
<br/>　　小三点头道：“我表哥是世外高人，不愿意与我们这些好汉打交道，刚才杀了这僵尸之后，我送他出去了。”
<br/>　　在场的土匪听他这么说，也信了八分，一个土匪皱着眉头说道：“可是你们从大当家的房间里冲出来的时候，明明叫人去把新娘子抓住来着？”
<br/>　　小三解释道：“当然，当时我是想大家一起上，擒住僵尸，让我表哥来对付，我没想到僵尸这么厉害。”
<br/>　　“不错，是这样。”这事人群中走出一个人，一身长衫，摇着折扇，冲着小三咧嘴一笑，“这件事，从头到尾，三爷是跟我商量过的，也说给大当家的听，大当家的为僵尸所迷，怒斥了我们一顿，我们只好背后做好了打算。多亏三爷提前安排，杀了这僵尸，不然我们山寨的兄弟，至少要死七七八八。”
<br/>　　说话的是铁算盘，山寨里的二当家，他说的话，大伙自然是深信不疑的。
<br/>　　小三跟铁算盘对视了一眼，两人的眼神，只有他们自己看懂了是怎么回事。
<br/>　　大伙低头看着粪坑里被烧成一团焦灰的女尸，想到这女尸之前杀人的模样——那些尸体还都在道场上没清理呢——一个个都是后背发凉，看待小三的目光也是充满了感激和尊敬，如果不是他提前谋划，真说不定道场上那些尸体里，就有一个是自己。
<br/>　　众人一起返回道场那边，清点了那些尸体，包括大当家的。小三临时站出来，吩咐大家先把这些尸体都焚烧了，免得上面有残留的尸气之类的，影响活人，也是因为这个缘故，当即为大当家的入殓，把那些唢呐乐器都拿出来，吹吹打打了一场，直接在林子里埋了，立了牌位，然后大伙一起来到聚义厅，点了四个火盆照亮，一起商量今后的大事。
<br/>　　这次会议是小三主持，先表示了一下对大当家的缅怀，然后说道：“俗话说国不可一日无君，大当家的不在了，但是山寨还在，不可一日无主，今天大伙都在这，我们要选出一个新的人，来继续带领兄弟们干活，吃香的喝辣的。”
<br/>　　大伙听他这么说，一起转头朝铁算盘看去：大当家的挂了，三当家的之前就挂了，四大金刚死了三个，这大当家的位置，自然是铁算盘来坐。
<br/>　　小三对笑着铁算盘说道：“二哥，这头把交椅，还是得你来坐。”
<br/>　　铁算盘一听，立刻摆手，说道：“不成不成，承蒙大当家的栽培提携，让我坐着第二把交椅，不过我只是个打算盘的，充其量有些小才，却无谋略，说白了我只能当个军师，不能掌舵，非是我谦虚，这第一把交椅，我绝对不能坐。”
<br/>　　大伙听铁算盘这么说，也知道他不是谦虚，于是寻找目标，有人朝小三看去，也有人朝四大金刚中的草上飞看去。
<br/>　　大凡是有名一点的土匪，一般都会取个诨名，显得自己牛逼，这草上飞就是这种诨名。
<br/>　　草上飞左顾右盼，有点跃跃欲试，在四大金刚中，他排名第四，年纪小，为人凶狠，是个敢拼敢杀的主，不过他性格太莽撞傲气，在山上人缘不好，只有鬼见愁那样的大老粗待见他，不过眼下的情况是，其余三大金刚之前都被女尸咬死了，比资历，除了铁算盘，只有草上飞有这个资格。
<br/>　　草上飞自己也对当老大充满了兴趣，目光烁烁地望着铁算盘。
<br/>　　铁算盘看了他一眼，说道：“不妥，我不是针对四金刚，只是四金刚性格刚烈，冲锋陷阵可以，但若是掌舵……容易意气用事，我们得找一个处事稳重，有眼光有谋略的人来做大哥。以我之见，三爷是不错的，这次帮山寨除掉僵尸，立下大功，之前安排善后，有理有条，是个能做大事的人，我建议还是让三爷坐这第一把交易……”
<br/>　　“什么！他上山才几天！我是最早跟着大当家打江山的！”
<br/>　　草上飞一步跳到桌子上，指着小三的鼻子，唾沫横飞地骂起来。
<br/>　　小三丝毫不动声色，叹了口气，对铁算盘说道：“四金刚说的对，我上山的日子短，恐不能服众，这头把交椅还是给四金刚坐的好。”
<br/>　　众兄弟听他这么说，突然就愣住了，山寨头把交椅，谁不想坐？况且草上飞几乎都跳到他面前去骂，居然还能忍得住，这些土匪都是没什么心机的大老粗，看到这一幕，就算是再挺草上飞的，内心也开始朝小三倾斜。
<br/>　　铁算盘看见这一幕，说道：“大家举手吧，这个最公平不过，现在革/命军都流行这个了，这样，支持小三当家的举手，支持四金刚当家的，就不用举手了，最后看看支持谁的更多，开始吧。”
<br/>　　铁算盘说完，自己先举起了手。
<br/>　　小三的几个手下，自然也高高地把手举起来。
<br/>　　众多土匪你看我我看你，有一些人，也默默地举起了手。
<br/>　　草上飞眼看着举手的人越来越多，跳上桌子，大声指着那些举手的人，一个个骂过去，让他们不要举手。
<br/>　　然而他越是这么做，举手的人就越多，直到最后，不用数，光用眼睛看都超过了半数。草上飞记得脸红脖子粗，猛地摸出盒子枪，指着小三，大声骂道：“你特么的才上山几天，你有什么资格当老大！”
<br/>(本章完)
