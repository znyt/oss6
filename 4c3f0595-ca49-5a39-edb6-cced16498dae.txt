　　第714章
<br/>　　送君千里，终须一别。在凉州逗留几日后，李玉念还是踏上了回程的路。
<br/>　　对于这位闲散王爷，程大雷表现得份外尊重。率领着麾下的文官武将亲自将李玉念送出城外。
<br/>　　“程先生留步，只等八月十五，再与程先生一叙。”李玉念道。
<br/>　　“一定，一定。如果到时候身体没问题，我一定赶往长安。”事到如今，程大雷依旧没有把话落实。
<br/>　　“诸位请回，诸位请回。”
<br/>　　李玉念冲送行的诸人一一拱手，这位王爷面对一群草莽出身的绿林豪强一点架子都没有。自然也不可能有架子，这帮山上的魔头，水里的恶徒，说不得就拔刀亮剑，将自己剁成肉馅包子。
<br/>　　诸人对李玉念也是颇有好感，一是因为对方态度随和，二是因为对方毕竟是天潢贵胄，自己也好沾沾王气。
<br/>　　最后，李玉念行到吴用面前，眼神有些暧昧。吴用的眼神也很暧昧，白鲤玉佩挂在腰上，颇有显摆的意思。
<br/>　　人多眼杂，李玉念也不好把话挑明，他拱拱手道：“吴军师，期待与你在长安一会。”
<br/>　　吴用也不敢答应他，至于去不去长安，还要看程大雷的意思。
<br/>　　李玉念登上马车，一队人摇摇远去。程大雷送了口气，送走这尊神，自己也能处理一些手边的事。
<br/>　　“大当家，咱们真要赴长安的约会么？”吴用走过来道。
<br/>　　“回头再说。”程大雷大手一挥：“回城。”
<br/>　　去不去长安城趟这次浑水，现在程大雷还没有打定主意。他现在要做的事，还是将凉州这一亩三分地收拾好。
<br/>　　浩甲钱通玉、武威白空集、金城孙别山。
<br/>　　在程当家接待李玉念的这段时间，与三家的联系一直没断，书信往来，信使来回奔波，到今天，差不多已快谈出眉目。
<br/>　　程大雷回到了城主府，问随行的刘悲：“现在他们三家什么态度？”
<br/>　　“我们提出的条件他们大多都能够答应，只有一条，他们还在考虑，就是派遣人质的问题。”刘悲。
<br/>　　李乐天想要程大雷往长安派人质，同样，程大雷也希望钱通玉能派出人质搁在凉州城。结盟这种事，如果彼此不握着对方一些把柄，终究是不太放心。
<br/>　　“人质最好能派过来，否则还是没办法对他们放心。”程大雷道：“还有其他事么？”
<br/>　　“他们想见见大当家。”刘悲道。
<br/>　　“喔……”
<br/>　　程大雷顿了顿，双方的交流已经到这种程度，委实该互相见上一面，这也能够表现双方的诚意。
<br/>　　“他们愿意来凉州城？”程大雷问。
<br/>　　“呃……好像不太愿意。”刘悲。
<br/>　　程大雷白了他一眼，心想若钱通玉真敢来凉州城，程大雷倒是要对他们说声佩服。
<br/>　　“那怎么碰面，该不会让我去他们的地盘吧。”
<br/>　　“他们好像有这个意思。”
<br/>　　“呃……”程大雷无语的摸了摸鼻子，对方不敢来凉州城，自己就敢去他们的地盘么，万一他们一拥而上，将本大当家杀了怎么办。
<br/>　　谈判进行到现在，双方都不能信任对方。程大雷要防着三家同京州勾结，打自己一个里应外合。对方也同样提防程大雷会吞掉他们。
<br/>　　“知道程当家也不会答应，他们也没敢往这边想，只是能在城外约一处地方，双方碰一碰面，谈妥一些具体的细节。”刘悲道。
<br/>　　“嗯。”程大雷点点头：“谈判地点约在哪里合适？”
<br/>　　“他们三家的想法，是约在凉州的凌云渡，那里四面开阔，谁也不担心有伏兵。如果大当家点头，我便立刻同他们回信，约定日子在那里见上一见。”
<br/>　　程大雷眼珠转了转，忽然道：“不必约在凌云渡，这次我就卖个狂，去他们的地盘闯一闯。”
<br/>　　刘悲一楞：“大当家三思呐。”
<br/>　　程大雷摇摇头：“咱们出身本就不高，说不得他们背后就骂我们是山贼。现在让他们归附一伙山贼，他们纵然同意，心里也多少也会有些不痛快。这个时候，正是显摆我们风度的时候，想要人诚心待你，首先要示人以诚。这次本当家就往龙潭虎穴闯一闯，让他们瞧瞧本当家是如何的气量。”
<br/>　　刘悲皱眉思索片刻，觉得程大雷的话也有些道理。双方都抱着怀疑，这样达成的同盟其实也十分脆弱。
<br/>　　“可万一他们动了杀心，大当家这次行动可是极其凶险。”
<br/>　　程大雷摇摇头：“看上去凶险，其实不然。只要他们敢要我的脑袋，我的大军就能压上去将他们赶尽杀绝，他们不敢玩这么大。况且，我一人一剑，他们想动手，也得看他们有没有这个本事。”
<br/>　　谈到这里，程大雷主意打定，道：“向他们送书吧，说我愿意走一遭，让他们商量，选一个地方碰面。”
<br/>　　刘悲立刻着手做这件事，有来往双方的信使，将程大雷意思送到三家。
<br/>　　浩甲钱通玉、武威白空集，金城孙别山，这三家几乎同时收到程大雷的信，一时间心里都开始打鼓，琢磨不明白程大雷的用意。
<br/>　　程大雷作为就像双方在战场相遇，各自剑拔弩张，可程大雷突然丢掉了兵器，卸掉了弓箭，毫无防备立在枪林之前。
<br/>　　他这么做，要么是料定可以吃定己方，要么就是在找死。
<br/>　　三方也在紧急商议，浩甲、金城、武威三地相距甚远，他们也是通过书信往来。
<br/>　　本来三家都在宋伯康麾下做事，其实关系也不睦。但程大雷将宋伯康取而代之，三家也不得不同舟共济。
<br/>　　就算他们三个，互相也不敢到对方地盘去。画虎画皮难画骨，知人知面不知心，谁知道对方怎么想的，万一对方动了杀心，要自己项上人头怎么办。
<br/>　　可这次程大雷竟然敢到他们的地盘，还让他们选地方，如此作为，就实在令人摸不着他心底的想法了。
<br/>　　快马在城池之间来回，一封一封信传递，恨不得马都要累死几匹。三方闹得不可开交，差点互相先打起来。
<br/>　　最后经过诸多扯皮，终于敲定在浩甲城与程大雷碰面，时间是三天后。
<br/>(本章完)
