　　嗖…嗖…
<br/>　　两道火红色风刃，疾速打来。
<br/>　　萧逸的思索，陡然停下。
<br/>　　这一次，他早有准备，故并不需要退，手中冷焱剑一扬，轻松挡下。
<br/>　　嘭…嘭…
<br/>　　火红色风刃，打在剑身之上，顷刻被拦腰劈碎。
<br/>　　不过风刃破碎，威力却并未消散。
<br/>　　别忘了，疾风烈鸟，可是同时操控风火的妖兽。
<br/>　　风刃破碎，但里头蕴含的火焰，将瞬间爆发。
<br/>　　萧逸扬剑挡于身前，一股热浪，陡然在他两旁爆发。
<br/>　　萧逸自是无碍。
<br/>　　倒是两旁地面，顷刻在这股热浪之下化作焦土，树木焚作灰烬。
<br/>　　金光险地，本就非寻常地方。
<br/>　　这里的参天之树，土壤沙泥，尽皆存在了无数年，不多不少也会吸收了些许金光力量。
<br/>　　故这里的树木、土地，尽非寻常。
<br/>　　可即便如此，这些树木土地，还是在顷刻之内化作焦烬。
<br/>　　可想而知这热浪温度之高。
<br/>　　嗖…嗖…
<br/>　　又是两道火红色风刃疾速打来。
<br/>　　萧逸如法炮制，仗剑挡之，这些火焰还伤不了他。
<br/>　　嗖…嗖…嗖…嗖…
<br/>　　疾风烈鸟的攻势，明显在增强。
<br/>　　打出的火红色风刃，数量不断激增。
<br/>　　萧逸一一挡下。
<br/>　　但不过数分钟后，火红色风刃，再不是一道道打来，而是铺天盖地，一群群密集而来。
<br/>　　萧逸脸色微变，这般密集的火红色风刃，可就不是单单仗剑挡于身前就能挡下的。
<br/>　　“喝。”萧逸低喝一声，一剑横扫而出。
<br/>　　剑身之上，冷白色剑芒陡然激增，一缕缕剑气，狂风暴雨般打出。
<br/>　　剑气，显然逊色于这些火红色风刃。
<br/>　　毕竟，萧逸的修为仅有圣王境六重。
<br/>　　而这头疾风烈鸟，可是实打实的圣皇境四重妖兽。
<br/>　　二者的元力，根本不是一个层次。
<br/>　　剑气与这些火红色风刃交锋，往往数道剑气才能抵消掉一道火红色风刃。
<br/>　　数秒过后。
<br/>　　剑气悉数溃散，火红色风刃被劈碎大半。
<br/>　　但剩下的一小半，则以极快的速度，越过萧逸的剑影防御，重重地打在了萧逸身上。
<br/>　　嘭…嘭…
<br/>　　一连数道剧烈爆响。
<br/>　　萧逸瞬间被轰退数十米。
<br/>　　胸膛、双手上的衣衫，已然被焚烧破烂。
<br/>　　胸膛、双手上，明显出现了一道道剧烈的灼烧痕迹。
<br/>　　一股强烈的灼烧疼痛感，顷刻涌现。
<br/>　　萧逸微微咬了咬牙，脸色轻变，手中握着的冷焱剑，紧了紧。
<br/>　　“圣皇境八重巅峰实力。”萧逸脸色凝重地看着远处高昂扬着头颅的疾风烈鸟。
<br/>　　疾风烈鸟，本就是圣皇境四重妖兽。
<br/>　　吞食金光后，实力直达圣皇境八重巅峰，隐隐有接近圣皇境九重之势。
<br/>　　不得不说，这里的妖兽的凶猛程度，以及狡猾谨慎程度，远超萧逸以往见过的妖兽。
<br/>　　这头疾风烈鸟，即便本就是实力极强的妖兽。
<br/>　　但在对上萧逸这个年轻的人类武者时，还是吞食了金光，达到实力极致后再发动攻击。
<br/>　　嗖…嗖…嗖…
<br/>　　这时，疾风烈鸟的攻击，仍旧源源不断。
<br/>　　萧逸仍旧仗剑挡之，并未着急。
<br/>　　八万星光之力加身下，他的战力，并不亚于圣皇境八重。
<br/>　　这些密集而威力强横的风刃，对他而言，绝对是一次极好的历练。
<br/>　　萧逸一手握剑，剑影重重。
<br/>　　脚步，则玄妙而疾速地动弹着。
<br/>　　或挡，或避。
<br/>　　身影，宛若一道道幻影，不断游走穿梭其中。
<br/>　　整整半个时辰后，这些密集的火红色风刃，未伤他分毫。
<br/>　　看似轻松穿梭其中，实则，对萧逸的反应力、速度、步伐、剑之操控等等能力的要求，缺一不可。
<br/>　　时间渐渐过去，萧逸的身影，愈发变得玄妙。
<br/>　　速度，愈发变得快。
<br/>　　穿梭于密集火红色风刃，亦愈发轻松。
<br/>　　一个时辰后，萧逸嘴角咧过一道笑意。
<br/>　　穿梭火红色风刃中，宛若闲庭信步，简单而自然。
<br/>　　轰…
<br/>　　忽然，高空中，道道星光降临。
<br/>　　八万星光齐至，幻化星河。
<br/>　　萧逸疾速跃动的身影，已然停下。
<br/>　　取而代之的，是一剑重重劈出。
<br/>　　星光长河随剑而出，碾压而过。
<br/>　　沿途，一切火红色风刃溃散不堪。
<br/>　　“该结束战斗了。”萧逸淡淡一笑。
<br/>　　一手握剑，身影一闪，直朝疾风烈鸟而去。
<br/>　　啾…
<br/>　　一声鸟鸣，忽然响彻高空。
<br/>　　疾风烈鸟那高傲的头颅，冷眼对着萧逸。
<br/>　　啾鸣声中，显然带着一丝不屑。
<br/>　　呼…呼…
<br/>　　疾风烈鸟那双火红色的羽翼，已然张开，随后剧烈抖动。
<br/>　　几乎是一瞬间，一缕缕剧烈罡风顷刻打出。
<br/>　　罡风，渐成暴风之势。
<br/>　　萧逸的脚步，顷刻被挡下。
<br/>　　“好强的风力。”萧逸脸色一变。
<br/>　　此刻的他，竟觉置身峡谷之内。
<br/>　　峡谷之中，无数罡风化作风暴而来。
<br/>　　他的身影、脚步，竟在这股罡风之下瞬间止步，甚至不断后退。
<br/>　　锵…
<br/>　　一声清脆剑鸣。
<br/>　　萧逸一剑挡之，抵消了身前罡风。
<br/>　　但身旁两侧，仍旧罡风环绕，凌厉至极。
<br/>　　恰在此时，疾风烈鸟一双火红色羽翼之上，道道火焰迸发而出。
<br/>　　火焰，并不多强。
<br/>　　但在这股剧烈罡风的吹袭下，陡然成燎原之势，幻化火海。
<br/>　　一大片火海，随风而出，直朝萧逸吞噬而来。
<br/>　　“不好。”萧逸脸色一变，连忙一剑重重劈出。
<br/>　　一道巨大的星光长河，宛若惊涛骇浪，层层而出。
<br/>　　一开始，星光长河与火海的交锋，呈僵持之势。
<br/>　　但待得星光长河连绵不尽，一层接一层轰出后，火海瞬间被轰散。
<br/>　　一道道星光长河，以摧枯拉朽之势，一路碾压。
<br/>　　所过之处，罡风碎尽，火海散尽。
<br/>　　疾风烈鸟高昂的头颅上，终于闪过一丝恐慌。
<br/>　　火红色双翼的拍打，已然结束；取而代之的，是双翼一震，试图御空飞离。
<br/>　　然而，萧逸的剑，何等之快。
<br/>　　星光长河的速度，又何等之惊人。
<br/>　　在疾风烈鸟飞起之前，数道星光长河，已然将其吞噬。
<br/>　　数道星光长河之内，蕴含无数星光之力，自主化作星光漩涡。
<br/>　　漩涡之内，疾风烈鸟试图抽身而去，飞跃而起，却始终挣脱不了漩涡的吸摄。
<br/>　　不过片刻，漩涡旋转不停，里头绞杀之力，骇人至极。
<br/>　　数秒之后，漩涡停下，星光消散。
<br/>　　里头，疾风烈鸟已然身死。
<br/>　　“呼。”萧逸轻呼出一口气，大手一挥，收了内丹以及妖兽精血。
