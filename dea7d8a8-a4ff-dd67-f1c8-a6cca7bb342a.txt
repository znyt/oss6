　　按照造化大陆上的修炼体系，修行之人修炼造化之力，一共又可以分为八个境界：
<br/>　　胎动、入窍、丹成、道基、破神、洞虚、不朽、造化！
<br/>　　造化之力，具有生化万物的能力，所以这个世界上的修行者，战斗力普遍不怎么高。
<br/>　　而在造化大陆之上，有两种最为显赫的职业，分别是炼药师和傀儡师。
<br/>　　炼药师和修真界中炼丹师比较类似，只不过这个世界并没有丹术的存在，炼药师可以炼制出各种媲美宝丹的灵药。
<br/>　　至于傀儡师，则是运用造化之力，将灵木炼制成各种可以战斗的傀儡，来代替主人作战，这也是造化大陆上的主流战斗方式！
<br/>　　对造化大陆的修炼体系有了一番了解之后，叶尘又向小兰询问了一些基本的修炼方法。
<br/>　　在造化大陆，人人修行，虽然小兰只是一个小小的婢女，但也懂得一些基本的修炼之法。
<br/>　　叶尘研究了一番之后，很快就掌握了修炼的基本原理。
<br/>　　“少爷，我只会这些最基本的吐纳之术，要是老爷还在就好了，他一定会传您这个世上最好的修炼法门……”
<br/>　　小兰将自己知道的说完之后，一脸的惋惜道。
<br/>　　叶尘微微一笑，
<br/>　　“小兰，你要记住，越是最简单的修炼之法，就越能直达本源！”
<br/>　　小兰似懂非懂，不由得歪着小脑袋，眼眸中透着一抹疑惑之色，她总觉着自己的这位小少爷，似乎变得和之前不太一样了。
<br/>　　叶尘没有再继续多说，当即盘腿坐在床上，开始按照小兰所讲的呼吸吐纳之法，然后结合自己的领悟和理解，开始摸索着修炼了起来。
<br/>　　要说他现在的这具身体的原主人，的确是够废柴的，不但肉身孱弱不堪，修为也仅仅只有胎动二重，竟然比自己的婢女小兰还不如！
<br/>　　而且造化之力本来就不擅长战斗，所以这具身体的体能，可能连地球上的同龄少年都还不如，在造化大陆上，绝对是不可多得的超级废柴！
<br/>　　这也难怪为何，这个叶枫的父亲一死，叶家分支一脉就立即将他扫地出门。
<br/>　　这样一个毫无任何前途的废柴，的确没有任何的利用价值。
<br/>　　好在如今掌控这具身体的，已经是叶尘的意识，他自然不会让这种情况再持续下去。
<br/>　　叶尘摸索了一小会，很快便掌握了修炼的基本法门，并且根据自己的领悟，做了一番改善，大大提高了修炼的速度。
<br/>　　呼呼呼！
<br/>　　周围的天地灵气，顿时犹如潮水般，从四面八方朝叶尘的身体汇聚而来，最终通过经脉炼化，转化成一股精纯的能量，流入丹田之中……
<br/>　　“这就是造化之力吗？倒是和万古长青珠上面的能量有些类似……”
<br/>　　叶尘在心中暗暗嘀咕了一句之后，当即将这股能量，运转到自己身上的伤口处。
<br/>　　滋滋滋~~
<br/>　　叶尘身上的伤势，在造化之力的滋养之下，果然开始飞速的愈合，不一会便已经好了七七八八。
<br/>　　大约一个小时之后，叶尘已经将修为，提升到了胎动三重境界。
<br/>　　一旁的小兰，看到叶尘的气息提升的如此之快，一双大眼睛早已经瞪得老大，露出满脸的不可思议之色，
<br/>　　“少爷，你……你突破了？”
<br/>　　叶尘笑着点了点头，并没有做过多的解释。
<br/>　　虽然修炼方法有所不同，但是总的来说，大道相通，殊途同归，对于拥有八百多年修炼经验的叶尘来说，掌握这个世界的修炼法门，自然是一件很简单的事情。
<br/>　　不过，让他感到郁闷的是，这个似乎只能修炼造化之力，因此很多强大功法和武技，根本无法使用，所以他此时的战斗力，甚至可能还不如修真界炼气一重境界的修士！
<br/>　　“对了！我前世的那些功法之中，有没有什么功法，是以造化之力为根基的呢？”
<br/>　　这道念头冒出之后，叶尘当即在脑海之中，经过了一番筛选，别说，还真被他找到了一门，符合条件的功法。
<br/>　　“青帝长生体！帝级炼体之术！以造化之力为基础，淬炼肉身，修炼至最高境界，肉身几近不死不灭……”
<br/>　　“就是它了！”
<br/>　　叶尘将这门炼体之术，仔细研究了一番，又根据这个世界的情况情况做了一番改良，发现真的是无比适合，心中顿时大喜过望。
<br/>　　然而，就在叶尘准备马上着手，开始修炼青帝长生体的时候，
<br/>　　门外忽然传来了一阵脚步声，
<br/>　　紧接着，
<br/>　　砰！
<br/>　　房间的大门，被人从外面狠狠一脚踹开！
<br/>　　继而，
<br/>　　一名满脸倨傲之色的锦衣少年，带着三名随从，从外面大摇大摆的走了进来。
<br/>　　小兰看到这几人出现，顿时吓得浑身一颤，不由的仅仅抓住叶尘的胳膊，小脸煞白无比，低声冲着叶尘，道：
<br/>　　“少爷，成少爷来了，这可怎么办？”
<br/>　　“哦？”
<br/>　　叶尘闻言，不由得抬起头，在那锦衣少年的身上扫视了一眼，嘴角微微一扬，
<br/>　　“原来就是他把我打伤的呀！”
<br/>　　自言自语了一句，叶尘双手在床上一按，便直接从床上跃下，稳稳的站在了地面之上。
<br/>　　“嗯？”
<br/>　　那锦衣少年看到叶尘居然站了起来，明显微微一愣，继续露出一抹嘲弄之色，道：
<br/>　　“可以啊！叶枫，看来本少爷昨天是下手太轻了，你居然这么快就能下床走路了！”
<br/>　　“不过这样也好！家主有令，从今往后，你不再是我们叶家的人！既然你现在可以下地走路了，那就赶紧收拾一下东西，带着你老爹的尸体，滚出叶家吧！”
<br/>　　叶尘听到这话，眉头不由得微微一皱，
<br/>　　他能够感觉到，眼前这个锦衣少年，修为要高出他不少，而他的那三位随从，应该也全都在他之上！
<br/>　　虽然，叶尘凭借着丰富的战斗经验，和精妙的拳脚功夫，未必就会输给他们，可是对方背后还有一个叶家，以他现在的实力，恐怕暂时还没有撼动的能力。
<br/>　　一念及此，叶尘决定先暂时忍耐，等到他再提升一些实力之后，再为叶枫讨还这个公道。
<br/>　　然而，
<br/>　　就在叶尘拉着小兰的手，准备离开的时候，
<br/>　　“等等，本公子是让你滚！可不包括这个奴婢！”
<br/>　　锦衣少年指了指小兰，似笑非笑的开口道。
