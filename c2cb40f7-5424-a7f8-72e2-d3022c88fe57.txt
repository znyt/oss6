　　“林峰小兄弟，我们刚接到情报，在距离我们这里大概三十里地的地方，有一位陈姓的病毒研究专家和几位物理学家需要我们解救，与他一行的还有十五个当地派出所的警察和三名武警战士，这几人也是好不容易联系上我们基地，我准备派人去营救，怕是要麻烦林峰小兄弟了。”陆勇开门见山道。
<br/>　　“我就说你给我打电话准没好事。”林峰无语了。
<br/>　　“哈哈，林峰小兄弟，这也是陆将军器重你啊。必须要有你去才肯放心，这个陈博士是病毒领域里面全国赫赫有名的专家。”周华道。
<br/>　　陆勇也是点头道：“林峰小兄弟，这一次，任务非常重要，这个陈博士，对我们科研部来说有很大的战略意义，除了陈博士之外，其余几个博士也都是人才，我们必须要救援，当然了，林峰小兄弟不是我军部之人，这一次若是答应帮忙的话，以后你老龙口的煤矿，我军部高于市场价两倍全部收了！”
<br/>　　林峰闻言呵呵一笑这才差不多。
<br/>　　“除了林峰小兄弟外，还有周华也会陪你一起去，陈博士被困在了汉市，路途上还是比较遥远的。我让周华带三个团的兵力和你们一起。”陆勇道。
<br/>　　林峰没想到九阶的周华也去，这陆勇对这一次营救任务居然看的这么重，这个陈博士到底何许人也。
<br/>　　“事不宜迟，林峰小兄弟，我们这就出发吧。”
<br/>　　周华拍了拍林峰肩膀然后就去准备了。
<br/>　　一刻钟后。
<br/>　　基地门口。
<br/>　　三个团的兵力已经整装待发。
<br/>　　“林峰战神。”
<br/>　　让林峰意外的是，林峰和周华来时，这么多人其中居然不少人还认识林峰。
<br/>　　“林峰小兄弟在我们军部之中，威望可不低啊。”周华笑道。“这些都是参加过先遣部队的。”
<br/>　　一群人清点了装备，没有任何墨迹，上了车直接朝汉城开去。
<br/>　　汉城是个四线城市，距离江城不堵车的话，要两个小时。
<br/>　　出奇的是，高速公路上居然不堵车，唯独出城的这一段上面可以看到一些报废的车辆，离开了江城之后，一路的畅通无阻。
<br/>　　想想也是，末世来的如此突然就算有人想着要开车走，估计也都堵在市内，根本没有机会可以出城。
<br/>　　高速公路上偶然会出现几头变异野猫之类的怪物袭击，但在三个团的重火力之下，全部被打成了马蜂窝。
<br/>　　“林峰小兄弟，你上次说的那个内力的事情。”周华忽然开口道。
<br/>　　林峰瞥了对方一眼，“怎么了，有屁就放。”
<br/>　　“算了，没什么。”
<br/>　　周华尴尬的挠了挠头，他都三十多岁的汉子了，可是看到林峰一个高中生居然有些难以启齿的感觉。
<br/>　　一群人没有说话，在两个半小时后。
<br/>　　众人抵达了汉城。
<br/>　　隔着很远就可以看到一个安静的城市。
<br/>　　唯独几道怒吼时不时的响起，一些大楼房也都被不知名的绿色植物所攀爬，看上去就好像来到了坐落于原始森林之中的现代都市一样。
<br/>　　“这些都是什么植物，怎么可以爬这么高。”
<br/>　　“不会是爬山虎吧。”
<br/>　　“不太像，爬山虎的藤蔓没有这么粗啊。”
<br/>　　“白痴，变异爬山虎呗。”
<br/>　　“言之有理。”
<br/>　　周华扫了眼，“留下五百兵力原地设卡，布置防线，保护运输车辆。”
<br/>　　林峰直接传令下去，让手下人开始突进了。
<br/>　　三个团剩下的兵力分为若干个突击小队朝汉市第一人名医院靠拢过去，陈博士和若干博士就被困在那个地方。
<br/>　　吼吼吼。
<br/>　　一路上，丧尸自然是无数。
<br/>　　数百只丧尸扎堆的冲过来，但在强大火力下全部倒下。
<br/>　　呼！
<br/>　　几个手持火焰喷射器的士兵直接发射了火焰，将那些还没死的丧尸全部烧成了灰炭。
<br/>　　不过丧尸实在太多，队伍推进的速度也十分的缓慢。
<br/>　　刷刷刷。
<br/>　　也就在这个时候。
<br/>　　忽然水泥马路直接破裂，一根根成年男人手臂粗细的藤蔓从地下刺破了出来，直接刺破了十几个士兵的身体。
<br/>　　“什么东西？”
<br/>　　“该死。”
<br/>　　“射击！”
<br/>　　周遭士兵立即全部瞄准那藤蔓，甚至还有勇猛的士兵拿着匕首就去刺。
<br/>　　但下一刻。
<br/>　　砰砰砰。
<br/>　　更多的藤蔓钻出了地面，直接把他们全部都缠起来，然后从他们嘴巴之中钻进去。不过区区三秒钟，这些被藤蔓缠住的人都成为了人干。一瞬间不下于七八十个军人全部死于这个藤蔓之下。

<br/>　　周华立即对跟在他身边的一个士兵道。

<br/>　　林峰拔出斩灵刀就是冲了过去，几根枝条从地底下冲了出来，几十个士兵全部都疯狂射击，虽然打烂了许多，但依然还有源源不断的树枝冲出地面，就好像变成了一片树木组成的监狱一样。

<br/>　　周华立即大喊道，“喷火兵上！”
<br/>　　几个拿着火焰喷射器的士兵立即上前去烧，顿时直接烧掉了几十根树枝。但没多久他们自己脚下也全部钻出藤蔓，这东西就好像是战矛一样，直接从他们的菊花之中刺进去，然后从天灵盖上面穿出来，瞬间就好像羊肉串一样，几个喷火兵直接被秒杀。

<br/>　　林峰眯着眼，他也没有隔岸观火，拔出斩灵刀就是冲了过去，帮着一群人对付这些触手一样的鬼玩意。
<br/>　　刷。

<br/>　　“林峰战神，我看到他的本体了，在距离这里大概三条街的一个楼房内，那里有巨大的生命气息波动，应该是这个树怪的本体！”
