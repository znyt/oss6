　　半个月很快过去，苏重一直宅在山上，偶尔下山看一下院落重建进度。
<br/>　　关景福手下团队经验丰富，技术娴熟。小院几乎一天一个样。
<br/>　　之前说半个月还是往多了说，也没见怎么加班。现在竟已经大体完工。在过个两三天，等房间大稍微晾一晾，苏重就可以着手搬迁了。
<br/>　　端着一盘干煸豆角走出厨房，苏重就看到坐在石桌旁，拿猫粮逗弄胖猫的孙建中。
<br/>　　“我说孙老头，你怎么又来蹭饭？”
<br/>　　“装修费我都给你出了，管我一顿饭怎么了。”这老头理直气壮。
<br/>　　“这是一顿吗，是谁天天往我这山上跑，你心里就没点儿数吗？”苏重放下盘子斜睨着对方。
<br/>　　“不要这么小气，你看我连装修费都给你出了，你就管我几顿饭，你赚大了我跟你说。”
<br/>　　“装修费是我凭本事挣来的，赚的再大那也是我的！”
<br/>　　“也不知道是谁用特制蚯蚓作弊，还凭本事赚来的，有本事咱们用同样的蚯蚓比一次。”孙老头两眼放光。
<br/>　　苏重翻了个白眼：“吃你的去吧。”
<br/>　　他一钓鱼菜鸟，哪里是这个老小子的对手。
<br/>　　孙建中也不傻，那天苏重走后，专门扒拉了点儿苏重剩下的蚯蚓钓鱼。结果显而易见。
<br/>　　然后他就缠上苏重了，想着法子搞到特制蚯蚓，搞不到也赖着不走蹭吃蹭喝。反正怎么算，都是他赚到了。
<br/>　　“你那小院快修好了，是不是请一下亲朋好友的聚一聚？”孙老头嘴里塞的满满的，口齿不清的问。
<br/>　　“过两天，等我把那个院落收拾出来，就请人过来。”苏重早有打算。
<br/>　　“到时候你可不能留一手，一定要搞一顿大餐！”孙老头眼睛一亮。
<br/>　　“我说要请你了吗？你激动什么呢。”
<br/>　　“没事，不用你请，我自己去。”为了美食，这老头也是不要脸了。
<br/>　　苏重说等几天，不只是为了等待房间晾干，而是要给院落布置阵法。
<br/>　　从山上挖了一些植物，又通过关景福购买了一些上佳盆景，苏重开始给院子动工。
<br/>　　每一株植物的朝向，每一块石头的位置，都经过苏重精心演算后才摆放好。
<br/>　　苏重还亲自动手，从山上砍了几棵树，做了一个凉亭。
<br/>　　本来潮湿的木头是不能直接用的，但苏重阵法神奇。等他建造好，阵法作用发动，木头很快就失去水分干燥起来，不怕蛀虫也没有变形。连带着潮湿的房屋，也快速干燥起来。
<br/>　　然后再整个阵法的运作下，小院内草木精气会逐渐浓郁，很快就会变得清凉静谧起来。
<br/>　　布置完一切，已经是三天后。苏重开始着手准备大餐，他要请几个朋友过来聚一聚。
<br/>　　苏重不是本地人，来这里一年多，朋友也就孙建中姜长飞苗村长几人。
<br/>　　打电话给他们几个，定好晚上聚会，苏重开始准备。
<br/>　　食材他也不用自己去买，直接给孙老头打电话，让他给送过来。
<br/>　　孙老头吃饭从不将就。他专门开了个农副产品公司，除了正常运营，主要就是为他自己培育安全干净食材。每年他还要去视察好几次。就为了吃的美味放心。
<br/>　　下午六点，天色正明亮，孙建中、关景福、姜长飞、苗老村长几人不约而同走进苏重小院。
<br/>　　新建的凉亭下，已经摆放好了圆桌。苏重招呼几人坐下喝茶：“你们先坐这里聊着，我去做饭。”
<br/>　　“赶紧去吧，这里用不到你。”孙老头大咧咧，和苏重混熟了，他也没把自己当外人。
<br/>　　苏重笑着钻进厨房。
<br/>　　“这小院整得还不错啊，你们有没有觉得，进来这个小院，就和走进空调房似的，格外凉爽舒适。”姜长飞是第一次来这个小院，一下就感觉出不同。
<br/>　　外面是炎炎夏日午后，正是燥热难挨的时候，小院里却好似装了空调，不仅空气清新，而且温度凉爽舒适，让人心情舒畅。
<br/>　　“还真是那么回事？关胖子，这是你们公司设计的？”孙建中发现周围景致自然简单，排布却有些规律。
<br/>　　“我要有那本事，哪还能给您打工啊。”关景福苦笑。
<br/>　　“大概是苗家村环境好。”姜长飞随口道。
<br/>　　苗老村长笑而不语。环境好？自己家怎么就没这功效？
<br/>　　几人默契不在谈论，知道这个小院子里的布置有些神奇，虽然心里好奇，却都心照不宣的停止深究。
<br/>　　孙老头想的更深，明白这肯定是苏重手笔。他可是知道苏重妖孽，短短时间就精通建筑设计。
<br/>　　起草高楼大厦设计可能力有未逮，但这种精致小院、度假别墅反而更有心得。
<br/>　　“找时间一定要向这小子搞明白，让他也给我那别墅弄一份独特设计。”他见多识广，知道这种天然环境，对身体极其有好处。
<br/>　　几人正说着各自趣闻，一股诱人香味突然从屋内传来。
<br/>　　“这是什么东西，这么香？”关景福不自觉的吞咽了一口唾沫。
<br/>　　“我去催一催，这怎么还不上菜，不能让我们这些客人久等不是！”孙老头馋的要命，找了个借口，就跑进厨房。
<br/>　　站在门口，就看到苏重左右开弓，一手一个炒锅的不断翻腾，竟同时做两个菜！
<br/>　　他看了一会儿，惊奇的发现，虽然同时做两个菜，苏重动作却丝毫不乱，而且每一个步骤行云流水，甚至有一股美感！
<br/>　　“可以啊！你这技术，就是去做国宴都行啦！”孙老头拼命吞咽唾沫，不由自主的感叹。
<br/>　　“那不行，我顶多也就是占了个熟手的光。不能和国宴大师们比。他们的菜兼具美味与艺术，是整个国家的脸面，我这几下庄家把式可上不了台面。”苏重不认为自己多厉害，他只是占了五六妙法的便宜。
<br/>　　妙法让他身体素质提升，有力量操控两个炒锅；还让他敏锐五感，使他动作有条不紊无有错乱。和那些经年累月磨炼出来的国宴大师们比，自己这只不过是作弊。
<br/>　　“好了，开始上菜！”苏重手脚麻利，一次制作两个菜，八菜一汤只剩最后两个。关火出锅，今天的菜就全做好了。
<br/>　　孙老头不用苏重招呼，就颠儿颠儿的帮忙端菜，他早就忍不住了。
<br/>　　饭菜一上桌，众人就忍不住。不用苏重招呼就筷子不停，嘴里塞的满满，话都来不及说，只顾着闷头大吃。
<br/>　　客人满意饭菜，苏重这个主人也就心满意足。他依旧慢条斯理的吃着。
<br/>　　胖猫早就等不及了，前腿搭在苏重身上，瞪着圆溜溜眼睛，脖子伸的老长，不住打量餐桌上的菜盘。
<br/>　　“小苏子，赶紧给爷准备传膳。”
<br/>　　“少不了你的！”拍了拍胖猫脑袋，苏重早有准备，拿出干净碗碟，每样菜肴都夹出来放进里面，最后再配上一碗鲜美鱼汤，放在凉亭边缘木质长椅上，让胖猫享用。
<br/>　　寂静夜晚，偶尔虫鸣，几个朋友坐在凉亭下吃饭喝酒，苏重背后感觉到了久违的放松。
