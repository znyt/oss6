<br/>　　“轰轰轰！”
<br/>　　伴随着一阵惊涛骇浪，巨大的浪涛扑面而来，断浪和剑晨几人都有些意外，神将等人对此毫不在意。
<br/>　　毕竟，他们可都是天境强者，区区浪涛对他们，根本造不成威胁。

<br/>　　很快，猩红的血液就在附近海域弥漫，天门弟子的尸体也都浮出水面。
<br/>　　帝释天发现，跟随自己前来的天门弟子，损失惨重，神色开始愤怒起来：“真是一群没用的废物，还得本座亲自出手。”

<br/>　　当海水拍在保护屏障上，不断的蒸发，在屏障的保护下，船只破开巨浪，快速接近神龙岛。

<br/>　　看来，凤血带给他的，不仅仅是永恒的生命，连圣心诀所修炼出的灵气，都产生了异变，比寻常人的灵气，更加的狂暴，威力强大数倍。
<br/>　　慕清霄进入天门以来，还是第一次见帝释天出手，刚刚的气息，比半年前倒是进步了一点，可距离金丹九层巅峰，依旧遥遥无期。
<br/>　　在帝释天出手后，发现敌人前所未有的强大，海族族长也不是蠢货，当下不敢在做停留，一声令下，命族中弟子带领伤员，快速撤退。
<br/>　　否则，下场就是全军覆没。
<br/>　　海族弟子全部撤退，再也没有人阻拦去路，船只顺利抵达神龙岛，登陆岛屿后，帝释天命令天门弟子，追击海族中人，大开杀戒。

<br/>　　早在半年前，他加入天门前，就已经在霍家庄和步惊云等人商量好，半年后的惊瑞之期，必然是帝释天的死期。
<br/>　　步惊云语气淡然道：“帝释天为了屠龙，大开杀戒，看来海族在劫难逃。”
<br/>　　剑晨神色愤怒的瞥了眼身后的皇影，以及右侧的慕清霄：“无论如何，我都不会帮他大开杀戒。”
<br/>　　慕清霄倒是毫不在意，剑晨并不知道他和步惊云有所联系。

<br/>　　若是慕清霄想，便能轻而易举的找到海族的藏身地，毕竟神识这种东西，可跟开外挂一样。

<br/>　　毕竟，这种言听计从的狗，可不好找。
<br/>　　屠龙在即，在场所有人，各有各的心思，帝释天活了上千年，自然知道断浪几人不怀好意，但他也并不在意。
<br/>　　只有他自己知道，到达这种境界后，实力有多恐怖，哪怕在场所有人在多上一倍，也不是自己的对手。
