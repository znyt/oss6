陆子羡了解的点了一下头，“那你知道曼陀罗花么，曼陀罗花，传闻开在生死两界的彼岸花，剧毒无比，这世上都很难寻到一株。”

<br/>        “夕绾想用曼陀罗花来治愈你，所以她在试毒炼药。”

<br/>
<br/>        “我说夕绾在试毒炼药，她第一次试毒就是你在药房里揍我的那晚，她昏倒在了我的怀里，今天是她二次试毒，花毒导致她精神错乱，她将我误认成了你，她求你不要生气了，求你抱抱。”
<br/>
<br/>        陆寒霆的心脏被狠狠的攥了一下，突然之间连呼吸都变得疼痛了起来。
<br/>
<br/>        陆子羡拿起了那张纸，纸上是夏夕绾写下来的方程式，“很遗憾，二次试毒差了一步，还是失败了，我估计夕绾很快就会第三次试毒，陆寒霆，夏夕绾为了你在拿自己的命在博，她为了救你连自己的命都不要了！”
<br/>
<br/>        陆寒霆松开了陆子羡，向后退了几步，一开始是一道声音在告诉他，夏夕绾在试毒炼药，然后是无数道声音在告诉他，夏夕绾为了救他连自己的命都不要了。
<br/>
<br/>        他不知道…她竟然去试曼陀罗花毒！
<br/>
<br/>        她是不是…疯了？
<br/>
<br/>        “陆寒霆，夏夕绾这一身珍贵的血液是她妈咪留给她的，她才20岁，刚回海城，人生刚起点的时候还没有发挥血液价值的时候，她就为了你废掉了，我不知道她会因此而错过些什么，我只知道她的血液已经变平凡了，所以，她绝对承受不了第三次试毒，陆寒霆，在她为了你第三次试毒之前把她用力的推开吧，让她好好活着！”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾一直被关在沐浴间里，她不停的敲门，但是没有人给她开，她不知道那两个男人在外面说了些什么，做了些什么，有没有打架，还有陆子羡有没有乱说话，试毒炼药的事情绝对不能告诉陆先生。
<br/>
<br/>        刚才才试过毒，虽然意识清醒了，但是她的脸色苍白的像张纸，夏夕绾纤柔的身体摇晃了两下，眼见着就要摔跤。
<br/>
<br/>        她迅速用手扣住了盥洗台，勉强站稳，这时视线里一黑，什么都看不到了。
<br/>
<br/>        纤长的羽捷颤了两下，她迅速用力的闭上眼，再睁开，视线又恢复光明了，好像刚才只是错觉。
<br/>
<br/>        这时沐浴间的门突然打开了，陆寒霆走了进来。
<br/>
<br/>        夏夕绾回眸，身材高大英挺的男人逆着光伫立着，俊脸上的表情讳莫如深，看不真切，“陆先生…”
<br/>
<br/>        陆寒霆走进来，脱下身上的黑色大衣披在了她的肩上，将纤柔莹润的她都裹在自己的大衣里，然后将她打横抱起。
<br/>
<br/>        他一路抱着她出了酒店，上了劳斯莱斯幻影的副驾驶座。
