        &lt;h3&gt;第三卷第二十二章  郝景文回家乡拓展业务&lt;/h3&gt;
<br/>
<br/>        郝景文全国各地到处飞，他比寒伯安和江俞轩小了几岁，但是这不妨碍他们做兄弟。家里是茶叶世家，父亲在前几年研发了一种新茶，口感醇香，一上市就受到了好评。
<br/>
<br/>        如今，茶叶生意遍布全国，而且还卖到了国外，他喜欢和寒伯安和江俞轩一起做事情，他不想依靠家里的资源，倒不是他不喜欢茶叶这个产业，无论是哪个行业都是值得尊敬和鼓励的。
<br/>
<br/>        他只是想开拓一片新的天地，用他学到的来改变他父亲给他的既定人生。
<br/>
<br/>        父亲的茶叶已经红红火火，而且能够买得起他们茶叶的人都是非富即贵，一般人是喝不起的，当然，他们的茶叶本身就是走的高端路线。
<br/>
<br/>        他回到家里，看着自家的茶园和父亲的产业也是充满了自豪，他的父亲不像一般的人，只会守着那片茶山，日出而作，日落而息。他是一个大胆而喜欢创新的人，所以他的事业红红火火，一帆风顺，他们现在已经是整个地区的大佬。无论哪种茶叶，好的茶叶都在他们的手中。
<br/>
<br/>        父子俩个人品着自家的茶开启了聊天模式，他和他的父亲就像是一对老朋友。
<br/>
<br/>        “爸，你这几年不错啊。”
<br/>
<br/>        “哈哈，那是当然，你也不看看我是谁。”
<br/>
<br/>        “这些年也是辛苦你了，我都没有帮你什么。”
<br/>
<br/>        “要你帮什么，你老爸我现在还年轻着呢，你尽管在外面折腾吧，万一折腾不下去了，就回来。”
<br/>
<br/>        “谢谢老爸，不过你放心，我现在和寒大哥，还有轩轩一起做事情呢，外面另外开辟了一片新天地，如果成功了，我们就了不得了。”
<br/>
<br/>        “哈哈，那你们就尽情的折腾吧。”郝父是一个通情达理的人，他并不逼着自己的孩子干什么，他放手让他出去闯。
<br/>
<br/>        “谢谢老爸支持。”郝景文真的很感谢父亲给予他的支持，只要他想干什么，父亲从来都没有阻止过他，反而支持他各种折腾，从大学一直到现在。
<br/>
<br/>        “如果缺资金了就告诉我。”
<br/>
<br/>        “爸，我们现在还不怎么缺资金。”他们三个人都有工资的，开那个公司并没有拿家里的钱，都是他们自己的工资加上平时的私房钱，别说男孩子没有私房钱，他们家境都好，私房钱不是一般的多，拿出来都是好几个中等家庭的几年的费用。
<br/>
<br/>        “好，那就祝你们成功。”年轻人有年轻人的想法，爱折腾是好事情，如果郝爸爸当初不爱折腾，那他估计只会吃着老祖先留下的产业，如今也不会成为业界大佬，所以郝爸爸是喜欢儿子到处折腾的。
<br/>
<br/>        看着儿子那信心满满的样子，他充满了欢喜，儿子长大了。
<br/>
<br/>        “爸，我们现在做的是运输业，只是面对的群体不同，随着社会的发展，网购的人会越来越多了，我们针对的就是喜欢网购的这一群体。”
<br/>
<br/>        “我也听说过啊，不过现在竞争也激烈了吧？”
<br/>
<br/>        “现在是百花齐放百家争鸣，只是看谁能够坚持到最后，谁能跟上社会的步伐。”他们也是意识到了行业竞争的残酷性，所以才和陈子昂联手。
<br/>
<br/>        传统的行业里，有些人是想不到他们会和信息化挂上钩。目前他们走在了前端，只是要看后期发展的速度和结果。
<br/>
<br/>        在南方也撅起了几个行业精英，竞争越激烈，也就证明这个行业越有发展前景，好的是他们三个都有一点背景，家族也是不差。
<br/>
<br/>        陈子昂的app软件虽然只是初期的各个点对点的连接，目前只是配送货物，但是后期不能不说还有更大的需求和用途。
<br/>
<br/>        “哈哈，现在是你们年轻人的天下啊。小伙子加油干，我们这些老家伙怕是跟不上你们的脚步了。”
<br/>
<br/>        “爸，你能进行改良和研制，怎么能说跟不上社会的发展呢，要不是你有开拓的精神，我怕也会是个束手束脚的人了。”郝景文对父亲是充满敬佩的，他们俩个人沟通起来真的很好，父亲开明，似乎就是一个知心朋友，像别的家长那种权威性在他们家里根本不存在。
<br/>
<br/>        读书的时候父亲谆谆教导，上班的时候他去了寒伯安的公司，父亲也是支持和鼓励他的，如今他们自己出来创业，父亲说：“你尽管折腾吧，折腾不下去的话，还有我呢。”
<br/>
<br/>        “年轻人就要有闯劲，不能墨守成规，但是也要有恒心和毅力，要懂得变通。”
<br/>
<br/>        “是，爸。”
<br/>
<br/>        “创业初期很困难吧？”
<br/>
<br/>        “还好。”他们几个人东跑西跑的，每天都忙得顾不上吃饭，按照常规，他们只要继承家族事业就好了，但是他们总是不甘心，想要证明自己！
<br/>
<br/>        他们是世家子女不错，他们常常在想，如果抛开他们的身份，他们会不会过得很好？
<br/>
<br/>        “你比我强，当初我也想出去干一番事业，可是来来回回怕失败，就回来继承了家族事业。”郝爸爸有些遗憾的说。
<br/>
<br/>        “你现在已经非常好了。”他们就像是朋友一样相互鼓励，谁能想到这是一对父子呢。
<br/>
<br/>        “文仔，这次回来是有事情吧？”知子莫若父啊。
<br/>
<br/>        “嗯，我回来拓展业务，目前这边还没有我们的点。”
<br/>
<br/>        “你们是怎么做的？”
<br/>
<br/>        “目前好多是加盟入股。”
<br/>
<br/>        “这种模式挺好。隔壁冷三叔家有一套院子空闲着，你可以去问问，看看他们出租不。”
<br/>
<br/>        郝景文听从了爸爸的建议去找冷三叔的时候：“我这房子自己住，不能放那些个乱七八糟的东西，你去别的地方找吧。”
<br/>
<br/>        “三叔，你看你这地方空着也是空着，租给我，你也有一份收入不是吗？”
<br/>
<br/>        “你放从外面运回来的乱七八糟的东西，我嫌脏了我的房子，给再多钱也没有用。”
<br/>
<br/>        冷三叔拒绝了郝景文，冷三叔的房子是一个大院子，是适合当库房的，郝景文又在附近找，可是当大家听说冷三叔都不愿意将房子租出去时，大家也不愿意。
<br/>
<br/>        因为冷三叔说了郝景文放的东西都是不干不净的，大家的房子基本都是自己以前住过的，心里就不太愿意。
<br/>
<br/>        郝景文在家里呆了一个多礼拜都没有租到地方。
<br/>
<br/>        最后郝爸爸将他们的一个院子腾出来，对郝景文说：“先用这几间房子凑合着吧，等业务拓展开了能赚钱了，自然就会有人来找你。”
<br/>
<br/>        郝景文没有想到在别的地方拓展业务很顺利，回到家乡会碰壁。
<br/>
<br/>        不过像父亲所说，业务拓展开了就会有人找上门了，而且会和郝景文联手。
