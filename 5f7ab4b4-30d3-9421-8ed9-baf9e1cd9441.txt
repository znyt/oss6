　　第139章
<br/>　　这天午后，阳光很好。
<br/>　　李如搬了一张躺椅在屋檐下，慵懒的晒着太阳，丫鬟立在一旁，随时将她想吃的干果，点心递到手中。
<br/>　　“小妹。”李大头跑来：“人我已经帮你找到了。”
<br/>　　李如坐直身子，如庭前的翠竹般笔直：“怎样，有本事么，莫再像上次一样，找个废物诓我。”
<br/>　　李如一直在找能用得上的人物，苏家能称得上人才的人不多，程大雷闯进苏家，竟然无一人能挡，更是让她感觉到这件事的急迫性。托李大头找了几次，但顶尖人物不好找。
<br/>　　“这次看着像个有本事的，脾气很不好。”李大头。
<br/>　　有本事才有脾气，没本事也没办法有脾气，那些既有本事也无脾气的人，天下难寻。
<br/>　　“请他进来给我看看。”李如。
<br/>　　李大头将一个人从院外请了进来。从对方走进院子的那一刻，李如的视线就没离开过对方。
<br/>　　身高七尺有余，头上扣着一顶破皮帽，身上裹着一件灰不溜秋的皮袍子。看上去也没什么出奇之处，只是一双眼睛有些特别，看了让人怕。
<br/>　　李如来了精神，微倾身子：“叫什么名字啊？”
<br/>　　对方没有回答，立在院子当中一动不动。
<br/>　　李如皱眉，微微有些奇怪。
<br/>　　李大头上前道：“他叫公羊哀，不太爱说话，本身是城外的猎户，父母饿死了，来城里找活路。”
<br/>　　高手嘛，都不爱说话，李如也没有不满。她从躺椅上站起来，步下台阶，来到公羊哀对面，捏了捏他的胳膊，仰着脸看对方。
<br/>　　公羊哀眼里有道窘意一闪而逝。
<br/>　　李如心里忽有一二分得意：“倒是长了一副好身板，多大了？”
<br/>　　“二十二。”公羊哀此时才开口。
<br/>　　“正当年呢。”李如目光往下望，注意到公羊哀的双脚：“这大冷的天，怎么还穿草鞋，桃儿，去把我新做的鞋拿来，看合不合脚。”
<br/>　　公羊哀有些窘迫，尴尬的摆手：“不必，不必，我……”
<br/>　　“嗳，你以后就是我的人了，我自然要对你好些。”
<br/>　　……
<br/>　　乔野一整体都在忙碌，抓住程大雷可以说是大功一件，城主已经准备向上面请功。今天拉着乔野说了一上午的话，无非是『国家栋梁』『我之虎将』之类的话。
<br/>　　赏赐还没有下来，估计也少不了。下午又被几个同僚拉着喝了顿花酒，酒席宴间说些：乔将这次立了大功，以后飞黄腾达可别忘了兄弟之类的话。
<br/>　　当然，最后是乔野做东。可这钱花得顺心，似乎领兵这么多年，好久没有这么痛快过了。
<br/>　　月上柳梢，乔野才恋恋不舍的离开青楼，骑着马晃悠悠往家走。
<br/>　　如果平常时候，自己就在青楼留宿了。但自己的第三房小妾刚添了个儿子，乔野视若掌上珍宝，一天不见心里就想得慌。
<br/>　　升官发财，添丁进口怎么这好事都被自己赶上了。莫非自己年前在寺里烧得香又用了，等开春的时候该再去烧烧香。
<br/>　　“福子，过来牵马。”
<br/>　　进门吼了一声，没有人回应。乔野心里骂了一声：这帮下人越来越没规矩，这时候都死哪里去了。
<br/>　　乔野拍拍脸，觉得院子里有些静，以前官运不顺，也请不起太多仆人。等这次赏银发下来，自己定再买几个丫鬟。
<br/>　　“宝儿，爹的心疙瘩，爹回来了。”
<br/>　　醉晕晕，乔野走进三姨太的屋子，张开双手就想抱自己的儿子。
<br/>　　刚推门进步，乔野的酒意瞬间烟消云散，他看到自己的家人都被绑在屋子里，嘴里塞着布，几条蒙面壮汉用明晃晃的刀押着他们。
<br/>　　乔野下意识去抓自己腰间的佩刀，却抓了个空，这才想起自己今天去喝花酒时穿着便衣去的，没有带兵器。
<br/>　　“是那条道上的朋友，想要银子尽管开口，别伤我妻儿老小的性命。”乔野开口，目光一边寻思着屋内，看有什么可以做武器。
<br/>　　“乔将军的账上，如今还能拿的出多少银子？”
<br/>　　一个匪人扭过头来，他脸色也蒙着黑布，可不知怎的，一看到他的眼睛，乔野就觉得特别悲伤。
<br/>　　“乔将军在城主虚领百夫长，这位置并无多少油水，平日乔将军还爱赌两把，却也是输多赢少，还向城里的大户借了高利贷。如今乔将军手中拿得出十两银子嘛？“
<br/>　　乔野心中一激灵，他在城中的官职的确不高，如他这样的百夫长，城主手中不知有多少个。这次抓程大雷，也是调了很多不属于自己的兵卒。
<br/>　　“既然你们查得这么清楚，就该知道我没钱。那你们为何还来我家中？”乔野没再在自己老婆孩子身上看一眼，但心却紧紧揪着。
<br/>　　“实话实说，我们今天来不是管乔将要钱的，而是给你送钱的。”
<br/>　　为首的匪人拍拍手掌，有人推出一口木箱，打开后里面是白花花的银子，粗略估计有上千两。
<br/>　　“无功不受禄，这笔钱我可不太敢拿。”乔野账面上不好看，盯着箱子里的银子多看了几眼。
<br/>　　“我们这次来，就是想和乔将军做一笔交易。”匪人道：“答应我们的事，白银一千两奉上，如果乔将军不答应，就准备绝后吧。”
<br/>　　匪人忽然拔出刀，重重斩在桌子上。家里几个小孩吓得哇哇大哭。
<br/>　　乔野一激灵，刚想开口阻止，就见对方眼怀笑意，直直的望着自己。
<br/>　　“你们是蛤蟆寨的人？”乔野突然意识到什么。
<br/>　　“乔将军果然英明！”
<br/>　　“我是不可能把程大雷放了的，就算再多银子也不行，不是我不想，是我做不到。”
<br/>　　“乔将军误会了。我家寨主一时走差，陷在落叶城，当兵抓贼，我们也没有怪乔将军的意思。只是我家寨主身子骨娇贵，吃不得苦，还请乔将军能行个方便，在牢里照料一二。”
<br/>　　“他不归我看管，我就算想帮你们也没办法。”
<br/>　　“放心，这一千两是给乔将军的，这里还有一千两是给乔将上下打点的。”说着，他们又推拎出一口箱子。
<br/>　　乔野倒抽一口凉气，蛤蟆寨就这么有钱么！
<br/>
