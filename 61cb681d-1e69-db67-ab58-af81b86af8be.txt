　　乱葬海。
<br/>　　这乱葬海，本是危险无比，其中有着无数虚空兽，成群结队，袭击武者。
<br/>　　但是今天，却是完全不一样，乱葬海中已经没有虚空兽的踪影。
<br/>　　反而，无数强者的遁光，在其中穿梭这。
<br/>　　其中的无数虚空兽，仿佛都被人全部斩杀了一般。
<br/>　　“这次北域，倒是彻底大震动了，不知道有多少强者，从中得到了好处。”
<br/>　　楚风眠跟白萱，两道身影，在这乱葬海中穿行这。
<br/>　　这如今乱葬海的变化，如今楚风眠也是已经弄清楚了。
<br/>　　这一次乱葬海中，开启的洞府，却远远不止颜古龙帝洞府这一个。
<br/>　　足足是一连开启了六个洞府。
<br/>　　六座远古洞府，同时从乱葬海中出现，引得北域中的无数天才，甚至是整个九域的天才，都齐聚在了这里。
<br/>　　才令现在的乱葬海，突然变得如此热闹。
<br/>　　“得到好处做多的人，是你吧，等回到北邙学院中，只怕你根本不需要晋升内门弟子，可以直接晋升种子弟子了。”
<br/>　　白萱看着楚风眠，笑着开口道。
<br/>　　楚风眠现在的实力，回到北邙学院中，也是一尊风云人物了。
<br/>　　一旦是晋升为种子弟子，也就算是真正成为了北邙学院的核心成员，足以进入北邙秘境，有着无数好处。
<br/>　　楚风眠如今没有得到北邙学院的资源，便是实力就足以达到这一步，等到楚风眠进入到北邙秘境中修炼，只怕是实力，更是会再进一步，变的更加可怕。
<br/>　　“白萱，你在这洞府中，也得到了不少好处，等到回去之后，估计也足以晋升为种子弟子了。”
<br/>　　楚风眠看着白萱，笑了笑道。
<br/>　　白萱的实力，在这颜古龙帝洞府中，也已经是达到了神力境极限的地步。
<br/>　　算得上是有着不少的好处。
<br/>　　现在的她，比起曾经的地榜第一落长恨，只怕都要强大。
<br/>　　也算是得到了不少好处。
<br/>　　“不错，这也多亏了你，我才有着如此收获。”
<br/>　　白萱开口道，语气中也感叹不已。
<br/>　　她邀请楚风眠，来探索这颜古龙帝洞府，可以说他们两个人，都有着极大的收获，算是真正的双赢。
<br/>　　而真正得到颜古龙帝洞府奇遇的司空轩，却是因为算计楚风眠，直接陨落了。
<br/>　　“走吧，楚风眠，我们先离开乱葬海，这乱葬海现在可是一片混乱，其中可是有着无数武者，从中截杀其他武者，收集好处。”
<br/>　　白萱谨慎的看了一眼四周，开口道。
<br/>　　“哦？”
<br/>　　听到白萱的话，楚风眠散开灵识，在这乱葬海中探察这。
<br/>　　果然他看见，有着几尊武者，正在截杀了一些刚刚从洞府中离开的武者。
<br/>　　这些人，没有进入到古洞府中去寻找宝藏，却是在外面，守株待兔，截杀其他人，获取宝藏。
<br/>　　虽然收获不多，但是风险极低。
<br/>　　“无妨，一些小人罢了，敢来一个，我就杀一个。”
<br/>　　楚风眠笑了笑，开口道。
<br/>　　“走吧，我们先离开乱葬海。”
<br/>　　开口之间，楚风眠便是化为了一道遁光，直接向着外面飞去，白萱也是同样，遁光跟随在一旁。
<br/>　　乱葬海，庞大无比。
<br/>　　两人的遁光，虽然极快，却也是需要耗费一段时间。
<br/>　　“我们从颜古龙帝洞府中，得到了无数好处，现在直接回去北邙学院，倒不是一个明智的选择，不如趁着这个机会，我们先去虚空海参加交易会如何？”
<br/>　　白萱跟楚风眠，两人一边飞行，一边交谈着。
<br/>　　白萱开口道。
<br/>　　“我们得到了无数奇物，不过其中的一部分奇物，实际上却是不适合我们，我们可以从交易会上，去换取适合我们的奇物。”
<br/>　　奇物，虽然价值极高，但是不是每一件奇物都是有用的。
<br/>　　像是楚风眠修炼战龙决，他所需要的奇物，自然都是跟龙族有关的，至于其他一些奇物，则是对楚风眠没有太多用处。
<br/>　　所以一般来说，得到奇物，都是交换最为适合自己的最好。
<br/>　　如今乱葬海开启了如此之多的古洞府，其中也会有着无数奇物，正是购买奇物的最好机会。
<br/>　　楚风眠现在有着大笔财富，正是需要购买奇物，他增添他的实力。
<br/>　　楚风眠想了一下，点了点头道。
<br/>　　“好。”
<br/>　　楚风眠现在拥有血海，财富巨大，他的力量虽然已经达到极限，但是一些奇物，还是足以提升楚风眠的战力。
<br/>　　像是强化身躯，提升血脉等等，虽然不改变力量，却也足以增添楚风眠的实力。
<br/>　　在度过生死大劫之前，肯定是积累的越多越好。
<br/>　　就在楚风眠跟白萱二人，在虚空中穿行交谈的时候，突然就在周围，几道遁光，猛然出现。
<br/>　　这几道遁光，一共五人，出现一刻，顿时凝聚灵力，封锁了周围的空间。
<br/>　　“两位，这交易会，我看你们就别去了，不如就将你们两位身上的所有奇物，都交给我吧。”
<br/>　　这五人，皆是中年男子，每个人的实力，都得到了生死境三重左右的地步。
<br/>　　比起许多大势力的天才，都要强大许多。
<br/>　　“这五人是，虚空五霸！是虚空海中的强者，在虚空海中天天截杀武者，他们五人联手起来，除非是生死境高阶，不然少有人可以从他们手中逃走。”
<br/>　　白萱看到这五人，也是面露忌惮，显然是知晓这五人的来历。
<br/>　　不过白萱也没有多少慌张，楚风眠的实力，她是清楚的。
<br/>　　这虚空五霸，虽然强大，但是远不可能是楚风眠的对手。
<br/>　　虚空五霸中的一人，听到白萱的话，大笑着开口道。
<br/>　　“哦？看来你还知道我们虚空五霸的名声，那就好，乖乖将空戒跟身上的所有灵器交出来，可以绕你们一命。”
<br/>　　楚风眠冷哼一声道。
<br/>　　“虚空五霸？原来是五个杂碎，在这虚空海中截杀武者？既然今日你们碰到楚某，楚某就杀了你们，也算是为虚空海，除了个害！”
<br/>　　“你！”
<br/>　　虚空五霸听到楚风眠的话，阴冷说道。
<br/>　　“敬酒不吃吃罚酒，不老老实实交出空戒，那我就自己来取！”
<br/>　　这虚空五霸其中一人，猛然一拳向着楚风眠打了过来。
<br/>　　但是他的拳风，还未打出。
<br/>　　楚风眠的手，已经动了，一掌凭空打出。
<br/>　　这虚空五霸，全部都被震飞出去，一个个口吐鲜血。
