　　这一古怪的理论让在场的其余人都听得目瞪口呆，不仅很有道理的样子将原本内心的愧疚一扫而空，而且还莫名的感觉很爽。
<br/>　　三名妇人眨巴着眼睛，看向这名妇人的眼神都冒着光，纷纷讨论起来。
<br/>　　“说的没错啊！我们不挑剔谁挑剔？”
<br/>　　“咱们这些一开始就支持狼牙的本来就应该得到这些权利，谁让那些人总是不信任狼牙，挑三拣四还整天针对，咱们能住在这安罗城的居民，哪个不是一开始跟着狼牙吃过苦的？”
<br/>　　“记得那会儿没搬过来前，咱们还住在基地里，没现在这么多丰富的食物，吃的最多的就是一种变异番薯。”
<br/>　　“是啊！还记得那时候有人刻挑拨离间，咱们就是不走，就是支持楚涵上将！现在看看这日子过的多好？”
<br/>　　少年的母亲更是兴奋无比，朝着那妇人问道：“哎真没看出来，你这说话一套一套的跟辩论似得，以前文明时代是做什么工作的？”
<br/>　　年轻的妇人笑而不答，指着桌面道：“碰。”
<br/>　　四名妇人继续打着麻将，话题也从狼牙战团转移到了这名妇人身上，显然刚才的一套言论，让这几人都分外好奇。
<br/>　　一开始愤怒交加的少年，也在这时候神态变化起来，他望着说出那番话的妇人眼神复杂。
<br/>　　这名阿姨是最近才出现的，谁也不认识只知道姓袁，少年也管她叫袁阿姨，谁也不知道她住哪，只知道前两天闲逛到这里，跟少年母亲闲聊了两句后，第二天便带了一副麻将过来，接下来的发展也就显而易见了，少年家就变成了这四个妇人打麻将的聚集地。
<br/>　　一开始少年还很惊艳这名漂亮袁阿姨的相貌和气质，可当这四个妇人整天打麻将的日常开启后，少年对袁阿姨的好感就大打折扣，觉得太没有追求。
<br/>　　但今天听到袁阿姨的这一番言论，却让少年内心有些冲击，他从来没想过在他口中的这些没有追求的行为，很有可能就是狼牙战团希望见到的场景？
<br/>　　甚至少年更觉得，这名袁阿姨说的一点没错，安罗城的居民过的越好，越无忧无虑，越是对一些细节挑剔，才越能体现出楚涵上将的厉害。
<br/>　　因为他们有这个资本！
<br/>　　这是狼牙战团拼尽全力后带给他们的，是别的基地幸存者都没有的权利，安罗城越是如此富饶的生活，才越是狼牙荣耀的体现！
<br/>　　就在少年震撼中逐渐观念改变的时候，忽的房门被敲响。
<br/>　　咚咚！
<br/>　　带有节奏，和一丝礼貌感。
<br/>　　“谁啊？”少年的母亲自然而然的起身开门。
<br/>　　在安罗城没有人有胆子捣乱，那名恐怖的军法官刘玉定所颁布的严格城市法规，更让这座城市里的犯罪死绝，所以少年的母亲起身到打开门的过程中，都从没想过安全的问题，这根本不存在。
<br/>　　但随着她将门打开的一刹那，便突然面色大变，而后脱口而出的失声一喊：“哎呀我的妈啊！”
<br/>　　屋内的气氛顿时一变，少年立即反应过来连忙奔过去，两名妇人吓了一跳，仿佛是连锁反应般的在屋内大喊大叫了起来。
<br/>　　唯有那名姓袁的妇人只是皱了下眉，而后朝着门口望去。
<br/>　　很快在一阵鸡飞狗跳之后，有些尴尬的对话声就传了过来。
<br/>　　“不好意思！这名军官！我没想到有军官过来，请问有什么事吗？”这是少年母亲的声音。
<br/>　　屋内两名受到惊吓的妇人愣住，而后面上好不尴尬。
<br/>　　原来只是来了个军官啊！
<br/>　　不过在这片居民区，的确很少会有军官来敲门，哪怕是警戒队也不会凭白打扰他们的生活，所以那少年的母亲会惊讶的失态也不奇怪。
<br/>　　“我来找个人。”随着一个声音落下后。
<br/>　　一名身穿军服，浑身上下打理的一丝不苟，胸前佩戴着明显少校徽章的年轻军官，便由少年母亲领进了屋。
<br/>　　少年紧跟其后，面带激动的潮·红，双眼更是冒着无比闪亮的神色，显然对少校军衔者，少年有着本能的崇拜和向外。
<br/>　　这名少校的进入，很快让屋内的气氛冷凝起来，几人都有些不知所措。
<br/>　　找人？找谁？
<br/>　　少校进屋后环视一圈，在目光扫到那名袁姓妇人后，双眼一亮快步走上前。
<br/>　　啪！
<br/>　　一个标准无比的军礼，又把在场的几人吓了一跳。
<br/>　　敬完礼后的少校恭敬出声：“袁熙叶院长，您果然在这里！”
<br/>　　话一出口，屋内的气氛顿时为之一变，三名妇人和那个少年全体呆愣原地，完全没能搞懂情况的看着这一幕。
<br/>　　姓袁妇人竟然被一名一看就很厉害的少校敬礼？
<br/>　　还是最为标准的军礼，这代表的可不仅仅是身份，还有另一重尊重的意思！
<br/>　　这个每天在这里打麻将的袁阿姨，什么来头啊？
<br/>　　而且……院长？
<br/>　　院长是什么情况？
<br/>　　哪里的院长？
<br/>　　袁姓妇人自然就是袁熙叶，在结束为期三个月的解毒剂调配后，她便离开了几乎成空的狼牙基地，住在了更有人气的安罗城里，这是尚九娣的意思，而聪慧过人的袁熙叶，也很清楚很多事情她还是不要知道太多的好。
<br/>　　“我记得你。”袁熙叶望着这名少校，露出了一个微笑：“前几天刚见过。”
<br/>　　“是。”少校点头：“上回是属下唐突了，也是太过心急。”
<br/>　　“理解，那老头的安全的确很重要。”袁熙叶没有透露姓名，但口中所谈论的自然便是行·事冲动的洛冥，与这名少校的第一次见面，也是对方着急寻找洛冥下落，而不打招呼的闯入了袁熙叶的住宅，为此还差点遭受狼牙最后驻守在安罗城的部分警戒队成员的围杀。
<br/>　　少校显然不想再回想前几天与死亡擦肩的经历，取出一封信递上：“上面派我来给您送封信。”
<br/>　　“信？”袁熙叶一脸惊讶。
<br/>　　“是的，请院长过目，事态紧急，只能请您出手了。”少校开口道：“我会在安罗城暂住一天，无论您是否能分析出原因，都希望您能给出一星半点的建议，第二天我会带着您的手信送到段氏基地。”
