　　“没什么好说的，这一战……大家尽力，不许有任何伤亡。”叶少阳照例发表“战前宣言”，也没有说太多，一眼扫过去，每个人脸上都闪耀着兴奋的光，对即将到来的战斗很是期待。
<br/>　　这一群都是暴力分子，对他们来说，没什么比在一起打架更爽的事情了。
<br/>　　叶少阳目光移到钟馗身上，钟馗大马金刀地坐在龙椅上，从一开始就一句话不说。
<br/>　　“钟天师，您有什么想说的？”
<br/>　　“没什么没什么，干就可以了。”钟馗捋着胡须，冲大家笑了笑。叶少阳拿出了那颗晶莹剔透的“核桃”，托在掌心，这时候有一个负责守卫的将军快步进来，走到林三生面前，躬身说道：“大帅，皇帝来了！”
<br/>　　皇帝？
<br/>　　大伙面面相觑，这个时候他来干什么。
<br/>　　自从这里的生灵全面觉醒之后，皇帝的作用就没了，林三生把他打发去了宫外住，派人跟踪（因为他没有记忆，也不知道本尊是什么，林三生对此存疑），不知道这会儿他来干什么。
<br/>　　“拦住他，让他不要耽误我们正事。”林三生下了命令，结果这将军刚出宫门，皇帝就到了。将军上去试图传达命令，皇帝一伸手，一股无形气息直接将他震飞出去，摔在地上，吐血不起。
<br/>　　宫门外的守兵大惊失色，一起冲上去，但皇帝什么也没做，只是抬了抬手，这些人就被震飞，从金銮殿前面的台阶上摔下去。
<br/>　　叶少阳一行人全站了起来。
<br/>　　钟馗也皱起了眉头打量着。
<br/>　　皇帝一直走到他们面前，站住，脸上带着镇定自若的微笑，再不是之前那副唯唯诺诺的样子，看上去还真有点九五之尊的范儿。
<br/>　　叶少阳突然发现，他一只眼睛是蓝色的，另一只是黄色的，像两颗宝石一样闪着光。
<br/>　　“这么重大的事，你们为什么不通知我。”皇帝慢悠悠说道。
<br/>　　身后，突然有数道人影飞来，直接撞破了叶少阳一行在皇城外布置的结界，急速飞来，落在皇帝身后。看上去都是普通的人类，有男有女，穿的什么都有，有几个还是乡下农人打扮，但所有人都面无表情，带着一丝邪魅的气息。
<br/>　　“哎，我可没看出来，你是深藏不露啊，你是星月奴那老货的人吧，想打架？”小马卷着袖子，把板砖操在手中。
<br/>　　这群人一动不动。
<br/>　　叶少阳拦住小马，凝视着皇帝，说道：“有话就说，要打就先报名，不要在这装酷。”
<br/>　　皇帝不理他，转头看着钟馗，道：“钟大士自然知道我是谁。”
<br/>　　钟馗往前欠着身子，深吸了一口气，面色一沉，缓缓说道：“猫生九面，其无本心，你是九面猫妖！”
<br/>　　九面猫妖？
<br/>　　叶少阳等人狐疑地互相看去，这个名字，他们隐约都听过，但想不起来有什么具体的传说。
<br/>　　皇帝大笑，身形突然变化，现出人形，是一个身穿纱衣、身材格外妖娆的女子，相貌极为狐媚，唯一不变的还是那对一蓝一黄的眼睛，轻轻眨动着。
<br/>　　钟馗摇头叹道：“我现在鼻子真的不行了，我早就觉得这皇宫里有一股怪味，却没想到是你。”
<br/>　　“几百年不见，劳钟大士牵挂。”九面猫道了个万福。
<br/>　　钟馗哼了一声道：“当年你在鬼域作乱，我追了三百里都没能追到你……不成想你却在太阴山立足了。”
<br/>　　太阴山！！
<br/>　　大伙心中一惊。
<br/>　　之前都以为他们是轩辕山的人，没想到竟然是太阴山的！
<br/>　　九面猫嫣然一笑，道：“多亏钟大士当初追赶，若不是你，小女子也不会逃亡太阴山，更不会有今天。”
<br/>　　“而今你为左手门下三凶之一，也算出人头地了。”钟馗用略带嘲讽的语气说道，转头对叶少阳他们说道，“她是九面猫，曾经是青冥界大妖，与九尾狐一向不和，结果没能打过九尾狐，被赶出青冥界，后来流浪到鬼域，干了不少坏事，后来逃到太阴山，倒是成了个人物。她跟四大罗刹是一个级别的。”
<br/>　　小马一听，立刻眼睛发亮，问她：“你最近见到玉面罗刹没有？”
<br/>　　“那骚蹄子，我跟她不熟。”
<br/>　　“靠，说的你好像很正经似的！”小马大为不满。
<br/>　　九面猫望着叶少阳，说道：“九尾狐那厮心比天高，真没想到居然会给你做了妖仆……叶少阳，凭这一样，我便当你是个人物。”
<br/>　　“别说没用的，你们要是不打架，就有事说事。”要是在人间，遇到太阴山的人，根本不用说什么废话，直接开打就好了，本来就是势不两立，但眼下最主要的敌人是法术公会，如果再跟他们动手，损耗太大，再说他们就这么冲到面前，似乎不是来打架的。
<br/>　　九面猫看叶少阳不动手，脸上露出满意地笑容，说道：“叶少阳，你们这群人，的确也都是强者了，但九星阁中，可是星月奴的老巢，天知道那里面有什么埋伏，你们就这么进去，未必是他们的对手，而一旦失败，必然就是全军覆没……”
<br/>　　“你有话就直说，不说就滚，没兴趣听你废话。”叶少阳没好气说道。
<br/>　　九面猫脸色变了变，依旧露出笑脸，说道：“如果你们要去对付星月奴，可以算上我们一份。”
<br/>　　大家都吃了一惊。
<br/>　　“我没听错吧？你们太阴山，愿意跟我合作？”
<br/>　　“此一时彼一时，坦白说，我们也是一样被困在这里，需要出去，只有合作，大家才有机会。”
<br/>　　叶少阳想了一下，转头看着林三生。
<br/>　　林三生问道：“先告诉我们，你们为什么会在这里？”
<br/>　　“三个目的，一，前来调查圣灵会的底细——”
<br/>　　她没说完就被林三生打断：“调查他们做什么，法术公会与人间法术界混战，你们应该最开心才对？”
<br/>　　九面猫道：“话是这样，但我们总要调查法术公会在做什么，星月奴又强大到了什么地步。”
<br/>　　“结果你们来了就出不去了？”
<br/>　　“我们来了之后，跟你们一样，起先都失去了记忆，八成是那帮人也不知道我们是太阴山来的，当做跟你们一样处理了。我因为实力强，被洗脑成了皇帝，这帮人，给我重塑身份也就罢了，连性别也换了，真是该死……”
<br/>　　叶少阳心中一动，道：“你之前也一直没想起自己是谁？”
<br/>(本章完)
