　　这两人飞到他们附近，大声喊道：“我等乃巡游司鬼曹，奉司主之命，前来接引诸位，若想窥探军情，请移步城南烽火台，如今战局危急，非阴司之生灵，不可在酆都城别处逗留！”说完对大家指明了城南烽火台的方向。
<br/>　　这时，又有数十道人影从法杖中穿梭而来，落在地上。
<br/>　　是王道乾、逍遥飞、释信无等人。
<br/>　　“师叔……”四宝过去跟逍遥飞打招呼。
<br/>　　“你怎么就走了！”逍遥飞狠狠瞪了他一眼，“我还有话没跟你说！”
<br/>　　“额，什么话？”
<br/>　　“回去再说！”逍遥飞扫了一眼附近的人，低声道。
<br/>　　叶少阳看着众人，猜测应该是有人把阴司发生的事传到了龙虎山上，正好他们都在，于是结队来围观了。
<br/>　　慕寒也该来了吧。
<br/>　　叶少阳正想着，又有不少人从法杖中飞出，陆陆续续地飞出来十几个人，其中就有慕寒，还有苏钦章。见到叶少阳，立刻过来打招呼。
<br/>　　“叶掌教，你也来了。”慕寒冲叶少阳阴阳怪气地笑了笑，“为何匆匆离去，是我龙虎山招待不周？”
<br/>　　叶少阳压根没理他。
<br/>　　“走吧。我们也去烽火台。”萧逸云说道。
<br/>　　叶少阳叫上苏钦章一起，跟着萧逸云走，其余人也都跟了上去。
<br/>　　他们所在的位置，是枉死城边上的一块空地，就在城墙边上，不远处就是南大门。
<br/>　　所谓的烽火台，并不在城墙上，而是城内的一座高台，比城墙要高出许多，目测过去，有几十层那么高，砖石打造，四面都是阶梯，叶少阳一行人爬了上去。慕寒和慈心师太等人选择了另外的方向。
<br/>　　烽火台的顶端，是一个大约十米长宽的正方形的平面，上限竖立着一方大鼎，纯黑色，四方形，四只脚，看上去有点像是司母戊鼎，上面刻着许多文字，像是大篆，叶少阳也看不懂。
<br/>　　在大鼎之中，漂浮着一股黑色的火焰，火苗有至少两米高，扶摇直上。但诡异的是，走在边上一点也感觉不到热量，相反感觉很冷。
<br/>　　上到高台上之后，所有人注意力都被这口鼎和上面的火焰吸引了。
<br/>　　“这是徐州鼎，天下九鼎之一，上有天子封印，一旦打开封印，就会冒出无量鬼火，火是冷的，就算百里之外，也能看得见。”萧逸云解说着。
<br/>　　“以前好像没有这个？”
<br/>　　“当然没有，这可是天下九鼎之一，一直放在冥王殿的，这不是出事了才被大帝拿出来的，这火焰就是给远处的人看的，不同情况点不同颜色的火，黑火就说明有战事，但还没到危机时刻。那些驻守在远方的元帅们，会派人定期前来附近探视烽火台，依照火光的颜色制定战略。”
<br/>　　这办法，虽然朴素，但也算实用。叶少阳心想。毕竟阴司没有电话和网络，不能远距离通信，一旦酆都城被围困，想要派人出去送信都难。
<br/>　　烽火台比城墙高的多，因此站在这烽火台上，城外的情况一览无余。
<br/>　　黑压压的军队，聚集在城外，在距离城墙几十米远的地方，好像被是一层看不见的东西阻挡了，无法靠近，在城外缓缓游荡着。
<br/>　　整个南边的城墙下边，黑压压全是人，另外三面看不到，但想来情况也差不多。
<br/>　　所有人的表情都很凝重，呆呆地望着眼前这一幕，没一个人开口说话。
<br/>　　叶少阳明白，他们内心的想法，都跟自己一样，四个字：不敢相信。
<br/>　　从他们当上法师、知道有阴司存在的那一天起，阴司在他们的概念中，就是一切权威的象征，是天地大道的维持者，最重要的是……是千百年来屹立不倒的朝廷。
<br/>　　而现在……居然被太阴山戝众包围，兵临城下。
<br/>　　之前听到橙子说的时候，叶少阳和四宝虽然震惊，但远远没有此刻视觉上带来的冲击强烈。
<br/>　　以前太阴山跟阴司相持，战争无数，从来没发生过被围城这种事，如今……这是怎么了？
<br/>　　“为什么阴司不反击？”叶少阳喃喃问道。
<br/>　　萧逸云苦笑。“反击，怎么可能不反击，但这次太阴山倾巢而出，兵分四路，截住了前来勤王的几支军队，一口气杀到城下，酆都城才有多大，里面这点兵力根本没用，于是大帝用苍穹印罩住了整个酆都城，有几位大佬一起作法加持，借助酆都城内的风水布局，能挡住千军万马，暂时他们攻不进来。”
<br/>　　“暂时？”叶少阳道。
<br/>　　“人家太阴山大军大老远开过来，总不会就是为了围城吧，围城对他们来说，只是第一步，谁知道他们下一步会怎么样？”
<br/>　　除了叶少阳几人，先头到的几个普通法师，还有慕寒一干人，也都在听着他解说，一个不认识的法师凑上来，神色十分恭敬地对他作揖，说道：“大帝和菩萨为何不出手呢，还有钟天师等诸位大佬，只要他们出手，对付这些杂鱼小鬼，不在话下吧？”
<br/>　　“你想太多了，”萧逸云白了他一眼，冷冷说道，“战争，从来就不是个别人能决定胜负的，我们这边有大帝和菩萨，人家那边也有鬼王、右手，照你这么说，阴司派出去那些军队，早就让鬼王一个人杀光了。”
<br/>　　叶少阳听了这话，深以为然，尤其是在见识了空界之战后，更加知道战争是怎么一回事，一个人再厉害，也不可能打得过整个军队，毕竟军队之中也不乏强者，一百个对你一个，甚至一千个对你一个，堆也给你堆死了，就算堆不死你，你自己也有灵力枯竭的时候。
<br/>　　就算是酆都大帝和无极鬼王，也不可能逃脱这天地规则。
<br/>　　“阴司打算怎么应对？”叶少阳问。
<br/>　　萧逸云耸了耸肩，“这我就不知道了，府君和转轮王他们都去冥王殿开会去了，到现在还没回来，不知道会想出什么办法，真希望快一点搞定，不然就撑不住了。”
<br/>　　“什么撑不住了？”叶少阳问。大家也都围上来听。
