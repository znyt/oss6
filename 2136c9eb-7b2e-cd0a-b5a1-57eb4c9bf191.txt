徐少南已经相当不爽了，现在陆寒霆这么一挑衅，他松开了夏夕绾，三并两步的上前，一个结实的拳头就往陆寒霆的俊脸上砸去。
<br/>
<br/>        陆寒霆躲都没躲，拳头砸上来的时候，他一个拳头也砸到了徐少南俊美的脸上。
<br/>
<br/>        两个男人当即扭打在了一起。
<br/>
<br/>        啊！
<br/>
<br/>        不知道是谁尖叫了一声，场面瞬间失控了，陆氏太子爷和徐家太子爷在帝吧里打了起来，大家吓得四处逃窜，这两个人都是帝都惹不得的大人物，大家深怕惹上什么麻烦。
<br/>
<br/>        厉嫣然脸上的笑容都没有了，脸色苍白的僵在原地，她从来没有看见过陆寒霆打架。
<br/>
<br/>        在她印象里，陆寒霆完美继承了陆家人的经商天赋，十几岁就开始玩转商界，他们这个圈子里的人还在嬉笑打闹的时候，他已经出入名利场，他比同龄的人更加成熟稳重，心思也深不可测，她从来没想过他也会打架。
<br/>
<br/>        厉嫣然记得他好像打架过一次，不过好些年前了，现在那场打架的痕迹也被洗干净了，几乎没人记得了，那一次打架他好像是为了自己的母亲。
<br/>
<br/>        那这一次打架，他又是为了谁？
<br/>
<br/>        厉嫣然看着此刻的陆寒霆，徐少南是从小打到大的，真是打遍帝都无敌手，但是这一次他碰上了硬茬，陆寒霆一点都不逊他，他出手快准狠，每一个拳头都带着无比凌厉的力道砸到徐少南身上，徐少南很快就挂了彩，但是陆寒霆也好不到哪里去，两个人都受伤了。
<br/>
<br/>        “不要打了！”厉嫣然冲上前，想去拉架，“寒霆哥哥，不要打了！”
<br/>
<br/>        这时夜明珠过来将厉嫣然拉开了一点，“嫣然，不要上前，待会儿他们碰到你身上，很危险的。”
<br/>
<br/>        场面逐渐失控，两个男人打碎了不少东西，夏夕绾跑上前，伸出小手一把拽住了陆寒霆的衣角，“够了陆寒霆，别打了！”
<br/>
<br/>        陆寒霆打的眼眶通红，他扭头看向夏夕绾，她此刻的劝阻在他看来就是袒护徐少南的实锤，他直接将她甩开，“滚！”
<br/>
<br/>        夏夕绾被他的力道给掀翻在了地上。
<br/>
<br/>        “丑丫头！”徐少南瞳仁一缩，暴怒道，“陆寒霆，你他妈的是不是男人，你对一个女人动手算什么本事！”
<br/>
<br/>        看见夏夕绾摔在地上，陆寒霆高大的身躯倏然一僵，他的心像是被一只大手给狠狠的揪住了，很疼很疼。
<br/>
<br/>        “陆寒霆！”徐少南冲过来，揪住陆寒霆的衣领结结实实的给了他一拳。
<br/>
<br/>        “绾绾，你没事吧？”范恬她们迅速跑了过来，紧张的查看夏夕绾的伤势。
<br/>
<br/>        夏夕绾纤长的羽捷颤了颤，“我没事。”
<br/>
<br/>        说完她站起身，没有再去看打架的那两个男人，她拔腿直接走了。
<br/>
<br/>        陆寒霆看着她走的，她纤柔的身影一点点的消失在了自己的视线里，他收回目光，突然对打架这件事有点意兴阑珊，他觉得自己真够无聊的，跑到这里来打架。
<br/>
<br/>        徐少南揍了他两拳，见他不动了，徐少南也就气喘吁吁的停下了手，“陆寒霆，我说你莫名其妙的发神经究竟是受了什么刺激，你被人给甩了，失恋了？”
