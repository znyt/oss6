　　第575章
<br/>　　一叶轻舟靠岸，船头一人跃上白沙岛，肩头扛着一柄重剑。
<br/>　　程大雷和燕不归从不同的地点出发，却一前一后来到同一目的地。
<br/>　　三天时间过去，燕不归已养好了伤。不得不说，戎族的生命力当真强悍。伤好后的第一件事，他就是前往白沙岛，找张六复仇。
<br/>　　当然，他想要杀死的人绝对不止张六一个。
<br/>　　“张六，你在哪里？”
<br/>　　这一声吼惊动了岛上的所有人。众人心里都是一激灵：程大雷还有帮手。
<br/>　　程大雷也稀里糊涂，向远处忘了一眼，心里奇怪道：他怎么来了。
<br/>　　燕不归的出现出乎所有人预料，但对程大雷来说，终究不是坏事。毕竟，有变化总是好事，他的出现至少能为自己分担一部分压力。
<br/>　　程大雷之所以如此想，是因为他的情况已经十分糟糕。
<br/>　　“张六，你在哪里？”
<br/>　　燕不归又喝了一声，扛着剑向岛中央走来。大家都想拦住他，免得他和程大雷汇合，二人强强联手。
<br/>　　强强联手是不可能的，程大雷也燕不归想要杀死的名单上。
<br/>　　不过，书院可不知道这件事。
<br/>　　张六哇呀呀大叫一声，提着大棍就扑了过来，口中暴叫：“小子，六爷爷送你去死。”
<br/>　　一见仇人，分外眼红。燕不归过往的伤心事都想了起来，他是个少说多做的男人，见到仇人，当场无二话，提着剑就冲杀了过去。
<br/>　　二人捉对厮杀，打得却也是难解难分。燕不归仗着年轻气盛，又一腔刻骨仇恨，打得张六抬不起头来。
<br/>　　张六今天的心情很憋屈，他当年也是大名鼎鼎的人物，可今天先后碰到程大雷与燕不归，结果都没有讨到便宜。
<br/>　　不过，这并不重要，因为燕不归孤家寡人，张六可不是一个人。一见他不占上风，书院的高手都围了过来，要合力斗杀燕不归。
<br/>　　燕不归顿时感觉压力山大，他报仇的决心是大的，可决心有时候什么都代表不了，并不能令他的实力突然暴增。
<br/>　　他的出现，倒是减轻了程大雷不少压力。可程大雷的情况依然不太好。
<br/>　　张六迎上了燕不归，入阵与程大雷决斗的是立地太岁孙飞鹰。这孙飞鹰能有立地太岁这外号，怎样的脾气，也就可想而知了。他的本事，就是背后九把飞矛，号称百步之内，例不虚发。
<br/>　　程大雷还有一招本事，唤作灵犀一指，能接天下兵器。说白了，他的属性专门克制孙飞鹰。
<br/>　　孙飞鹰刚才已丢出两柄飞矛，被程大雷轻而易举化解。这次更是使出看家本事，挥手三矛齐发，速度有快有慢，打的就是程大雷要害。
<br/>　　程大雷接一柄，打两柄，再次化解掉孙飞鹰的攻击。直到孙飞鹰的飞矛用尽，都没对程大雷形成有效攻击。
<br/>　　周围观战的诸人，哪怕是程大雷的敌人，也想为程大雷打一声好。程大雷可以轻易挡住孙飞鹰的攻击，不代表孙飞鹰弱，因为他们都了解孙飞鹰的实力，那就只能代表程大雷过分强悍。
<br/>　　恰好属性相克而已。
<br/>　　飞矛用尽，孙飞鹰就准备撤出剑阵。毕竟，连看家本领都奈何不了程大雷，更何况近身缠斗。程大雷自然不容他这样轻易离开，可孙飞鹰一走，剑阵就围了上来，掩护孙飞鹰撤退。
<br/>　　如此配合，的确能令孙飞鹰立于不败之地。
<br/>　　“想走。”
<br/>　　程大雷冷喝一声，手中抓着的飞矛脱手而出，只敢觉一道杀机扑面。孙飞鹰顿时一激灵，来得好急好快，他身子跃起，这飞矛陷入了他的大腿中，又生生将他从天空中扎了下来。
<br/>　　想要走，总要留下些什么。
<br/>　　程大雷眼神更加凌冽，今日这场硬仗不太好打，自己必须拿出全部实力，不容半点犹豫。
<br/>　　夫子和孟玄青在高处观战，事到如今，程大雷的表现出乎他们的意料。他已经足够强了，当然，凭目前他所展现的实力，想要打赢这场硬仗还是不够。
<br/>　　他们身后的木笼里囚禁着李莫愁与李婉儿。二人中了迷药，此刻都陷入昏睡中。所有人的注意力都集中在程大雷和燕不归身上，却没人注意到木笼中的二人。
<br/>　　李莫愁轻轻晃动一下脑袋，睁开了眼睛。
<br/>　　李婉儿纵然有些武艺，但其实和普通人也差不了太多。可李莫愁本就是用毒的高手，她苏醒的时间要比预料得早一些。
<br/>　　眼前的情况令她大吃一惊，她完全不知道发生了什么事，事实上，孟玄青一出手就将她迷倒了，之后怎么到这海岛上，中间又发生了什么事，她完全不懂。
<br/>　　不过，她毕竟也是老江湖，醒来后没有大呼小叫，只是悄悄观察着环境，然后恢复着麻木的身体。
<br/>　　她看到了陷入死战的程大雷，又看到了背影冲着自己的孟玄青。
<br/>　　心里已经有了敌我判断，这孟玄青一看就知道不是好人呐。
<br/>　　深吸一口气，摸到了腰里的针囊，冰魄神针入手，调整身体姿势后，挥手就打了出去。
<br/>　　孟玄青忽然一怔，感到自己被一股杀意锁定。他刚回头，一枚银针就射入了他的肩膀。
<br/>　　冰魄神针见血封喉，触者丧命，李莫愁冷哼一声，一掌拍碎木笼。
<br/>　　真特么当老娘是好欺负的么。
<br/>　　却见孟玄青脸色变了三变，挥手将肩膀的银针拔掉，盯着看了一会儿，心里奇怪道：“发生什么事了，她怎么突然醒了。”
<br/>　　李莫愁大惊失色，冰魄神针向来不会失手。可她怎么知道，这孟玄青本就是用毒的高手，他常和毒物接触，体内不知道已有多少毒素。本身早已变得百毒不侵。这也是夫子为何不在他面前喝茶的原因。
<br/>　　恰恰属性相克。
<br/>　　好吧，李莫愁立刻镇定下来，银针杀不死你，自己就一巴掌拍散你。
<br/>　　“来。”
<br/>　　孟玄青感觉很丢人，用毒的高手竟然出现纰漏，日后还怎么在老朋友面前聊天。
<br/>　　“你不要插手，我亲自料理她。”孟玄青要将丢掉的脸面挣回来。
<br/>　　李莫愁轻敌了，银针没能奈何孟玄青。而孟玄青明显也轻敌了，这李莫愁也是个绝世高手。
<br/>(本章完)
