        &lt;h3&gt;第三卷第九十章  忙碌的江大少&lt;/h3&gt;
<br/>
<br/>        江俞轩的招标工作已经完成，通过激烈的竞争，江俞琪和江俞轩姐弟俩拔得头筹，参与西城南郊城市建设改造中来。
<br/>
<br/>        从陈子建家里出来后，第二天陈子建带人来进行了两天认真仔细的勘查。对于冯睿插手他工作的事情给江俞轩他们造成了损失陈子建深感内疚，他工作上一直兢兢业业，却没有想到自己的枕边人背着他干了那么多见不得人的事情。
<br/>
<br/>        对于江俞轩他们这个事情，他是知道的，还有他不知道的呢？家里那么多的东西，冯睿祸害了多少人啊。
<br/>
<br/>        陈子建认真的勘查完整个地盘，对江俞轩说：“江总，南郊虽然也是名胜之地，但是这一块开发的地方都绕过了一些古迹，你们是可以接手开发的。”
<br/>
<br/>        “谢谢，陈总，你这样一说我就放心了。”江俞轩给几个工作人员每人拿了一条烟，大家不接。
<br/>
<br/>        “这烟也就是几百块钱，够不上行，贿，况且我也没胆，我也就是感谢大家这么辛苦的跑了几天。”江俞轩诚恳的说。
<br/>
<br/>        大家推让了一下，客气的将烟收下了。
<br/>
<br/>        陈子建对江俞轩说：“很抱歉，那天晚上的事情让你见笑了，我家里原本是有一只一模一样的那种手镯，看见那只手镯我太高兴了，我还想着我完成了父亲的心愿，没想到我老婆将手镯送人了。”
<br/>
<br/>        “没事，该来了自然会来，也许上天故意给你们亲人磨难，说不定以后都是大团圆，我也问问我朋友那个手镯是怎么回事。”江俞轩安慰陈子建。
<br/>
<br/>        他突然从心里有些同情陈子建了，如果陈子建知道他的老婆伙同别人将自己的妹妹偷梁换柱估计会崩溃吧？
<br/>
<br/>        “那就谢谢江总了。”陈子建其实也是很难为情的。
<br/>
<br/>        送走陈子建之后，他就开始投入了紧张的投标工作中，在他的行为准侧里就没有失败这一说，只要他想要去做成的事情，他必须的要拿下来，在陈子昂公司的那几年，他们从一个行业小白鼠变成业界大佬，凭的就是他们的智慧谋略、还有勇往直前不屈不挠的闯劲。
<br/>
<br/>        他带领他的团队经过一个礼拜没日没夜的奋斗，一份满意的方案呈现在他的眼前，他们经过一轮轮的选拔，终于脱颖而出。
<br/>
<br/>        南郊掌管规划的杨东杨总在招标现场握着江俞轩的手说：“江总，希望我们一起努力，建立一个美丽和谐现代化的新区。”
<br/>
<br/>        “杨总放心，我们江氏集团一定会为了美丽而浑厚的西城做一份贡献。”
<br/>
<br/>        西城南郊新区的规划沙盘也搬到了江俞轩的办公室。
<br/>
<br/>        陈子霖在期间找了江俞轩一次，也是因为洛城的项目，但是江俞轩忙得都顾不上和他说几乎话。
<br/>
<br/>        洛城的方案已经出台，要请江俞轩过一下，陈子霖将方案发给陈子昂的时候，陈子昂在外地出差，只批复了两个字个字：可行！
<br/>
<br/>        陈子霖就直接将厚厚的方案打印出来，直接去找江俞轩。
<br/>
<br/>        “子霖，你将方案放这里。我闲下来看。”江俞轩努了一下嘴。
<br/>
<br/>        南郊西城改造项目的头头脑脑都在这里，他们要开会。
<br/>
<br/>        陈子霖看了一眼江俞轩，围在他身边的都是大人物，有些人还在讨论着沙盘上的位置，“那你忙，我先走了。”
<br/>
<br/>        洛城有俩个项目，他们都参与了方案，但是后期花落谁家还真不知道，光去考察做方案的都已经有了十几家，似乎一个地市级的单位比西城还要让人关注，这似乎有些反常。
<br/>
<br/>        陈子霖实际上是想和江俞轩商量一下，看看后期怎么操作，毕竟他们俩个人一起去的洛城。
<br/>
<br/>        江俞轩没有想到陈子霖会将方案打印出来，看看厚厚的一份方案书，他苦笑了一下，可能是最近忙了，陈子霖发的邮件他都没有时间处理，果然，一个人是很难全部都兼顾得到了。
<br/>
<br/>        江俞轩在晚上抽时间看了陈子霖的放案，将熟睡中的陈子霖给叫醒了。
<br/>
<br/>        “子霖，方案我看了没有任何问题，你主要是要按照傅海生他们的需求来做。”江俞轩开门见山的说。
<br/>
<br/>        “我们就是按照傅海生的需求来做的。”陈子霖一个激灵从睡梦中醒来。
<br/>
<br/>        “你怎么现在还没有睡？”陈子霖看了一下时间，凌晨3：50分。
<br/>
<br/>        “最近太忙了，都时间时间关注公司的事情，才看到你发的邮件。”江俞轩抱歉的说。
<br/>
<br/>        “知道你忙，还是要注意休息。”
<br/>
<br/>        “最近这一段时间跟着那一帮子老爷们天天加班加点的进行项目论证，每天晚上回到家都1点多。”
<br/>
<br/>        “本来不想去找你，但是子昂最近也是到处飞，给我的批复只有两个字。”
<br/>
<br/>        “我看到了。”他们有内部的oa流程。
<br/>
<br/>        “我想给你说的是，这个项目好像关注度比西城所有项目的关注度都高，现在已经有了十几家做出了相应的方案。”
<br/>
<br/>        “你的意思是.......”江俞轩。
<br/>
<br/>        “有可能。”陈子霖说
<br/>
<br/>        “现在按照前期预计的进行吧，你可以调查一下都有哪些人参加。”
<br/>
<br/>        “光西城过去的不算我们就有四家。后期预审有可能过掉几家。”陈子霖说。
<br/>
<br/>        “如果是按照我们想象的那样，我们的机会怕是不大。”江俞轩想着他们和夏颜之间的纠葛。
<br/>
<br/>        “子霖，你按照正常的程序走，切记一定要按照正规的方式来走，原本这个项目是你姐帮忙介绍的，千万不能因为项目而去做项目，到时候得不偿失。”
<br/>
<br/>        “好的。”
<br/>
<br/>        “做任何一个项目前期都会投入人力和物力，但是不一定都会拿到项目，就算这个项目丢失了，对我们来说也没有太大的损失。”江俞轩叮咛陈子霖。
<br/>
<br/>        “好的。子昂说元旦过后要将耗材推上市场，进行多店联动，还有网络销售体系已经形成，公司内部的销售软件也在等各个网店开通投入使用，耗材的名称想等你来决定。”
<br/>
<br/>        “行，我已经看到子昂发的邮件了，我会回复她的。”
<br/>
<br/>        “那行，我没有别的事情，你还是要注意休息，不能透支健康。”陈子霖现在已经当江俞轩是朋友了。
<br/>
<br/>        “行，我知道了，才发现你是个妈婆。”江俞轩笑着说。
<br/>
<br/>        江俞轩挂了电话，却也没有睡意，邮件堆积了很多，他处理起邮件来，才发现该给寒伯安说手镯的事情也没有说。
<br/>
<br/>        “靠，陈子昂这个怪物化悲伤为动力，一晚上将大哥要做的软件思路都整理出来了？”江俞轩在一个多月之后才发现陈子昂发给他的邮件。
<br/>
<br/>        离陈子昂的耗材推向市场的日子也是越来越近了，用什么名称好呢？
