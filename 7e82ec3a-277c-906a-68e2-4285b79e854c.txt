　　“超脱六道之法？”
<br/>　　闻言，三生道人也不由得为之一叹，脱离六道，安享长生，就算是准圣高手恐怕也做不到吧。洪荒之中，恐怕只有那不死不灭的天道圣人，才算得上是脱离六道。可是，当无量量劫来临，天地覆灭，重归混沌之时，就算是天道圣人，也一样难逃劫数。
<br/>　　能够永恒不朽者，恐怕唯有那传说中人人追求的大道超脱境了。
<br/>　　同时，三生道人也有些感慨，后土不愧是后土，天资聪绝，自己只是稍稍的一点，她便能领悟，想到许多。若是换做自己，绝对做不到这样，自己也不过是仗着后世的先知先觉，才能够在这些事情来临之前有所感悟而已。
<br/>　　“修行！”
<br/>　　三生道人淡淡地吐了两个字，之后便不再多说。
<br/>　　“修行？”
<br/>　　后土微微一愣，但转瞬间她便想明白了，修行的确是一个超脱轮回的方法，但能够真正超脱的人，少之又少，这其中的希望，少到完全看不到。
<br/>　　“若真是如此，日后洪荒众生死后倒是有了归宿。”
<br/>　　后土好似想到了这其中的好处，不由莞尔，“巫族也能吗？”
<br/>　　三生道人微微摇了摇头，道：“这……巫族应该不会。”
<br/>　　“呃，也是，巫族都不修元神，又哪来轮回一说。不过，巫族一旦身死，便是灵识消散，倒是悲苦了些，天道果然平衡，有盈有损。”后土顿时想到巫族肉身强悍，却不修元神，也就不再计较这些。
<br/>　　“建立六道轮回一事，绝非易事。所谓得道失道，只在一念之间，有得必有失，建立六道轮回，需要付出巨大的代价，如此，你还要建立六道轮回吗？”
<br/>　　三生道友看向后土，出声问道。
<br/>　　“当然要建！”
<br/>　　后土眼中带着前所未有的坚定，她紧紧地盯着眼前的三生道人，断然开口道：
<br/>　　“还望道友告知后土具体如何行事？此番若是能够成功建立六道轮回，再来拜谢道友大德。”
<br/>　　三生道人闻言，苦笑一声，避开后土的目光，深深叹了一声，道：
<br/>　　“建立六道轮回一事，非一日之功，道友若是还有其它未了之事，尽管去完成。百年之后，我在此等候道友，共议轮回一事。如何？”
<br/>　　后土本就聪慧无比，听闻此言，已经隐隐有些猜测到后面可能发生的事情，虽然不太详尽，但也差不多了。她也没有拒绝，听了他的话，她的确还有一些事没有放下。她要回了巫族之中，交代一些事情。
<br/>　　“唉……”
<br/>　　看着后土走后，三生道人深深叹了一口气，他不知道这样做到底是对是错。沉默良久，他转身，抬头，望天。
<br/>　　此时，太阳早已落下，四周一片黑暗，远处时有时无的传来几道厉鬼的嘶吼声，声音尖端刺耳。
<br/>　　……
<br/>　　百年时间，转瞬即逝。
<br/>　　“道友，你可想好了。”
<br/>　　三生道人亦如之前后土见到的那般模样，抬头望天，背对众生。
<br/>　　“我已经确定了。”后土坚定的点了点头。
<br/>　　“也罢，既然你决定了，那么我们便开始吧。想必你也明悟了该如何做了吧？”三生道人的声音之中，带着莫名的感叹。
<br/>　　“的确，我已知晓，同时我也知道了道友的顾虑。”后土眼神闪烁，轻声应道。
<br/>　　实际上她在这百年时间中，因为有了之前三生道人指出的方向，也渐渐明悟了自己即将要做的事情，但为了天下苍生，她决不后悔。
<br/>　　“唉……如此，那便走吧。”
<br/>　　三生道人深深一叹，之后转身向着幽冥血海深处走去。在他后面，后土眼神闪烁，跟了上来。
<br/>　　血海是天地负面本源汇聚之所，洪荒大部分怨气，晦气，杀气，煞气都在无形之中向此汇聚。
<br/>　　天地没有轮回，生灵陨落后的怨魂、阴魂都飘散到了血海，后来被冥河老祖将它们聚拢，创造了阿修罗族。
<br/>　　两人一到血海之中，便被正在感悟天机的冥河老祖发现了，他起身踏步出了血海，来到两人面前。
<br/>　　“咦，这不是后土道友与三生道友吗，不知两位来我血海有何要事？”
<br/>　　显然，冥河是认识三生道人的，也对，三生道人说起来与冥河也算得上是邻居了，他们又怎能不认识。
<br/>　　“道友，此番我等欲建立六道轮回，以因果轮回补天道之损，定天地纲常，使那洪荒生灵繁衍兴盛，生灵善恶因果大成。需要道友帮忙。”三生道人也不客气，直接将目的说了出来，他知道冥河不会拒绝。
<br/>　　果然，冥河一听，顿时眼前一亮，大声道：
<br/>　　“好啊，如何建立六道轮回，贫道虽无大才，但也愿出一份力。”
<br/>　　冥河虽然不喜欢在洪荒之中走动，但也也知道这是一件大功德之事，如是做成，自己也就能够得到一部分功德。以自己眼下的修为，功德虽然不能再提升他的修为，但却也可以用来延长阿修罗族的气运。而且，有了这些功德的庇护，只要他自己不作死，洪荒之中就算天道圣人也不敢招惹他。
<br/>　　“六道之中，有阿修罗一道，此事需先向道友告知一声。”后土淡淡道。
<br/>　　“什么，有阿修罗一道？后土道友大慈大悲，泽庇苍生，冥河愿助道友一臂之力。”冥河闻之大惊，但紧接着便是大喜，如此，他那功德之事便再也跑不了。
<br/>　　冥河是怎样的人，两人都很清楚，他那点小心思，自然是瞒不过后土两人，不过，两人都没有点破。
<br/>　　“多谢道友指点，此番后土感激不尽。”后土接着对着三生道人感谢道。
<br/>　　然而，三生道人的兴致却一直不高，见后土如此，他也只是随便应了一声。
<br/>　　后土一步踏出，周身气势爆发，一股浓厚的戊土之气从身体之内散发而出。
<br/>　　“轰隆隆！”
<br/>　　陡然一股浩瀚、神圣而古老的天地威压镇压而下，无尽慈悲、怜悯苍生之意弥漫洪荒。
<br/>　　“哗啦啦啦！”
<br/>　　整个洪荒，突然天降血雨，洒遍大地。
<br/>　　此乃天下大哀，苍生大悲！
<br/>　　“什么？到底发生了何事？天地怎么会有如此剧变？”
<br/>　　“不好，这股慈悲之意，竟然能够影响我的心境，这……这要何等实力才能办到？”
<br/>　　“不行了，我忍不住要哭了！”
<br/>　　“啊！我怎么要哭了，到底发生了什么？”
<br/>　　“这……怎么回事，我的心中好悲痛！”
<br/>　　“不行了。我受不了了，哇！”
<br/>　　“哇啊！哇！”
<br/>　　……
<br/>　　洪荒六圣，顿时脸色大变，看向血海方向。玉虚宫，元始神色复杂，默默道：“你终究还是选择了这条路。唉……”
<br/>　　一声幽幽的长叹，叹尽世间的诸般无奈。
<br/>　　正当洪荒众强者议论纷纷时，一道巨大的身影，法天象地，骤然出现，屹立于天地间。
<br/>　　那是后土的祖巫之身！
<br/>　　“轰！”
<br/>　　后土身上散发出无尽霞光，祥云缭绕娇躯，一缕怜悯苍生、悲天悯人的慈悲已经从她体内爆发而出，弥散到整个洪荒。
<br/>　　这一刻的后土，宛若神邸，威势滔天，相比于天道圣人的气象，有过之而不及。
<br/>　　“天道有缺，众生皆苦。吾乃盘古氏血脉，巫族祖巫后土，今有感天地不全，轮回有缺，为天下苍生之大计，愿以身演化天地轮回，补全天地！”
<br/>　　……
