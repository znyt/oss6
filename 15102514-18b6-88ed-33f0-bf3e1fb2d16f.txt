　　“我的天，这下大魔王失误太大了，刚刚居然没锤石给钩中了，让小鱼人拿到一个人头，接下来他在线上会变的很难打的。”看到叶帝交出了一血，米粒顿时开口说道。
<br/>　　“刚刚如果大魔王在锤石闪现的时候闪现逃跑，他的这个人头是不会交掉的，估计大魔王是想省下一个闪现，这才发生了这样的事情，不过这样的话，中路可就有些难打了。”林雅楠跟着开口说道。
<br/>　　而叶帝这边呢，看到自己交出了一血，他的脸色也是有些难看，当然了他难看的真实意义倒并不是自己交出了一血，而是因为就在刚刚他要使用闪现的时候，他的肩膀传来一阵刺心般的疼痛。
<br/>　　也真是那一下让自己的动作慢了半分，这才让锤石将自己钩中的。
<br/>　　而此刻他的脸色难看，倒是被众人解毒为因为送出一血后，叶帝心里很不爽的缘故了。
<br/>　　这还是叶帝第一次在赛场上伤病复发，好在只是疼了那么一小阵，要不然的话这场比赛能不能够坚持下去还两说呢。
<br/>　　对于自己肩膀的疼痛，叶帝并没有开口说出去，反而是开口安排接下来的动作：“我的了，刚才有些大意了，一会夜雪不要来中路了，我猥琐发育，你多帮香蕉。”
<br/>　　说完之后，叶帝在心里默默的思索着关于自己肩膀上伤病这件事情，之前他已经做过康复治疗了，只不过不能痊愈而已，要不然前几周他也就不会让杨小艺一直上场了，其中正是有着这样的原因在。
<br/>　　今天自己这没有征兆般的疼痛，倒是给叶帝敲响了警钟，看来以后倒是要小心一些了，不能在像之前那么拼了，估计是自己这段时间打游戏打的太多的缘故。
<br/>　　1分40秒，双方的小兵相遇，这一次叶帝倒是没敢继续走上前去补兵，小鱼人因为拿到一血的优势，回到线上的时候已经是一个多兰戒和一瓶腐蚀药剂了。
<br/>　　这个时候如果被对方耗血，那么自己肯定是巨亏的，所以这种情况下自己只能等小鱼人推线，或者是利用自己的q技能补兵。
<br/>　　“这一边大魔王很难打啊，如果是一个远程英雄还好办一些，不过卡萨丁和小鱼人一样，都是近战英雄，而且前期的话，小鱼人本身就比卡萨丁要强，再小鱼人拿到一血的情况下，卡萨丁就更没法打了。”看着中路的架势，米粒开口说道。
<br/>　　“我倒是好奇这一局大魔王还会不会做出冥火这件装备了，卡萨丁这个英雄已经很久没有登场了，就算是冥火出来了也没人用过这个英雄，大家玩的更多的还是妖姬、狐狸这种英雄。”林雅楠跟着开口说道。
<br/>　　而场上的叶帝，这个时候倒是也不着急，反正这一局已经这样了，他心里早就已经断绝了在线上单杀对方的打算了，现在这种情况只要让自己猥琐到六级，那么一切就都安全很多了。
<br/>　　在别的队伍可能中路刚开始陷入劣势后，会呼叫打野过来帮忙，但是在白队、在叶帝这边却是绝对不会出现这种状态，每次遇到这种情况叶帝都会打一个反思维，让夜雪去gank别的路。
<br/>　　而他虽然劣势，但是他本人有着足够的自信能够稳下来，不被对方单杀，加上对方可能心里以为夜雪的打野会照顾中路，所以通常这种情况下他们这边倒是很好打。
<br/>　　小鱼人看到自己能够压制住叶帝这个大魔王自然是非常的兴奋，甚至这一刻在他的内心中已经萌生出了某种想法，也许这一把自己真的可以单杀大魔王呢，如果那样的话俱乐部给我的签约费是不是又要涨了。
<br/>　　想到这里小鱼人打的越发的激进了起来，仗着自己手中有e技能和闪现，他倒是真的不怕夜雪狂歌的打野会过来抓他。
<br/>　　只不过直到5分钟的时候，他依旧是没有找到能够单杀叶帝的机会。
<br/>　　“这一局大魔王打的真的是稳啊，到现在不仅没有被小鱼人消耗掉多少血量，反而是还消耗掉了小鱼人一些血量，不过他这边的补兵倒是被对方小小的压制了一些。”看着中路的对决，米粒开口说道。
<br/>　　“没错，不过在这么大的劣势下，5分钟才被对方拉开8个补刀，经验一点没有落下，大魔王的抗压能给力真的是恐怖。”林雅楠跟着开口说道。
<br/>　　而在时间到达5分钟的时候，白队这边上路的香蕉已经是配合着夜雪狂歌的盲僧击杀了对方的巨魔一次，香蕉手中的鳄鱼拿下一个人头，如今在上路打的异常的激进。
<br/>　　“雅楠你发现白队的套路没，这个套路其实已经见过很多次了，直到今天我才想明白。”米粒对着林雅楠笑问道。
<br/>　　“什么套路？”林雅楠有些好奇的问道。
<br/>　　“白队这边如果有一路突然陷入劣势，其他几路都会打的异常的激进。”米粒有些兴奋的开口说道。

<br/>　　“我明白了，他们这是想要通过其他路的激进来吸引对方打野的注意力，从而让自家那边的劣势路能够有着更好的发育机会，只不过这样的打法是在对自己有着绝对的自信才能够打出来的，如果一个不注意，可能就要被对方的打野过来抓死的。”林雅楠大声的分析着。
<br/>　　“可以看到这一局的比赛中因为上路的巨魔被击杀，RG这边的打野不得不花费很长的时间在上路，只不过每次当他过去gank的话，香蕉总是能够及时的撤退，不得不说这样有位移的英雄，对于地方的打野来说真的是一个噩梦。”米粒继续说着。
