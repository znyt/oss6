　　说完他气一只手，五根葱白一般的手指插进了自己的额头，拔出来之后，额头上多了三个孔洞，往外各自冒出一股气体，分别是红绿蓝三色。
<br/>　　“三尸之气？”李浩然喃喃自语。
<br/>　　天地之间的气息，有清气和浊气之分，晦气、死气、瘴气、尸气、阴气……等等，都是浊气，一切正修的生灵都避而远之，如果浊气浓郁到一定程度，会对生灵造成很大的损伤。但尸族却最喜欢浊气，吸食浊气进行修炼，在僵尸来说是最普遍的事情。
<br/>　　僵尸吸食浊气，原本只能修炼自身，提升修为。据说是当初的僵尸王将臣，以佛法历劫，却意外发现了佛门中最为摈弃的贪嗔痴三毒，反其道行之，用佛门排除贪嗔痴三毒的密宗法术，聚拢浊气，一分为三，化作三尸之气，是世上最为阴毒的气体，不过这秘法后世也只有三大尸王能够修炼。
<br/>　　李浩然也只是听说过相关，还是第一次与真正的三尸之气对抗。
<br/>　　女魃释放的三尸之气，在空中幻化出了各种奇形怪状，最后一分为三，仿佛是三团火光，围着李浩然上下旋转。
<br/>　　李浩然布置剑阵，以剑气化作一个类似半透明的鸡蛋壳，护住周身。
<br/>　　三尸之气的进攻，和李浩然的防守，看上去都无声无息，一点也没有激烈火爆的对攻，但只有李浩然自己知道，这三尸之气有多可怕。
<br/>　　不过这个画面，也是让众人不禁感叹。不愧是尸王女魃，让他这个斩却两尸的强者，也只能努力防守。换做是一般人，只怕早就死无葬身之地了。
<br/>　　道风从人群中飞出，去抵挡另外那些女魃身边的强者，之后空界诸多强者一拥而上。
<br/>　　捉鬼联盟的成员为了保护叶少阳和小九，除了留下小青小白保护芮冷玉，其余全都冲了过去。
<br/>　　“不要进攻，我们在东皇钟左右保护他们就行！提防所有人，包括空界的这些强者！”林三生突然说道。
<br/>　　“什么意思？”四宝不解地问。
<br/>　　“小心有人浑水摸鱼，觊觎东皇钟！”
<br/>　　众人醒悟，立刻依言而行。
<br/>　　“一半人去保护小天师，一半人先抢了转世鬼童！不许伤人！”
<br/>　　马面一声令下，朝小青小白扑了过去。
<br/>　　他们这么做，并不算趁火打劫，恰恰相反，他们内心认为，这是在帮叶少阳——他已经业障蒙眼，无法放弃芮冷玉这个转世鬼童。
<br/>　　但是阴司的态度很明确，一定要将转世鬼童带回阴司，这样就不得不跟叶少阳为敌，双方都很为难。
<br/>　　之前牛头等人也是达成了一致，趁机捉了芮冷玉，押送回去，等叶少阳去找，也救不回来了，他大不了去阴司闹一场，毕竟也不知道芮冷玉在哪，到时候发动大伙一起劝说，甚至把青云子也拉出来，不怕说服不了他。
<br/>　　最重要的是，他们认为，从他手中夺走芮冷玉，和趁他没有发觉，将这件事做了，对叶少阳的心态上完全是两种不同的反应。
<br/>　　毕竟，他会感觉到自己尽力了，但是没能保护的了芮冷玉，也是无可奈何的事，心中的愧疚感也会相应少一些。
<br/>　　所以作为熟人和朋友（其中还有他半个老师），他们这么做，也算是仁至义尽，对得起叶少阳了。只不过他们都把叶少阳想得太普通了一点。
<br/>　　面对阴司抓人，小青和小白只好架着芮冷玉四处躲避，根本不敢交锋，同时召唤伙伴们前来帮忙抵挡。
<br/>　　女魃对李浩然久攻不下，余光看到这一幕，心中一动，一个计谋浮上心头，撇开李浩然，朝着芮冷玉冲了过去。
<br/>　　小青小白等人正在抵挡阴司四神的攻势，女魃的速度又是极快，等到发现的时候，腹背受敌，几乎没有办法抵抗，正恍然之间，突然看到一道清影飞来，是道风，心里顿时轻松了很多。
<br/>　　“道风大大，女魃交给你了！”
<br/>　　小白说完，转身继续应战众位阴神，虽说这些阴神算是自己人，不会真的拼命，但他们一心捉拿芮冷玉，出手还是很狠辣，捉鬼联盟个个都奋力抵抗。
<br/>　　道风一抬手，三清鬼符飞出，暂时挡了女魃一下，自己后发先至，快到芮冷玉面前的时候，突然抽出打神鞭，对着芮冷玉当头砸下来。
<br/>　　他不是来救芮冷玉的，他是来杀她的！
<br/>　　小白和小青回过神来，急忙转身，用清风明月剑布置了一个剑阵。
<br/>　　两人最近修为大增，合力作法，就算是叶少阳都应付不了，不过对道风来说，击杀芮冷玉的机会可能只有一次，因此使出了全力。
<br/>　　剑阵挡住打神鞭之后，只有几秒钟时间，就完全崩毁了，小青和小白也被余波震飞了出去。
<br/>　　“道风大大不要！！”小白无能为力，尖叫起来。
<br/>　　这一声喊叫，也惊动了坐在东皇钟上的叶少阳，却根本来不及阻挡，情急中发出一声凄厉的惨叫：“师兄！！”
<br/>　　道风握着打神鞭的手颤抖了一下，头也不回，喃喃说道：“少阳，你尽管恨我吧……”
<br/>　　打神鞭毫不留情地砸了下去。
<br/>　　所有人都死死盯着道风手中的打神鞭，尤其是牛头马面和黑白无常，十分期待这一鞭子能抽下去，结果就是：一了百了。
<br/>　　转世鬼童若死，他们的任务也就算完成了，也不用再去向阴司交差。
<br/>　　就在打神鞭即将落在芮冷玉头顶的一刻，芮冷玉突然横着掠了出去，险而又险地躲过了这一击。
<br/>　　所有人目光都在道风身上，及至芮冷玉躲开这一击，方才移动目光，转头看去，却不是芮冷玉自己躲过了这一击，而是有个人提着她的胳膊，将她拉了出去。然后一只手抱着她，朝远处飞去。
<br/>　　这人速度极快，来去无踪，多数人根本没有看清这人的模样，只有叶少阳看清了，大吼一声：“后卿！”
<br/>　　在千钧一发之际救走芮冷玉的人，是后卿！
<br/>　　他一直潜伏在周围，把一切交给女魃，自己却在暗中观察着这一切，等到这最最关键的一刻，他终于出手了。
