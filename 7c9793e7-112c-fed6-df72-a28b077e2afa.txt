　　按照他一贯的作风，敌人即使倒下，他也会上前补上一击，以确认自己杀死了敌人。

<br/>　　“呼，奇怪……任老爷子刚刚变异成僵尸，怎么就有了接近百年的功力？”九叔用柚袍擦拭了一下额头上的汗水，皱着眉头嘀咕道。
<br/>　　说罢，九叔蹲下身体，翻看着地上残留的尸首，想找出一些蛛丝马迹。
<br/>　　这件事情，在他眼中怎么看都有些不太正常。
<br/>　　站在九叔身边，刘哔眼神微动，他自然是知道其中原因的，但肯定不会主动说出来。
<br/>　　“叮~恭喜宿主已经完成所有主线剧情，一个月后将会传送离开此位面，请宿主提前做好准备。”
<br/>　　系统的声音再次从脑海中传来，刘哔心中一喜，果然，他的猜想是没有错误的。
<br/>　　只是让他感觉意外的是，他需要一个月的时间才能从这个位面离开，这倒是有些出乎他的意料。
<br/>　　心中挂念着弟弟的安危，他并不想在这里待很长时间，不过也好，他能借着这一个月的时间，将手头上的任务完成。
<br/>　　“阿哔，文才，你们两个重新找一副棺材，把任老爷子的尸体给装进去，要不这种样子，我们也没办法和任老爷交代。”此时，九叔已经检查完地上僵尸的尸体，也没检查出什么不对来，只能吩咐两人给重新装敛一下。
<br/>　　听说任家那边已经找好坟地了，这两天估计就要重新入土，这种样子，未免有些太不像话。
<br/>　　回了回神，刘哔听见九叔的命令，心中一喜，连忙应了下来，他还在挂念，那具僵尸体内的尸丹，百年毛僵本来就不好找，再说他的时间也不充裕，这枚尸丹对他来说也蛮为重要的。
<br/>　　见到这边已经没有了什么问题，九叔环视一圈，从停尸间走了出去。
<br/>　　待九叔走后，刘哔和文才两人找出来了一具空棺材，将任老爷子的服饰整理了一遍，随后放了进去。
<br/>　　“师弟，你一晚上没睡，要不要去睡会，我来守夜？”做完事后，文才端起祭台上的水壶喝了一口清水，有些关切的向刘哔开口。
<br/>　　随手将因为先前战斗而产生的碎木屑收拾到墙角，刘哔摇了摇头，脸上带着微笑：“没事师哥，你先去睡吧，明天记得早点起来换我就成。”
<br/>　　应了一声，文才也不疑有它，在停尸间待了一会，方才打着哈欠走了出去。
<br/>　　见到众人都走了，刘哔稍稍犹豫了一下，转身拿起掉落在地上的画像，回到休息处，平躺在了竹床上。
<br/>　　……
<br/>　　一个多小时过后，刘哔睁开了双眼，从床上坐了起来。
<br/>　　之前他担心九叔没走，或者有什么事在折回来，故此他才没有贸然去收取尸丹。
<br/>　　毕竟在这个世界，尸丹的用途并不广泛，一般都是鬼修才可以用到，他这样小心也是怕引起别人的误会。
<br/>　　虽然同为修士，但鬼修可不如术士那般受人尊敬，反而是人人喊打的身份，毕竟他们的修行手段实在是太过于血腥残忍。
<br/>　　据说，鬼修的起源，是根据僵尸的修行方式改造而来的，绝大多数鬼修的修行功法，都是从别人身上抽取天地灵气或者尸气鬼气，用以修炼。
<br/>　　最初这种功法创造出来，自然是作用于僵尸，鬼怪妖物身上的，那时的鬼修，身份地位甚至比普通修士要高出很多。
<br/>　　但随着世间各种妖物的数量逐渐减少，有一部分鬼修，为了保持自身的实力，开始使用人类修士为修炼源泉，整个修真界，也发生了动荡。
<br/>　　意识到鬼修遗留下来的隐患，人们开始不拿鬼修当做英雄，而是奋起反抗，在不断的冲突摩擦之下，终于发生了一次覆盖整个修真界的大战。
<br/>　　战斗的过程，已经有些不可查了，但是鬼修从那之后，就鲜少出现在人们眼前。
<br/>　　而修真界其他的普通修士，自然也好不到哪去，几乎战损了十之八九的战力，很多传承，也从那时开始断绝了。
<br/>　　甚至到现在，整个华夏的修真界都一蹶不振，否则，又怎么能让那些境外蛮夷，猖狂至此。
<br/>　　也因此，鬼修这个职业，如今更像是过街老鼠一般，只要出现，都会引起整个华夏修真界不惜余力的追杀。
<br/>　　还有一个多月就要从这位面离开了，刘哔可不想蹚这趟浑水，就算是他，也没多大信心从整个修真界的追杀中生存下来。
<br/>　　走到棺材旁边，刘哔抛开脑海中的想法，查看了一下四周的情况，确认安全之后，方才推开了木棺的棺盖。
<br/>　　死去的那头僵尸，平静的躺在木棺之中，因为失去了护体的尸气，整具尸体都已经腐烂的不成样子了。
<br/>　　没花费多大的力气，刘哔就从一滩散发着恶臭的烂肉中，掏出了一颗瓶盖大小的透明珠子。
<br/>　　这颗透明珠子，像是玻璃制成一般，擦去上面的污秽，在烛光下面还挺晶莹，其中更是有一团乌黑的气体，缓缓盘旋着。，
<br/>　　那股黑色的气体，就是僵尸吸食了许多年的日月精华，方才凝聚出来的本命尸气，可惜，对于寻常人来说，这并没什么作用，反而是剧毒之物。
