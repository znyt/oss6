        &lt;h3&gt;第三卷第三百二十四章  春暖花开李长卿&lt;/h3&gt;
<br/>
<br/>        郝景文和寒伯安见江俞轩离开，俩个人相视一笑，都摇了摇头，他们现在都看不懂江俞轩和陈子昂之间的关系了，看来，俩个人都是放不下彼此。
<br/>
<br/>        寒伯安问郝景文：“你家小甜心不来？”
<br/>
<br/>        “来呢，下班就过来了，她们这几天比较忙，可能会晚来一会。”郝景文看了一下时间，北京堵车，陈思宇应该还得半个小时才能到。
<br/>
<br/>        李长卿的女朋友宫阳安静的坐在李长卿的身边，面带微笑，宫阳和寒伯安他们都不熟悉，今天是第一次见面，他们之间的谈话宫阳也插不上话，微微有点生疏，但宫阳一直保持着得体的微笑。
<br/>
<br/>        “长卿，这会没事，不如谈谈你和宫阳小姐姐的恋爱史？”郝景文打破微微沉滞的气氛。
<br/>
<br/>        他们平时在一起是很热闹的，今天因为有了宫阳，大家对宫阳不是很熟，作为寒伯安来说，李长卿是他身边的人，李长卿所结交的人，他都知道底细。
<br/>
<br/>        宫阳是桂林人，人说桂林山水甲天下，而桂林的女孩子长得是非常俊俏，宫阳是单亲家庭，母亲早逝，宫阳的父亲是一个警、察，又当爹又当妈的将宫阳拉扯大，宫阳非常优秀，毕业于某警官学校，在上海上班，这一次也是碰巧到北京出差，李长卿就叫上宫阳和他一起。
<br/>
<br/>        “我们？我才不会告诉你呢。”李长卿胳膊一伸，搂住了宫阳的肩膀。
<br/>
<br/>        “长卿，你肯定是不敢说，是吧，宫阳小姐姐？”寒伯安俊眉一挑，冲着李长卿一笑。
<br/>
<br/>        “怎么不敢说了，我们是不打不相识，打上就相爱！”李长卿嘿嘿一笑，反正他家宫阳是女中豪杰，他曾经被宫阳收拾的五体投地，不过话说回来，像宫阳这样漂亮的女孩子，他才舍不得动手，所以只有挨揍的份。
<br/>
<br/>        “怎么回事，怎么回事，说说。”郝景文八卦心起：“算了，你等一会说，等我家思宇来了也听听。”
<br/>
<br/>        “你是让你家思宇也要学习一下以后怎么让你乖乖的听话吗？如果是这样的话我很乐意等你家思宇来了让阳阳教教她。”李长卿慢条斯理的说。
<br/>
<br/>        “听你这口气，你是被揍听话的，哈哈哈！”
<br/>
<br/>        “我愿意！咋的了？再笑，我就让我家宫阳将你从这扔下去。”李长卿一副我由宫阳罩着的感觉。
<br/>
<br/>        “长卿，我发现你现在真的是小姐姐性格了，你这样好吗？”安男笑着说。
<br/>
<br/>        “宫阳觉得好就好，别的不重要啦！”李长卿宠溺的看着宫阳一笑。
<br/>
<br/>        “好了，别把你说得和苦情媳妇儿一样，你说，我那件事情不是听你的？”宫阳终于是温柔的开口了。
<br/>
<br/>        “我和长卿认识好多年了，那个时候我还在上学，有一天在路上错将他当小偷给揍了一顿，揍得好像挺狠的。”宫阳笑着说。
<br/>
<br/>        “她打我了，我又没有做错什么事情，我就要让她给我道歉，她不愿意，我就跟在她身后，跟了一个多月。”李长卿揉了揉宫阳的短发。
<br/>
<br/>        “他给我当了一个月的保镖，身边的男孩子都被他给吓跑了，然后我就成了他的女朋友。”宫阳温柔的看了一眼李长卿，往李长卿身边又靠了靠。
<br/>
<br/>        “你现在这样我们很难想象你和李长卿打架的样子，要不你们现在表演一下我们看看？”安男也学坏了。
<br/>
<br/>        “要打架我们也是回家打，怎么能当你们的面打呢？打架这种事情不易让外人观赏。”李长卿幽怨的看了一眼安男，他才发现安男小姐姐现在也很腹黑。
<br/>
<br/>        “我们是欣赏你们的功夫，你以为是什么？”寒伯安淡淡的开口。
<br/>
<br/>        “我们低调奢华深藏不露，功夫只有到用的时候才能体现出来。”李长卿这会特别的希望郝景文的女朋友能来，这样他就没有必要置于风暴中心，被郝景文和寒伯安攻击了。
<br/>
<br/>        “就你那三脚猫的功夫还深藏不露呢，长卿，你这自我攻略的功夫渐长啊，连自己的老婆都打不过。”寒伯安当然知道李长卿有几斤几两。
<br/>
<br/>        “长卿，你这功夫确实不行，在寒总身边又不能保护寒总，不如回家种地去，咱家还有几十亩地，修个院子盖个小花园，想一想都很美好呢。”宫阳冲着李长卿甜甜的一笑。
<br/>
<br/>        郝景文和寒伯安只觉得这狗粮糊了他们一脸。
<br/>
<br/>        好在，陈思宇及时到了。
<br/>
<br/>        “大哥，大嫂，文文，不好意思我来晚了。让大家久等了。”陈思宇一个个的打招呼，她不认识李长卿和宫阳。
<br/>
<br/>        “快来，思宇，给你介绍俩个人，这个是李长卿，大哥的助理，这个是李助理的女朋友宫阳女士。”郝景文将陈思宇拉到自己的身边，他看出来了，陈思宇很想坐安然的身边。
<br/>
<br/>        “助理啊，能当助理的都好厉害啊，我们江助理如今都已经是老大了，陈总将所有的事情都交给他了。”陈思宇夸张的比喻着。
<br/>
<br/>        “哈哈哈，长卿，你加油啊，大哥说不定哪一天就将所有的事情交给你了。”郝景文开怀大笑。
<br/>
<br/>        “我才不稀罕呢，劳碌命，天天被人惦记着！”李长卿摇了摇头，陆家这几天黏的很紧。
<br/>
<br/>        还有那个甸城的算算时间估计也会恼羞成怒，来找他们了。
<br/>
<br/>        李长卿的话让寒伯安和郝景文沉默了一下，他们知道李长卿话里的意思。
<br/>
<br/>        “寒总都被惦记了，你也躲不到哪里去吧？”宫阳看了一眼寒伯安问李长卿。
<br/>
<br/>        宫阳的工作性质决定了她总是能在各种场合分辨不同的味道。
<br/>
<br/>        “只要安男嫂子不被惦记就好。”李长卿不以为意。
<br/>
<br/>        “寒总，你和苓家是亲戚吧？”宫阳突然问寒伯安。
<br/>
<br/>        “是啊，我阿姨姓苓。”
<br/>
<br/>        “苓家如今生意做得很大啊，他们的业务你们家有接触吗？”宫阳看着寒伯安。
<br/>
<br/>        “我们是亲戚，之前我们有些业务是会交给他们来做的，这几年我回收了一部分业务，还有一部分业务在他们的手中。”寒伯安并没有打算隐瞒宫阳，这些李长卿都可以告诉宫阳。
<br/>
<br/>        “苓家这几年有没有开展新的业务？你知道吗？”宫阳继续问寒伯安。
<br/>
<br/>        “这个我不太清楚，毕竟我不参与他们的经营，还有就是他们有自己的事情。”寒伯安看了一眼宫阳。
<br/>
<br/>        “好啦，准备吃饭了，你这一问，我感觉你在审问一样。”李长卿伸手碰了一下宫阳的鼻尖。
<br/>
<br/>        “不好意思啊，寒总，我是职业习惯。”宫阳抱歉的一笑。
<br/>
<br/>        “宫阳小姐姐，你是做什么工作的啊？”陈思宇听他们说了半天，终于是逮着机会杜宫阳说话了。
<br/>
<br/>        “她呀，专门打击一些坑蒙拐骗的等乱七八糟事情的！”李长卿替宫阳回答了。
<br/>
<br/>        大家说说笑笑，寒伯安却在心里思量开了。
