　　叶尘也没有想到，他出手帮了这些人，反被当成了别有用心。
<br/>　　不过，对于这些人的看法，他原本也没有放在心上。
<br/>　　……
<br/>　　一晃眼，已经入学好几天，平凡的校园生活，也继续进行着。
<br/>　　入学第二天，新生军训也随之而来，不过无非也就是一些正步、军姿，队列之类的项目。
<br/>　　如果不是因为，叶尘如今的修为，困在了筑基境巅峰，短时间内无法突破，才不会在这里浪费时间。
<br/>　　不过对于他来说，回味一下前世的校园生活，感觉倒也不错。
<br/>　　而每天中午，叶尘依旧会和萧若曦，在学校外的别墅中相见。
<br/>　　曦月的神魂力量，也在一天天变强，重新凝聚元神，已经指日可待。
<br/>　　而且，让叶尘感到欣喜的是，随着曦月的神魂力量越来越强，能够占用萧若曦身体的时间，也随之越来越长了。
<br/>　　在曦月修炼之余，叶尘带她将整个天海市的美景逛了一遍。
<br/>　　在外人看来，叶尘和萧若曦，俨然是一对处于热恋之中的情侣，每天成双入对，甚至学校里，已经开始流传，萧女神已经被这个大一新生同居，两人已经做了不可描述的事情。
<br/>　　之前那些，还对两人的关系，有所质疑之人，也彻底闭上了嘴巴。
<br/>　　而这个结果，顿时让那些萧若曦的爱慕者们忍无可忍，甚至有不少人在论坛上扬言，要给叶尘一个教训。
<br/>　　……
<br/>　　这天上午，
<br/>　　叶尘正在操场军训，忽然心念一动，感应到萧若曦出了大学校门。
<br/>　　以叶尘如今的精神力量，足以将整个天海市覆盖，萧若曦的安危，事关曦月，叶尘一刻也不敢放松，始终分着一道神念在她的身上，稍微有费吹草动，叶尘便会知晓。
<br/>　　起初，叶尘以为，萧若曦只是和同学一起出去玩，便没有太放在心上，可是很快，他发现事情，似乎并没有那么简单！
<br/>　　与此同时，
<br/>　　另外一边，一辆出租车上，
<br/>　　“玲玲，你到底准备带我去哪儿啊？你要是再不说，我可让司机停车了啊！”
<br/>　　萧若曦望着身旁一名身材姣好的女孩，秀眉微蹙道。
<br/>　　原来这女孩名叫白玲，是萧若曦的室友加闺蜜。
<br/>　　就在刚刚不久前，白玲死活要拉着萧若曦出来玩，去哪儿也不说，让萧若曦既好奇，同时又隐隐觉得不对头。
<br/>　　白玲听到这话，顿时急了，连忙拉住萧若曦，可怜兮兮道：
<br/>　　“好若曦，我可以告诉你，但是你要答应我，一定要帮我这个忙啊！”
<br/>　　萧若曦听到这话，眉头顿时皱的更紧了，
<br/>　　“好，我答应你！你快点说，到底出了什么事？”
<br/>　　白玲这才支支吾吾道：
<br/>　　“其实，是朱博文让我把你叫出来的……”
<br/>　　“什么？”
<br/>　　萧若曦听到这话，顿时两眼一瞪，
<br/>　　“玲玲！你怎么可以做？我和他没什么好谈的！”
<br/>　　萧若曦说着，便要招呼出租车司机掉头回去。
<br/>　　白玲顿时急了，
<br/>　　“若曦，你听我说，我也是没有办法呀！”
<br/>　　“你是知道的，我上学期挂了九门课，补考又没有过，照这样下去，铁定是毕不了业的！”
<br/>　　“朱博文是学生会主席，他已经答应我了，只要带你去见他一面，他不但可以保我顺利毕业，还让我当咱们学院的学生会主席！”
<br/>　　白玲说完之后，一脸期待的望着萧若曦，再次恳求起来，
<br/>　　“好若曦，你就帮帮我吧，反正就是见个面而已，朱博文又不会把你怎么样，好不好嘛……”
<br/>　　在白玲的一番恳求之下，萧若曦也顿时有些难以拒绝。
<br/>　　而且她也不相信，朱博文真的敢做什么出格的事情，于是最终点头答应了下来。
<br/>　　大约十几分钟后，出租车在一处私人会所前停了下来。
<br/>　　“就是这儿！”
<br/>　　白玲一脸兴奋，拉着萧若曦下了车。
<br/>　　萧若曦虽然隐隐觉得，事情可能没有那么简单，可是也不认为，朱博文敢把她怎么样。
<br/>　　毕竟，她和叶尘的这层关系抛开不谈，她本身好歹也是广南省首富萧天佐的掌上明珠。
<br/>　　而且，她既然已经答应了白玲，此时也不好再反悔了。
<br/>　　很快，两人进入会所之中，
<br/>　　“若曦，你们来了！欢迎欢迎！”
<br/>　　一名西装革履的英俊青年，笑着走了上来，赫然正是天海大学学生会主席朱博文。
<br/>　　萧若曦眉头一皱，直接开门见山道：
<br/>　　“朱博文，你让白玲把我骗来，到底目的是什么？直说了吧！”
<br/>　　朱博文见萧若曦对自己如此冷淡，甚至还带着一丝厌恶，又想起叶尘和萧若曦在一起的场景，顿时妒火中烧，不过他城府极深，很快便又掩盖了下去，微微一笑道：
<br/>　　“若曦，难道在你心中，我就这么不堪吗？我只是想请你吃顿饭，和你好好聊一聊！”
<br/>　　萧若曦冷哼一声，
<br/>　　“我和你没有什么好聊的，如果没有其他的事情，我先走了！”
<br/>　　说着，萧若曦转身便要离开，却被一旁的白玲拉住，再次哀求道：
<br/>　　“若曦，求求你了，你就和朱学长吃顿饭吧！只是吃一顿饭而已……”
<br/>　　朱博文也跟着插口道：
<br/>　　“是呀！若曦，只是吃顿饭而已，你何必这么紧张呢？”
<br/>　　萧若曦被白玲缠的不行，而且谅这朱博文也不敢把她怎么样，于是便勉强答应了下来。
<br/>　　不一会，
<br/>　　三人来到二楼一个环境幽雅的包厢，朱博文早已经准备好了一桌，精美高档的西式菜肴。
<br/>　　三人入座之后，萧若曦也不动筷，淡淡道：
<br/>　　“朱博文，我只在这里待十五分钟的时间，你有什么话，最好快点说！”
<br/>　　朱博文微微一笑，
<br/>　　“好吧！若曦，那我就跟你说实话吧！我这次请你来，除了想和你共进晚餐之外，其实还有一件更为重要的事情要做！”
<br/>　　说到这里，朱博文微微一顿，脸色也阴沉了许多，
<br/>　　“坦白讲，你和那姓叶的小子在一起，让我真的很失望！”
<br/>　　“我不是不能接受你喜欢其他人，但绝对不能接受自己，败给一个没背景、没实力，只会装逼的loser！”
<br/>　　“他，根本配不上你！”
<br/>　　没等朱博文说完，萧若曦忽然一口打断，道：
<br/>　　“不！朱博文，你错了！不是他配上我，而是我配不上他！”
