　　第479章
<br/>　　借着地形的优势，三千龙字骑一股脑冲杀下去。赵子龙率领的这支骑兵，可以说是蛤蟆城战斗力最强的一支队伍，当然，养活他们也是颇费钱的一件事。
<br/>　　无论如何，骑兵在这个时代都有无可匹敌的战斗力。好钢就要用在刀刃上，钱的事也是一样。
<br/>　　如今这个时代的战斗，程大雷也玩不出什么花样来。无非骑兵冲完了步兵冲，此外就是些天时地利人和什么的。
<br/>　　不过，蛤蟆军和帝国军还是有些不同的，不同之处就在于武器上。
<br/>　　先用投石机投射一轮火药，狠狠的打压了一下敌军的士气。之后，龙字骑冲杀出来，在接近敌军时，开始投掷火雷。
<br/>　　轰然的爆炸声响，敌方的骑兵步兵死伤一片。其实目前蛤蟆城掌握的技术，火药的威力也无所谓有多强，只不过动静是吓人的。轰隆、轰隆的声响，震的敌军人人胆寒。
<br/>　　赵子龙的队伍在敌方大军中趟开一条路来，几乎没遇到怎样的抵抗。在他们后面，是手持连击驽的鱼字军，弩箭上都抹着剧毒，又是蛤蟆城炼金术师的手笔，堪称见血封喉。
<br/>　　在弩箭射尽之后，他们也不装填，挥舞着大刀便冲杀了出去，将魂飞丧胆的帝国军赶尽杀绝。
<br/>　　蛮字军、张字军，豹字军都同时冲杀了出去，这次蛤蟆军全军出击，一个不留。
<br/>　　程大雷骑在黑牛背上，左边是扛大旗的银眸，右边是举着金顶枣阳槊的单雄信，暂时没有差事给单雄信做，他便做了程大雷的亲卫队长。喜字队这次也出动了，手中拿着长刀、铁棍、狼牙棒之类的武器。
<br/>　　倾巢而动，程大雷刚发动，便跑得没影了，身后的喜字队根本跟不上他的背影。大家乱糟糟一片，只能跟着银眸所举的大旗，在乱军中冲杀。
<br/>　　程大雷坐下的墨玉乌月兽果不愧有异兽之名，野蛮冲撞的技能施展开来，敌人没办法形成有效的阻击。
<br/>　　程大雷挥舞着大斧子，一轮乱杀乱看，杀得身上鲜血淋淋。也就是银眸和单雄信勉强能跟得上程大雷的行动。
<br/>　　“大当家，大当家……”单雄信口中大叫。
<br/>　　敌军的前锋力量已经被击溃，中军也形不成有效的战斗力，他们的大部队已有逃跑的迹象。单雄信在询问程大雷的下一步行动计划。
<br/>　　程大雷目光转了一圈，窥见了李星的帅旗，手中大斧一挥，口中道：“走，去砍他的脑袋！”
<br/>　　乱军丛中去上将首级，这是每个武夫的梦想，单雄信听罢程大雷的话，明显亢奋起来。在程大雷的带领下，向敌方的中军冲了过去。
<br/>　　李星在今天这一战，也是亲自上阵指挥，战斗刚开始，便成这般局面，完全出乎他的预料。
<br/>　　敌人的骑兵在队伍中趟来趟去，爆炸声不断响起，己方的军心彻底被击溃，已经有人开始逃跑。
<br/>　　这些士兵都来自于扬州，江南富庶之地，士兵也是养尊处优，至于战斗力么……在帝国各路大军中，属于战斗力较差的一支。
<br/>　　“大人，现在该怎么办？”
<br/>　　眼看程大雷以势不可挡之势冲杀过来，李星的亲兵队长也有些着急，再耽搁下去，让程大雷冲过来，李星一颗脑袋就要报销了。
<br/>　　李星恶狠狠的咬紧牙，道：“往北走，相府和将军府在哪里布了口袋，在哪里将他们一网打尽。”
<br/>　　主帅一逃，士兵几乎已无继续战斗下去的欲望，他们开始跟着帅旗溃逃，士兵有些跟不上的，便被如狼似虎的蛤蟆军咬着尾巴杀了，而大部分人，则四散溃逃。
<br/>　　赵子龙负责追杀李星率领的溃兵，其他人七杀八杀，饮马坡下尸横遍野。
<br/>　　李星逃得想没头苍蝇一般，跑得气喘吁吁，身后的追兵如跗骨之蛆，似乎不杀了自己不罢休。
<br/>　　“伏兵呢，伏兵呢……”
<br/>　　李星握着剑大吼大叫，他已经抵达事先商定后的地点，可面前根本没有将军府或相府的伏兵。有的，也只是身后叫喊叫杀的蛤蟆军。
<br/>　　李星恍然意识到一件事：自己被相府和将军府的人卖了。他们根本没有按照事先的计划行动，至于他们，肯定有更诡异的阴谋，但他们究竟背着自己在搞什么阴谋，李星完全不知情。
<br/>　　一个棋子有什么资格知道布局人的想法，而自己和这五万扬州兵，不过是随手可丢掉的棋子而已。
<br/>　　“大人，咱们现在怎么办？”
<br/>　　李星双目充血，咬牙切齿恨到极致。他忽地一咬牙：“回扬州！”
<br/>　　再也不和你们这些混了，你们愿意怎么打，就怎么打好了。
<br/>　　“大当家，大当家……”
<br/>　　程大雷挥舞着大斧头正厮杀得痛快，高飞虎领着他的手下冲杀了过来。
<br/>　　“黄沙城的情况怎样了？”
<br/>　　“相府和将军府的人偷袭黄沙城，现在已经将黄沙城占了。”
<br/>　　“啊，果然是这样，我就知道那帮龟孙子不会闲着。”
<br/>　　“大当家，下一步该怎么办，我们身后还跟着追兵。”
<br/>　　“走，收兵，扑回去。”
<br/>　　程大雷开始集结兵力，追杀逃兵的队伍又撤了回来，程大雷将队伍集结在一起，转过身，冲着追击高飞虎的帝国军冲杀了过去。
<br/>　　追兵被击溃，程大雷率领大军直扑黄沙城。
<br/>　　趁着李星和程大雷决一死战的时候，相府和将军府的队伍悄悄绕路到后方，一举攻破了防守空虚的黄沙城。
<br/>　　几乎没有遇到任何实质性抵抗，他们便占领了程大雷的大本营，时间算起来，正是李星调动队伍开始攻打程大雷的时候。
<br/>　　黄沙城本就是一座小城，但此刻却挤满了人，帝国军在今夜攻占此地，这个时候正灯火通明。
<br/>　　他们清点战利品，据得到的情报，程大雷近些日子搜刮来的粮草物资，都存在黄沙城。他们命人清点物资，看还剩下多少。程大雷抢了他们好多次，他们终于可以抢回来一次。
<br/>　　在帝国军上上下下，都因这件事而欢欣鼓舞的时候，李行哉的心情却怎样都开心不起来。
<br/>　　李行哉总觉得，这场胜利来得太过轻易了。
<br/>(本章完)
