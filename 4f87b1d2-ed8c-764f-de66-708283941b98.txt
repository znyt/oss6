<br/>　　程大雷轻咳一声，问道：“樊家姑娘，你我往日无冤，今日无愁，突然来挑我山寨，不知所为那般？”

<br/>　　“哼，凭你们几个小毛贼，也想胜过姑奶奶，我若输了，一条命随你处置。”
<br/>　　“你这命我可不要，要也是你这个人。”程大雷笑眯眯道。
<br/>　　“登徒子！”樊梨花眉放寒光。
<br/>　　“子龙贤弟！”程大雷冷声喝道。
<br/>　　“子龙向来不与女人动手。”赵子龙。
<br/>　　“呃……”程大雷被噎得够呛，目光看向关鱼：“二爷，不若你试试？”
<br/>　　“呵，哈哈，呵呵。”关鱼扶须笑道。
<br/>　　“二爷，定是觉得她是土鸡瓦狗，如插标卖首？”程大雷。
<br/>　　“不不，唯女人与小儿难养也，关某不屑与女人动手。”
<br/>　　“呃……”程大雷无语，这两个人该不会是怕丢人故意找的借口吧。
<br/>　　“小娘皮休狂，二爷爷前来会你。”
<br/>　　忽然一声高喝，高飞豹拍马横刀而来，程大雷还没琢磨过味来，二人已经战在一起。
<br/>　　程大雷对高飞豹的实力还是有一定了解的，他重点是关注樊梨花。樊梨花的绣绒刀一出手，程大雷就知道她是玩花刀的，高飞豹用的则是重刀。
<br/>　　花刀讲快，讲乱，高飞豹讲势沉力猛，讲一力降十会。
<br/>　　高飞豹打来打去，不过抡、砸、摔三个字，而樊梨花的刀法则复杂的多，身子在马鞍桥上闪转腾挪，刀光抖开，真如千树万树梨花开，打得煞是好看。
<br/>　　果然，她抖了一阵刀花，将高飞豹的大刀圈进来，随着刀柄在高飞豹肩上一砸，直接把高飞豹砸落马下。
<br/>　　“小娘皮，你……”高飞豹咬牙切齿，从地上站起来还要再战。
<br/>　　“高二退下。”程大雷喝了一声：“输了就是输了，没什么输不起的。”
<br/>　　败在一个女人手中，高飞豹有些咽不下这口气，他目光落在张肥身上：“张三爷，你还不动手做什么？”
<br/>　　“和一个女人动手，不太好吧？”张肥不想动手。
<br/>　　“嗳，是兄弟就该同荣共辱，你不会看兄弟我这样不管吧。”高飞豹的意思是，是兄弟就该一起丢人，要么就一起被女人打趴下。
<br/>　　张肥想了想，翻身上马，纵马攻来，眨眼便与樊梨花打在一处。
<br/>　　程大雷一看，不由得吓了一跳，看三爷这状况有些堪忧呐。
<br/>　　三爷是传说潜质的人，拼潜力他是胜过樊梨花的，只不过二人目前同级，而张三爷上场时又有些轻敌，小窥了这个女人。
<br/>　　高手相争，轻敌是大忌。
<br/>　　张三爷吃了这方面的亏，竟被樊梨花拍落马下。
<br/>　　啧啧……
<br/>　　程大雷撇撇嘴，还真小看了这个女人。
<br/>　　高飞豹心里平衡了，和张肥、秦蛮三个蹲在一起画圈圈。
<br/>　　连胜蛤蟆寨三员大将，樊梨花骑在马上身材飞扬，两腮微微起伏。
<br/>　　“好啦。”程大雷让人牵了一匹马来，翻身上马：“本大家亲自会会她吧。”
<br/>　　高飞豹把程大雷的大斧递过来：“大当家，我们三个等你。”
<br/>　　“嗳，什么话。”程大雷白他一眼：“和一个女人动手，还用得着斧头么？”
<br/>　　程大雷只有三板斧，未必对付得了这女人的繁杂的梨花刀。不过，已经有三人消耗这女人体力，只要自己想办法把战斗时间拖长，耗尽她的体力，那就怎么打怎么都有了。
<br/>　　所以，程大雷只动用了匹夫剑，而且剑未出鞘。
<br/>　　樊梨花马上凝眉，英气逼人：“狗贼，你看不起你姑奶奶么！”
<br/>　　“哈哈，和一个女人动手，本大人怎么能动用全力，真若用上斧头，你一招都接不了。”
<br/>　　“狗贼，你太狂妄了，吃姑奶奶一刀！”
<br/>　　一记抽刀法向程大雷劈来，不是亲自对敌，程大雷不了解这个女人的强悍，有如此凌厉的刀法，秦蛮、高飞豹、张肥三人输得并不冤枉。
<br/>　　他们输了，程大雷可不愿意输，很丢人的好不好。

<br/>　　程大雷也不强攻，他打了拖延的主意，反正时间越长，对程大雷来说越加有利。
<br/>　　果然，樊梨花呼吸越来越重，刀法越来越慢，程大雷却是游刃有余，一边出剑，口中还语重心长道：
<br/>　　“姑娘，我劝你还是回去把，姑娘如花似玉，在下怎忍心辣手摧花，唐突佳人。”
<br/>　　樊梨花有哭说不出，她今日的确有些托大，单枪匹马就敢来挑人家山寨，不说其他，就是对方用车轮战也能将她累死。

<br/>　　她的刀法突然变慢，可招式却更加复杂，节奏一变，程大雷竟有些反应不过来。梨花怒放，忽然变成和风细雨，但刀幕密不透风，程大雷的剑刺不进去。
<br/>　　刺不进去，就无法限制住樊梨花，忽然间，樊梨花手中的刀一变，变刀柄刺向程大雷咽喉。
