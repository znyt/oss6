　　这毒尸，不知疲倦一般，连续不断的向着楚风眠轰击而来。
<br/>　　不过楚风眠已经是熟悉了这毒尸的战斗模式，只是纯粹直白的蛮力，楚风眠应对起来，也是游刃有余。
<br/>　　轻而易举的躲避了所有的攻击，不过比起这毒尸的攻击。
<br/>　　他身上散发出来的剧毒，才是无比棘手，以至于这毒尸一旦是靠近，楚风眠便是就会急忙的退后。
<br/>　　纵然是楚风眠有着半步不朽真身，还有吞天秘术这样的依仗。
<br/>　　但是也未必可以完全抵挡住这剧毒，这剧毒，一旦是触碰一点，都足以让楚风眠陷入到万劫不复的境地，楚风眠也不敢小视。
<br/>　　这天元商会的会长，究竟是修炼了什么毒功？
<br/>　　楚风眠躲过这天元商会会长的一次次轰击，也是不由的眉头一皱。
<br/>　　这天元商会的会长，绝对是一位半步天人境的强者，距离天人境，只怕就只有一步之遥了。
<br/>　　可这样的一尊强者，居然是被这毒功反噬，化为了一尊毒尸。
<br/>　　不过也幸好是这天元商会的会长已经是成为了没有意识的毒尸了。
<br/>　　不然的话，楚风眠想要对抗一尊半步天人境的强者，几乎是没有丝毫的可能。
<br/>　　楚风眠连连闪动，手中剑锋已经是斩杀了这毒尸数百次，将这毒尸斩杀的千疮百孔。
<br/>　　同时这毒尸的力量，也已经是越来越弱。
<br/>　　“该结束了！”
<br/>　　“三阳！”
<br/>　　轰隆！
<br/>　　纯阳天火，凝聚在楚风眠剑锋之上，一剑斩出，漫天火海直接笼罩在了那毒尸之上。
<br/>　　只见这一尊毒尸，在火海之中疯狂的咆哮挣扎，但是身形还是一点点的被燃烧殆尽。
<br/>　　火焰一点一点的燃烧，那毒尸也被燃烧殆尽，最终化为了灰烬，在那毒尸的地方，只剩下了一枚碧绿色的晶体。
<br/>　　“居然还有残留？”
<br/>　　楚风眠看到这碧绿色的晶体，眼中闪过几分惊讶。
<br/>　　纯阳天火，足以焚烧一切，这毒尸在坚韧，也逃不出纯阳天火的灼烧，可现在这碧绿色的晶体，居然是足以从纯阳天火灼烧之下，完好无损。
<br/>　　楚风眠走了过去，刚刚靠近这碧绿色的晶体，一股毒雾，猛然之间的就是从这碧绿色的晶体之中，喷涌而出。
<br/>　　轰！
<br/>　　楚风眠手心凝聚，一股吞天之力笼罩在面前，将这一团毒雾，全部挡住。
<br/>　　“这碧绿色的晶体，就是毒源？”
<br/>　　楚风眠注意到，这周围的毒雾，应该都是从这碧绿色的晶体之中，散发出来的。
<br/>　　那天元商会的会长，也是被这碧绿色的晶体所掌控，成为了一尊毒尸。
<br/>　　“碧绿色，龙鳞状，这难道是毒龙晶？”
<br/>　　楚风眠观察了一眼这碧绿色的晶体，看到样子像是几分龙鳞状，不由的说道。
<br/>　　毒龙，是龙族之中的一种异类，天生便是就带有无比恐怖的剧毒。
<br/>　　虽然是龙型，但是被一直以来却是被龙族所排斥，天地之间的毒龙数量，不足数百，所以楚风眠也没有见过。
<br/>　　至于毒龙晶，则就是毒龙身躯之中一切毒素的毒源，据说毒龙，许多曾经都是其他龙族。
<br/>　　后来发生身躯发生异变，长出了毒龙晶，便是就化为了毒龙。
<br/>　　毒龙晶是毒龙的本源，也是天下间最为厉害的剧毒之一，所以天下之中许多的毒修，都想要得到毒龙晶，去修炼毒术。
<br/>　　楚风眠曾经在机缘巧合之下，也曾经得到过一枚毒龙晶，只是那枚毒龙晶，才不过拇指大小。
<br/>　　可眼前这碧绿色晶体，足足是有着拳头那么大。
<br/>　　所以虽然楚风眠一眼看过去，觉得这碧绿色的晶体，有点像是毒龙晶，可是却不敢确认。
<br/>　　毒龙的境界越高，身躯之中的毒龙晶也会越大，毒素也就越强。
<br/>　　楚风眠曾经见到过的那一枚拇指大小的毒龙晶，应该是属于一名真帝级别的毒龙的。
<br/>　　可是根据记载，古帝级别的毒龙晶，也就是两指，三指大小。
<br/>　　可眼前这一枚毒龙晶，可是足足一个拳头大小，这样庞大的毒龙晶，这毒龙生前，得是多么强大。
<br/>　　只怕是最弱，也是一尊半步天人境的毒龙，甚至是达到了天人境的毒龙，才有如此巨大的毒龙晶。
<br/>　　“毒龙晶，是以纯粹的灵力所化，虽然剧毒无比，但是能够化解其中的剧毒，其中便是都会转化为精粹的灵力，足以借助冲击境界所用。”
<br/>　　楚风眠看着眼前的毒龙晶，也猜测出了一二，那天元商会会长，为何会落到如此下场。
<br/>　　毒龙晶珍贵无比，不止是其中蕴含的剧毒，而是毒龙晶，也是一条毒龙一生力量的凝聚。
<br/>　　可以说其中蕴含的灵力，比起任何的古圣丹，都要精粹的多。
<br/>　　所以在远古时代的记载之中，也有一些强者猎杀了毒龙晶，以独特手段，抹去了毒龙晶的剧毒，或者以手段抵挡住毒龙晶的剧毒，借助毒龙晶，一举突破。
<br/>　　那天元商会的末代会长，实力只怕是已经达到了半步天人境的巅峰，只差一步就可以踏入帝尊境。
<br/>　　苦于无法突破，所以才会铤而走险，想要炼化了这毒龙晶，一举踏入天人境。
<br/>　　只是可惜这天元商会的会长，有些小看了这毒龙晶，同时也是自己没有做好准备。
<br/>　　以至于被毒龙晶反噬，成为了一尊毒尸。
<br/>　　“毒龙晶是好东西啊，不过却也是一把双刃剑，稍有不慎，就是万劫不复！”
<br/>　　楚风眠看着那毒龙晶，目光闪烁。
<br/>　　这毒龙晶之中蕴含的灵力，只怕是比的上数十枚，甚至是上百枚原始圣丹蕴含的灵力
<br/>　　吞噬了这毒龙晶，楚风眠有着百分之百的把握，足以突破帝尊境。
<br/>　　甚至是踏入帝尊境之后，都还可以得到无比巨大的好处。
<br/>　　“吞天！”
<br/>　　楚风眠打量了一眼这毒龙晶，突然间运转起吞天之力来，一把对着那毒龙晶抓了过去。
<br/>　　这毒龙晶之中的剧毒无比可怕，就连半步天人境的强者，都无法炼化。
<br/>　　不过现在，楚风眠却是打算要炼化了这毒龙晶，楚风眠有着吞天之力，足以化解这毒龙晶之中大部分的剧毒。
<br/>　　而且在加上楚风眠拥有的半步不朽真身，炼化这毒龙晶，未必没有可能。
