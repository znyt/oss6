　　慕容家府邸。
<br/>　　原本的紫云城三大家族之一繁盛的府邸，此刻如同废墟一般。
<br/>　　到处是打斗的痕迹，以及被庞大真气肆虐过的满目苍痍。
<br/>　　当然，还有遍地的慕容家族人尸体。
<br/>　　虽然萧家和慕容家大战，但死者为大，萧家子弟凭着良心，没有破坏他们的尸体，连原本弃尸荒野的慕容墨、慕容山等人的尸体也被拉到了这里。
<br/>　　算是给他们死后有个落叶归根之所。
<br/>　　萧家子弟做到这个份上，已算仁至义尽。
<br/>　　浓郁的血腥味，恶心的场面，让得平日里没人会来这里。毕竟值钱的东西早就被萧家全拿走了，连小偷都不会光顾这里了。
<br/>　　但这时，一道人影却忽然翻墙而进。
<br/>　　人影正是萧逸。
<br/>　　萧逸走到逐个尸体面前，太阴太阳之眼出现，将所有已经死去的慕容家族人的武魂力量吸收掉。
<br/>　　他现在迫切需要力量，迫切想要提升修为。
<br/>　　正常来说，黄阶以下武魂，由于太弱，武者死后，武魂便会在十天内自动消散于天地间。
<br/>　　如今时间还未到十日。
<br/>　　吸收罢武魂，萧逸眉头一喜。
<br/>　　慕容墨、两个他带来的陌生中年武者，慕容山还有四个慕容家长老的武魂，皆是黄色武魂。
<br/>　　另外，在精英执事中，也有两个黄色武魂。
<br/>　　总共十个黄色武魂。
<br/>　　控火兽武魂，竟在一瞬间突破到绿色。
<br/>　　之后，剩下的慕容家子弟，皆是橙色和赤色武魂。虽然武魂等阶低，但胜在数量多。
<br/>　　控火兽武魂的绿色，稍微深色了一些。
<br/>　　赤橙黄绿青蓝紫七色武魂，绿色武魂已是非常强大，起码在紫云城中是最强的。
<br/>　　这一次，萧逸观察了一下冰鸾剑武魂。
<br/>　　发现，原本就深紫色的冰鸾剑，颜色没有多大变化。
<br/>　　果然，对于最强的紫色武魂来说，这些黄色以下武魂根本起不了什么作用。
<br/>　　当然，萧逸已经很满意了，笑了笑，转身离去。
<br/>　　.....
<br/>　　回到萧家，众长老早就在等他了。
<br/>　　三长老将一个乾坤袋交给他，关怀地说道，“逸儿，这是给你准备的各种物品。有丹药、银两等等，还有一把武器，紫云精铁剑。”
<br/>　　“你虽天赋极高，但奈何天意弄人，竟让你觉醒了最低等的赤色武魂控火兽。之前我曾见你用火焰凝聚出火剑来战斗，今日便特地给你准备了更好的武器。”
<br/>　　一般的紫云铁剑，售价百两，是不少家族子弟的标配武器。
<br/>　　而紫云精铁剑，由紫云铁精炼而成，锋利坚固得多，售价千两，算是紫云城最好的武器了，一般后天执事才用得起。
<br/>　　萧逸接过乾坤袋，道谢一声，“谢三长老。”
<br/>　　这时，大长老也递出一个乾坤袋，道，“萧逸，这是你让我帮你找的培源朱果。这东西是三品药材，难找得很，只有三个。”
<br/>　　“谢大长老。”萧逸同样道谢一声，他早就想炼制那金脉丹了，奈何找不到培源朱果。
<br/>　　“先别急。”大长老摆摆手，继续道，“里面还有一个炼药炉，名为离火金炉，乃是紫云城唯一一个四品炼药炉。当年我年轻时，在外历练偶然所得，陪伴了我大半辈子，如今送与你。”
<br/>　　“希望它能在外助你一臂之力。”
<br/>　　“四品炼药炉？”萧逸吓了一跳。
<br/>　　炼药炉，乃是炼药师的左右手，对丹药的成功率、炼制速度等等，起着很大的作用。
<br/>　　所以炼药炉非常珍贵，平日里也极难买到有品级的。
<br/>　　像鎏金炼药炉，那是不入品的炼药炉，较为多人用。
<br/>　　三旋火炉，已是紫云城明面上最珍贵的炼药炉，也不过是一品。
<br/>　　甚至其他两大家族的三品炼药师，用的也不过是二品炼药炉。
<br/>　　至于三品炼药炉，紫云城中从未出现过。
<br/>　　萧逸万万没想到，大长老的贴身炼药炉，竟然是一个珍贵的四品炼药炉。
<br/>　　正常来说，四品炼药师都不一定能拥有四品炼药炉呢。
<br/>　　实在是入品级的炼药炉太过珍贵了，有钱也买不到，大长老萧离火也是年轻时机缘巧合才得到的。
<br/>　　“谢大长老。”萧逸重重地鞠了一躬。
<br/>　　“这就高兴成这个模样了？”大长老乐呵呵地笑道。
<br/>　　“嗯？”萧逸微微疑惑。
<br/>　　这时，大长老再次拿出一物，递给萧逸，那是一双透明的手套。
<br/>　　“这是？”萧逸疑惑地问道。
<br/>　　大长老道，“下品灵器，烈焰手套。乃是我们萧家祖先留下的东西，相传是在陨星山脉中一处极热之地，取黑玉冰蚕蚕丝制成。”
<br/>　　“入手冰凉柔顺，虽轻柔软薄，却刀枪不入，坚硬如斯。戴于手上，可自动藏于皮肤之中，如若无物。”
<br/>　　“使用时，可调动真气触发，大幅度增强控火能力。”
<br/>　　萧逸接过烈焰手套，顿时一惊，“我们萧家竟然有灵器？”
<br/>　　灵器，这是炎龙大陆上最珍贵的东西，没有之一。
<br/>　　因为，它是能最直接增强武者战力的东西。
<br/>　　灵器又分四品，下品、中品、上品和极品。
<br/>　　虽然只是下品灵器，但却比四品炼药炉还要珍贵。
<br/>　　绝对能让武者提升数倍战力，达到越阶战斗的效果。
<br/>　　萧逸敢说，若是自己使用这烈焰手套，就算不用任何武技和招式，都能轻易和先天武者战斗。
<br/>　　若是用上武技和招式，战力之强就更不用说了。
<br/>　　但也因其珍贵，整个紫云城都没有，连盛宝商行也不见有卖的。
<br/>　　萧逸忽然发现，萧家的宝贝真不少，紫云洞、妖兽雕像还有这烈焰手套，都绝不是萧家这样的家族能拥有的。
<br/>　　想必，许多年前的萧家，比现在强盛得多吧。
<br/>　　大长老见萧逸那惊讶的模样，不禁笑道，“臭小子，你以为我区区半步洞玄的修为，当初是怎么突破慕容墨的灵气牢笼的，还不是靠这东西。”
<br/>　　“这本是萧家历代家族使用的武器，现在传于你。”
<br/>　　萧逸摇摇头，拒绝道，“还是大长老你留着吧，若是日后萧家再遇到强敌，你也能凭此护萧家周全。”
<br/>　　“拿着吧。”大长老道，“我感觉，自己用不了多久就能正式突破到洞玄境。而下品灵器对洞玄境武者的增幅已经很有限了，有跟没有差不多，要是中品灵器还差不多。”
<br/>　　说着，大长老强行将烈焰手套放到萧逸手中。
<br/>　　萧逸心中不禁淌过一丝暖流，他当然知道大长老此番话语只是想让他安心拿走烈焰手套。
<br/>　　下品灵器，终归是不可多得的武器，有跟没有还是存在区别的。
<br/>　　就算对洞玄境的增幅没有数倍那么多，但起码也有数成，比没有要好。高级武者间的战斗，往往差了一些，就会影响胜败。
<br/>　　“既然如此，我便收下了。”萧逸没有说破，而是收下，好让大长老放心他出外历练，心中也铭记着今日的恩情。
<br/>　　“说什么傻话，这本来就该属于你的。”大长老摆摆手，示意萧逸不用多想。
<br/>　　“但你记住了。”大长老认真地嘱咐道，“下品灵器乃是重宝，不可轻易外漏。若无生命之危，不要轻易动用。一旦动用了，必须灭口，免得惹来不必要的麻烦。”
<br/>　　“外面的世界，人心，可比家族内复杂多了。”
<br/>　　萧逸点点头，道，“我晓得的。”
<br/>　　大长老嘱咐完，二、四、六长老也相继拿出一个乾坤袋，递给萧逸，都是些价值不菲的材料、丹药等等，还有银两。
<br/>　　“我们能帮你的，就这么多了，你这次离开萧家，前往裂天剑派，务必要小心。”众长老严肃地嘱咐道。
<br/>　　“还是那句话，萧家有你，便有未来，务必事事小心谨慎。”
<br/>　　众长老，就像在嘱咐自己要出远门的孩子般，满是关爱和嘱咐。
<br/>　　“我知道了。”萧逸点点头，鞠了一躬，认真道，“诸位长老，等着我，逸儿一定会扬名北山郡，回来之日，定叫他人不敢再欺辱我萧家分毫。”
<br/>　　说罢，萧逸转身离去，身影闪烁间，已经离开了萧家府邸。
<br/>　　.....
<br/>　　府邸门前，众长老迟迟没有离去，久久看着那道早已消失的身影，而后蓦地叹了口气。
<br/>　　“今后的萧家，日子怕是难熬了。”二长老满是忧色。
<br/>　　“北山慕容家不会明面上灭我们，但一句话就能将我们打压得我们喘不过气来。”三长老同样忧愁至极。
<br/>　　大长老眼中闪烁着智慧的光芒，道，“再难熬，也要熬过去；再被打压，就算憋屈，也要挺过去。”
<br/>　　“我们萧家，还有萧逸。萧逸归来之时，便是萧家崛起之日。”
<br/>(本章完)
