　　拦住了她去路的是两个人，一个年轻女人，和一个年级不过十一二岁的少年。
<br/>　　正是白允儿，和她的小跟班白羽。
<br/>　　“我也觉得甚是巧合。”白允儿口气平淡。
<br/>　　“说起来，我怎么每次出门都会遇到你？”高漫秋淡笑着开口，不见任何其余情绪，就好似真的只是出门闲逛时遇到熟人。
<br/>　　总之嫡系一般不出动，一旦出动都不可能是闲逛，而任何两家神秘家族的嫡系碰面，往往都伴随着某件大事发生。
<br/>　　但眼下如高漫秋和白允儿这般，在倾巢大战时的银市遇上，又在分界线大战的战区遇上，此刻甚至还在段氏基地的屋顶遇上。
<br/>　　怎么看怎么不同寻常！
<br/>　　白允儿难得一见的一笑，只是笑容里依旧带着冰冷：“大概是你我关心的事，恰好又是同一件吧。”
<br/>　　高漫秋纤细的柳眉微微一抬，典雅的神态中一丝锋芒炸闪：“你去看过楚涵了？速度真是快啊！”
<br/>　　白允儿与之直视：“想必你也是吧？”
<br/>　　高漫秋眼底的笑意都快溢出眼眶，相当于是默认的开口道：“白家真是难得出了一个你。”
<br/>　　白允儿冷然反驳：“我从不认为我是白家人。”
<br/>　　南署基地的大尸潮并不是神秘家族所关心的事，但楚涵却近乎是所有神秘家族的重点关注对象，他的昏迷近乎在第一时间就被很多人知晓。
<br/>　　所以白允儿速度极快的去了狼牙基地，虽然她向尚九娣承诺楚涵没有大碍，但却只是她感受到楚涵体内的能量一切正常的情况下。
<br/>　　为何昏迷，何时醒来，这其中隐藏了什么？
<br/>　　一切都需要从大尸潮的战役中调查，所以理所当然的，白允儿的第二站就是段氏基地。
<br/>　　当下大尸潮的战役已经结束，段江伟理应知晓其中过程。
<br/>　　而高漫秋在段氏基地的出现，以及她在段江伟面前问出的那些话，则同样证明了白允儿所猜测，只不过依照高漫秋贵为神女的战力，会比白允儿快一步也理所当然。
<br/>　　更何况白允儿还在狼牙被某些事绊住了脚步……
<br/>　　也就是说两女都去过狼牙基地，亲眼见到了楚涵，而后又来到了段氏基地想打听某些情报。
<br/>　　能让神秘家族两家的嫡系第一时间双双出动，足以见得此番楚涵的昏迷，本身就不同寻常。
<br/>　　“不是白家人……”高漫秋轻声道念着这句话。
<br/>　　这不是她第一次听到白允儿这么说，每一次白允儿都说的很坚定，也很直接，丝毫不顾及旁边白羽的心情。
<br/>　　高漫秋的心思顿了顿，看向白允儿的眼神意味不明：“你说自己不是，是否客观？你的身旁站着白家某旁支的继承人，想必那一支脉是追随于你的吧？而这一脉的旁支在白家，地位可不低，尤其是在末世爆发之后，战力能进前三。”
<br/>　　“有这么强大的旁支支持，其他旁支怎么会没有动静？”高漫秋步步分析着，将白允儿的话分解拆开：“你是白家的返祖者，他们支持你都不支持当了这么多年少族长的白忧，可见白家内部也并非如外界传闻的那般和睦。”
<br/>　　“这些都跟我没有关系。”白允儿冷言回答，双目透露着漠视。
<br/>　　高漫秋闻言一笑，目光转移看向了白羽：“听到了么？她不承认。”
<br/>　　白羽没想到高漫秋会如此直面挑衅，但却反应很快的回答：“白家没有神女，所以返祖者在哪，白家就在哪。”
<br/>　　“哦？”闻言高漫秋的眉头一挑高，好似听到了什么令她惊讶的答案：“这么说来，还真有那么几支旁系，要反抗白忧？”
<br/>　　被一下子反推而上戳穿事实，白羽的面色大变，连忙闭口不敢再多乱说一句话。
<br/>　　白允儿却满不在乎，往前走近了一步，道：“我跟白忧不是同一阵营，想必高家也是如此，那么不如你我将这次事件的所知情报交换如何？”
<br/>　　高漫秋眯起眼睛，一言不发。
<br/>　　顿时周围的安静便好似一股沉重气压，静匿到可怕的气氛中，隐隐散发着某种恐怖感。
<br/>　　也不知过了多久，高漫秋的神情忽然一变，面上随着一个笑容的绽放，周围气氛也好似冰雪融化般荡然无存。
<br/>　　“好。”她笑着开口。
<br/>　　白允儿唇角轻勾，率先开口：“我所知道的有两点，一是白忧私下经常召见异种王木叶，二是异种王有个亲弟弟名叫木天，是被强行转化的类尸，异种王依照白忧的吩咐将木天驱逐到了南方，时间大约为两个月以前。”
<br/>　　“南方？”高漫秋眼神一厉，正色道：“你所说的南方，莫非是尸潮来源的那个南方？”
<br/>　　“正是。”白允儿点头：“埋骨之地，预言中丧尸王诞生的地方。”
<br/>　　高漫秋的呼吸微微加重，眼神也渐渐变得凌厉起来：“正好，我所知道的也是两点。一是回炉八门的水火两族尚存，二是丧尸王确认已经诞生，时间恰好为此次大尸潮理论上刚聚集的时候，也就是一个月之前。”
<br/>　　交谈结束，两女相视无言良久，随后同时点头错身离开。
<br/>　　只是在两人擦肩而过后，两女的神色都刹那间一变，带着乍闻消息的震惊，和不谋而合想到同一处后的深思。
<br/>　　当离开段氏基地不久，确定周围已经无人，并且那个高漫秋不可能听得到后，白羽这才快步走到白允儿身旁。
<br/>　　“小姐姐，刚刚那四件事都是真的吗？”白羽的声音带着急切和震惊，以及不能理解：“但这和我们追着楚涵昏迷事件的源头调查，有什么关系？还有楚涵昏迷的事还没调查清楚呢，我们就这么离开吗？不查了吗？小姐姐你不是喜欢他吗？”
<br/>　　这次出行不是为了调查楚涵昏迷的真·相么？
<br/>　　怎么聊着聊着，聊出了丧尸王？
<br/>　　白允儿当下的情绪已经缓和，她语气平淡的说道：“已经有结果了，自然不用继续查。”
<br/>　　“啊？”白羽更加一头雾水：“怎么就有结果了？咱，咱查出了什么？”
<br/>　　白允儿的眉眼暗了暗：“楚涵的昏迷，是遭受了丧尸王的精神攻击。”
