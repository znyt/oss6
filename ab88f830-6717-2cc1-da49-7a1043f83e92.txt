　　修炼室之中，苏莫再次进入了空间宝器之内，一边吸纳源石修炼，一边参悟吞噬法则。
<br/>　　当然，他的空间玉镯，已经崩溃了。
<br/>　　这一次，他进入了另外一件空间宝器之中，这是一枚空间戒指，是一件圣器，当初在神路所得。
<br/>　　吞噬法则，无比的玄奥，顺应天地大道，元神在体内不出，参悟起来进展可以忽略不计。
<br/>　　苏莫无奈，便在空间戒指之上，布下层层防护之后，释放出了吞噬元神。
<br/>　　漆黑的吞噬旋涡，闪烁着混沌之光，霸道而诡异。
<br/>　　任何的力量，都是天道规则的分支，吞噬之力，亦是如此。
<br/>　　法则等等级，和奥义相同，分为九阶，每一阶有初期、中期、后期、圆满之分。
<br/>　　苏莫微微催动吞噬元神，感知吞噬之力，精深吞噬发则。
<br/>　　吞噬法则，虽然只是辅助性的法则，但是，若是足够的高深，杀人只在转念之间。
<br/>　　修炼无岁月，转眼之间，一年多又过去了。
<br/>　　苏莫结束了修炼，因为他的源石耗尽了。
<br/>　　一年多的时间，他的进展不小。
<br/>　　虽然修为依旧是一品低级、法则也是一阶初期，但是，两者距离突破，也不是很远了。
<br/>　　“该去赚取源石了，也顺便拉拢拉拢沁儿姑娘！”
<br/>　　苏莫叹了口气，随即，他收起元神，回到了房间之中。
<br/>　　而后，他取出身份玉佩，查看宫内有没有类似于熔岩窟的地方。
<br/>　　如今，他修为已达真神，虽然还可以去熔岩窟，但是，已经不可能得到奖励了。
<br/>　　人王宫之中，能获得源石的地方不少，但是，大多都不适合苏莫。
<br/>　　因为，有些地方并不能得到大量的源石，对于苏莫来说，杯水车薪。
<br/>　　而有些地方，没有三四品真神的实力，又无法前去赚取灵石。
<br/>　　查看了片刻，苏莫在玉佩内记载的人王宫各种消息之中，找到了一个适合他赚取源石的地方。
<br/>　　这个地方，叫做天浑虚界，非常的特殊，要暂时离开人王宫。
<br/>　　随即，苏莫离开了自己的‘惊天府’，他要先去找沁儿姑娘。
<br/>　　很轻易的，他就打听到了沁儿的‘月沁园’，来到了月沁园大门之前。
<br/>　　咚！咚！咚！
<br/>　　苏莫轻轻的敲了敲门，响起沉闷的声响。
<br/>　　“进来！”
<br/>　　少倾之后，一声轻灵悦耳的声音响起。
<br/>　　苏莫推开院门，走进了月沁园之中。
<br/>　　很快，苏莫在院中的凉亭内，见到了沁儿。
<br/>　　只不过，除了沁儿之外，还有褚彦和骆妤，以及另外一名年轻男子。
<br/>　　这名年轻男子，一身白袍，风度翩翩，修为极高，比之骆妤还要高不少。
<br/>　　白衣青年和骆妤站在一起，神态显得有些亲昵。
<br/>　　“你来干什么？”
<br/>　　见到苏莫到来，褚彦脸色一沉，率先开口问道。
<br/>　　现如今，他心中对于苏莫极为厌恶，一看到苏莫来找沁儿，心中不免暗恨。
<br/>　　“段惊天，你有什么事吗？”沁儿轻声问道，自从上次听了骆妤的话，她心中对于苏莫也多了一丝警惕。
<br/>　　当然，她也不会故意疏远苏莫，毕竟，都是骆妤的猜测而已。
<br/>　　“无事，我只是最近准备出去历练一番，特意来看望你！”苏莫笑了笑，本来他准备邀请沁儿一起。
<br/>　　不过，看到褚彦和骆妤都在，还有另外一名陌生的白衣青年，他并没有提出这个邀请。
<br/>　　“你也要去历练？”
<br/>　　沁儿闻言惊讶，道：“我们也正准备近期出去呢，只是还没想好要去哪里？”
<br/>　　“这么巧？那正好，我们可以同行！”苏莫笑着说道。
<br/>　　“不好意思，你没有资格和我们同行！”
<br/>　　就在此刻，那名陌生的白衣青年开口了，冷淡的说道。
<br/>　　他和骆妤青梅竹马，也从后者口中，了解到了苏莫的情况。
<br/>　　所以，他相信骆妤的判断，自然而然的排斥苏莫。
<br/>　　苏莫闻言，眉头一皱，不过，既然别人不愿意，他也不会死皮赖脸的跟着。
<br/>　　“告辞了！”
<br/>　　苏莫淡淡的说道，言罢，他转身离去。
<br/>　　对于白衣青年的藐视，他并不放在心上，总有些人眼高于顶，他也懒得理会。
<br/>　　“哎，段惊天……！”
<br/>　　沁儿见苏莫离去，急忙呼喊了一声，不过，苏莫却是毫不停留，大步走出了月沁园。
<br/>　　见苏莫离去，沁儿无奈的叹了口气，目光看着白衣青年，道：“月松哥，你怎么能这么说话呢？”
<br/>　　沁儿心中有些无语，白衣青年名为姬月松，是她的一位堂兄。
<br/>　　但是，这位堂兄的话，未免太过伤人了。
<br/>　　“沁儿，你以后莫要和这种不三不四之人在一起！”姬月松告诫道。
<br/>　　“是啊，沁儿，人王宫顶级天才多的是，哪一个不比段惊天强？就算是褚彦也要好过他！”
<br/>　　骆妤沉声道，那段惊天虽然闯到了熔岩窟第十八层，但不过是仗着防御强而已。
<br/>　　况且，就算段惊天的天赋很厉害，但是目前的修为，还是太低了。
<br/>　　“沁儿，我对你的心意，你难道还不明白吗？”
<br/>　　褚彦适时开口了，郑重而深情的道：“我们一同长大，青梅竹马，感情不是那段惊天可比，而我的实力，足以碾压段惊天！”
<br/>　　听着三人的话语，沁儿无奈的拍了拍脑门，心中颇为无语。
<br/>　　“我说两位大哥、大姐，你们能不能不要这么敏感，大家都是同门，不过是一起历练而已，值得这么夸张吗？”沁儿无奈的说道。
<br/>　　“我们是怕你被那段惊天迷惑！”骆妤道，她实在不理解，沁儿为何如此照顾那段惊天？
<br/>　　若说是仰慕天才，这未免太可笑了。
<br/>　　人王宫天才多的是，很多妖孽在天赋方面，远远不是段惊天能比拟的。
<br/>　　比如，她的恋人姬月松，就是妖孽天才，各方面都能吊打段惊天。
<br/>　　在骆妤看来，沁儿若是想要选择恋人、或者道侣，也应该是宫中那些顶级妖孽，而不是段惊天。
<br/>　　沁儿就算是选择褚彦，她都没有什么意见。
<br/>　　“好了好了，不说他了，你们准备去哪里历练？”沁儿摆了摆手道。
<br/>　　她对于历练，更加的感兴趣。
<br/>　　每日憋在人王宫之中，实在是太枯燥了，出去玩玩才有意思。
<br/>　　“去天浑虚界吧！”一身白衣的姬月松道。
<br/>　　沁儿三人闻言，顿时眼眸一亮，天浑虚界可是一个有意思的地方。
