        郎安看向连星儿所睡的那间屋子紧闭的门，嘴角情不自禁地微微上扬。
<br/>
<br/>        不过，他今天倒是已经确定了明司寒背后的那个女人的身份，并将那个女人的事情告知了实验室里裴冉和连星儿的外公连中凯。
<br/>
<br/>        另一边，临海大学的男生寝室里。
<br/>
<br/>        明司寒是彻夜未归。
<br/>
<br/>        而贺思思最终是被男生宿舍的宿管员给赶出去的。
<br/>
<br/>        贺思思不死心，第二天又跑来了男生宿舍找明司寒。
<br/>
<br/>        好在这个时候，明司寒从外回来了。
<br/>
<br/>        看到贺思思，明司寒有些头疼地挠了挠后脑勺，最后还是把她给叫了出去。
<br/>
<br/>        两人肩并肩，走在了校道上。
<br/>
<br/>        “司寒哥哥，你是不是因为我哥哥的事情，你才连带着一起讨厌我的？”贺思思突然停下脚步，拦在明司寒的跟前，两人面对着面，她楚楚可怜地问道。
<br/>
<br/>        明司寒看着贺思思，十分坦然地回应道：“是！”
<br/>
<br/>        “可是，那是我哥哥所做的事情啊！又不是我！”贺思思满脸委屈地说。
<br/>
<br/>        她只知道，明司寒十七岁那年最初是和她的哥哥贺泽川一起创业的。结果，她的哥哥出卖了他，还拿走了明司寒的劳动成果，最终导致明司寒初创立的小公司破产。
<br/>
<br/>        同年，明司寒也因为这事高考失利。
<br/>
<br/>        “贺思思，你走吧！不要再来纠缠我了。虽然我们俩从小就认识，但仅仅只是认识而已。我从未喜欢过你，也从未对你有过任何其他想法。以前没有，现在将来就更不可能会有。”明司寒直截了当地拒绝道。
<br/>
<br/>        贺思思顿时傻眼了，鼻子一酸，带着哭腔质问道：“是因为那个连星儿吗？明明我比她更先认识你！你为什么不选我？我哪里不如她啊！”
<br/>
<br/>        明司寒一看到贺思思哭，顿时就心烦气躁，开始变得极其不耐烦起来，甚至口不择言地回答：“她比你长得漂亮，身材也比你好，家世背景比你家强，更何况，我和她还有婚约在身。现在你懂了吧？”
<br/>
<br/>        “司寒哥哥，你不是这种人的，我知道的。”贺思思潸然泪下地反驳。
<br/>
<br/>        明司寒皱着眉头，不耐烦地反问：“那你觉得我是哪种人？”
<br/>
<br/>        “我知道你是被迫的！其实，你压根就不喜欢连星儿对不对？而且，你也看不上她家的家世背景。司寒哥哥，你跟我交往吧！我有办法帮你解除这种豪门联姻的。”贺思思信誓旦旦地说道。
<br/>
<br/>        明司寒有些头疼不已：“我再跟你说一遍，我不喜欢你，你不要在我身上浪费时间了。”
<br/>
<br/>        其实，这句话，在连星儿没出现之前，他就已经跟她讲过了，只是她一直不死心。
<br/>
<br/>        明司寒说完后转身便走。
<br/>
<br/>        贺思思愣在原地，无声地掉着眼泪。
<br/>
<br/>        在明司寒离开后，不知何时，仲正浩走到了她的跟前，给她递上了纸巾。
<br/>
<br/>        “我不大会安慰人，但是我可以把肩膀借你用。”仲正浩摸了摸头顶，突然间变得十分憨厚地说道。
<br/>
<br/>        贺思思抬眸看了仲正浩一眼后，直接扑入了他的怀中，放声大哭了起来。
<br/>
<br/>        仲正浩将贺思思送到了她哥哥贺泽川家楼下后，返校一进寝室门，踱步上前，扬起拳头便向明司寒的后背挥去。
<br/>
<br/>        明司寒察觉到背后的动静，抱起面前的笔记本电脑，敏捷地从椅子上闪身。
<br/>
<br/>        而仲正浩的那一拳，直接打在了书桌上的搁物板上，“嘭”地一声，指骨的碰撞，顿时让他痛得眼泪都冒了出来。
<br/>
<br/>        “我靠！寒哥！你怎么‘挂’了啊！”张子晋惊呼道。
<br/>
<br/>        邢乐游这边看着自己操作的英雄人物也跟着团灭后，也抱怨道：“是啊！阿寒，关键时刻，你咋不动了？”
<br/>
<br/>        在仲正浩没回来之前，他们三个人正在一起多人排位游戏。
<br/>
<br/>        明司寒见仲正浩气势汹汹的样子，合上了笔记本电脑放在了一旁的矮桌上，然后伸腿将椅子勾到跟前，一屁股坐下后，双手抱臂，仰望着仲正浩，问道：“说说看，你是什么原因，对我有意见？”
<br/>
<br/>        “贺思思，那么好的一个女孩子。你为什么要对她那般无情？”仲正浩一边揉着拳头，一边振振有词地质问道。
<br/>
<br/>        张子晋和邢乐游见明司寒挂机了，这才停下手中的游戏，不约而同地望向此刻正站在明司寒身前义愤填膺的仲正浩。
<br/>
<br/>        “我不喜欢她，对她无情，难道不正常吗？”明司寒不以为然地反问。
<br/>
<br/>        张子晋和邢乐游不禁面面相觑，但谁也没在此刻插嘴明司寒和仲正浩之间的事情。
