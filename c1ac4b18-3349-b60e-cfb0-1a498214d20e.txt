　　“现在的后卿呢？”
<br/>　　“现在他炼化了将臣之血，实力大增，我不知道他强到什么地步，但肯定比你厉害。”
<br/>　　“我得先杀了他！”叶少阳咬牙道，“我之前发过誓的，就为他对冷玉做的那一切，我一定得杀了他！”

<br/>　　叶少阳摇头，“我怕我会死在鬼王手下，我有信念，但没信心，我得先杀了他，再去斗鬼王。”
<br/>　　道风看他坚持，知道劝说不动，想了一下道：“你可以跟军师商量下，趁着战役没结束，是有机会，平时他久居天弃山，又有部下包围，你很难动手。”
<br/>　　之后，两人密谋了有关终极一战的一些细节，叶少阳问起五把神剑到底怎么办，道风让他别管。
<br/>　　天亮之后，道风走人，叶少阳回山之后，为了避免被张小蕊缠着，只找了苏钦章和叶小萌，虽然这一对也很想跟叶少阳一起战斗，但被叶少阳以保护茅山为理由留下来。
<br/>　　带着瓜瓜回到石城。
<br/>　　回到家里，只看到芮冷玉一个人在。
<br/>　　“他们都走了，我专门等你的，我们也上路吧。”面对叶少阳的疑问，芮冷玉回答道。
<br/>　　“上路，去哪？”
<br/>　　空界，界河边上。
<br/>　　一队士兵躺在河边，一个个疲惫到了极点，不少人身上还带着伤，都在吐纳调息。
<br/>　　他们是刚刚经历了一场血战，幸存下来的联军士兵，然后被放在这里镇守界河。
<br/>　　每个人都恨透了尸王，恨他发动战争，将他们这些本来可以好生修炼的人卷到血腥的杀戮中来，殷切希望他们了不起的林大帅能带领他们战胜尸族，早点结束这一仗。
<br/>　　这一次战役，比上一次规模还要大，后卿五路大军从不同方向进攻，将整个空界都拖进了战火中，到处都不得安生。
<br/>　　日前，五路大军一起开进荒原，进了曲长山中。
<br/>　　此刻，曲长山的地图，就挂在林三生的大帐里，林三生把所有手下都赶出去守门，严令不许任何人进来，一个人站在地图前面，不知道站了多久，他手里拿着一根鹅毛笔，在地图山小心地标注着。
<br/>　　“如何了？”
<br/>　　一个声音突然在身后响起，林三生吓了一跳，本能以为是卫兵闯进来了，刚要发作，回头一看是道风，立刻把气咽了回去。道风想要进来，他的那些卫兵根本发现不了。
<br/>　　林三生继续研究地图，道风也走过来跟他一起看。
<br/>　　曲长山，不是一座山，而是一道延绵数百里的山脉，地形十分复杂，有数道峡谷，又长又曲折，因此被称作曲长山。
<br/>　　为了方便定位，林三生这几条峡谷都编了号，一二三四五这样排下来。
<br/>　　“这里，还有这里。”林三生用鹅毛笔在地图上点着，“这些峡谷，都是他必经之路，他有五路大军，但真正的主力只有一支。”
<br/>　　“为什么？”
<br/>　　“他要决战，主力就一定会集中起来，所谓的另外四路大军，不过是迷惑我们的幌子。我早就看穿了。”
<br/>　　“你都看穿了，那还担心什么？”
<br/>　　林三生转头看了他一眼，“我觉得你是不懂兵法吧。”
<br/>　　“我要是懂兵法，要你干什么！”
<br/>　　之前道风在山河社稷图里养息元神时，林三生就是他跟外界的唯一联系，所有修炼需要用到的东西，都是林三生帮忙弄的，两人经常打交道，也会一起论道，关系也熟悉起来，说话不用客气。
<br/>　　“他有五路大军，其中主力只有一支，我们兵力没他们多，不可能五条路上都设伏，因此必须找到他的主力，就算猜到他的计划，也还是五分之一的机会。”
<br/>　　道风抱着双膝，看了一会地图，说道：“你不是应该去搜集情报吗，这种事，研究地图有什么用。”4
<br/>　　“五条峡谷，各不相同，我在换位思考，如果我是后卿，我会选择大军走哪一条路……”
<br/>　　“有眉目了吗？”
<br/>　　林三生用鹅毛笔在两条峡谷上分别点了一下，“第三和第五条，两边的山不算太高，峡谷也足够曲折，就算遭到伏击，也能寻找掩体，支撑更久，实在受挫也容易退走，不至于全军覆没，他的主力，必然要走这两条中的一条。”
<br/>　　道风认真看了一会，道：“你能想到的，他也能想到，有没可能他故意不走这两条路？”
<br/>　　“当然，难度就在这里。”
<br/>　　林三生揉了揉脑袋，走到一边去，从桌上拿起一只香炉，将里面的火吹着，用力吸了几口。
<br/>　　道风没有再就这个问题谈下去，他是元帅，这种勾心斗角的事，交给他就好，自己说多就是添乱。
<br/>　　“有没有可能，他一条都不选？”
<br/>　　“那他干什么来了？仗打到这一步，他就是要穿越山脉，不是云山就是曲长山，只有进入基地，才能把十几家门派都平推了，否则只在界河一带转悠，没任何意义。”

<br/>　　“就那样。”
