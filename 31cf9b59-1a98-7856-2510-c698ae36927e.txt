        “呃——”连星儿汗颜，厚着脸皮回答道，“我这不正给您打电话嘛！”
<br/>
<br/>        “叫明司寒接电话。”傅夜沉接着说道。
<br/>
<br/>        此时，明司寒刚把连星儿的行李箱从后备箱里拿下来，就只见连星儿把她自己的手机递了过来。
<br/>
<br/>        “我爹地找你。”连星儿喃喃地说。
<br/>
<br/>        明司寒怔了一下，有些匪夷所思地接过连星儿的手机。
<br/>
<br/>        “冷叔叔，您好。”明司寒彬彬有礼地问候。
<br/>
<br/>        傅夜沉直接开门见山地说道：“你爷爷跟我说，是你提议要在明年星儿十九岁生日那天举行订婚礼。既然如此，我有一前提条件。”
<br/>
<br/>        “冷叔叔请讲。”
<br/>
<br/>        “你当我傅夜沉的上门女婿。”傅夜沉不冷不热地答。
<br/>
<br/>        这话的意思是要他明司寒入赘傅家。
<br/>
<br/>        明司寒下意识地看了连星儿一眼，毫不犹豫地应了声：“好。”
<br/>
<br/>        “你可以回去跟你爷爷商量好之后再答复我！”傅夜沉接着说道。
<br/>
<br/>        傅家和明家的实力均衡，谁也不怕谁。
<br/>
<br/>        联姻的事，既然是明家提出来，让他们傅家被动，那么，傅夜沉决定让这被动成为主动。
<br/>
<br/>        他宠大的宝贝女儿可以牺牲婚姻的自由选择权，但不能牺牲婚姻的幸福权。
<br/>
<br/>        哪怕是豪门联姻，他也不会让自己的女儿过得不幸福。
<br/>
<br/>        傅夜沉说完那句话后，便把电话给挂断了。
<br/>
<br/>        明司寒将手机还给了连星儿。
<br/>
<br/>        连星儿见电话已经挂断了，立马抬头往那辆豪车那边看过去。
<br/>
<br/>        只见那辆豪车已经扬尘而去。
<br/>
<br/>        “你在看什么？”明司寒见连星儿正在发呆，便抬起手来在她眼前晃了晃。
<br/>
<br/>        连星儿回过神来后，好奇地问道：“我爹地刚跟你说什么了？”
<br/>
<br/>        “就是警告我不要欺负你。”明司寒笑道。
<br/>
<br/>        连星儿狐疑地睐了他一眼，喃喃起来：“我才不信！你们肯定又是在谈联姻的事情。”
<br/>
<br/>        “你对这事这么敏感？”明司寒一脸诧异。
<br/>
<br/>        连星儿从明司寒手中夺过自己的行李箱拉杆，一边拖着行李箱往校门口走，一边若有所思地回答：“我现在最大的愿望，就是跟你解除婚约。”
<br/>
<br/>        “真巧！”明司寒随即追上连星儿的步伐，不甘示弱地傲娇了一句，“这也是我的愿望！”
<br/>
<br/>        “那我祝你愿望成真！”连星儿立马接上明司寒的下文。
<br/>
<br/>        明司寒只笑不语，意味深长地看了连星儿一眼后，加快了步伐，走在了她的前头。
<br/>
<br/>        连星儿拖着行李箱看着明司寒渐行渐远的背影，脑海里又回想起了昨天在他小时候住过的房间里，看到的那张相片。
<br/>
<br/>        其实，那张相片里的小时候的他，给了她一种似曾相识的感觉。
<br/>
<br/>        那是一种第一眼，就让她想起小时候的悸动。
<br/>
<br/>        而且，她总是在不经意间地将他跟小时候救了她的那个小哥哥联想到一块。
<br/>
<br/>        连星儿想到这儿后，使劲地摇了摇头。
<br/>
<br/>        她的“小纽扣”是洛羽，不是明司寒，所以，不要去多想了。
<br/>
<br/>        回到宿舍后，连星儿将自己从北城带回来的特产分给了室友顾千凝、赵安彤和路娅玟。
<br/>
<br/>        放完行李后，连星儿又拖着一个小的行李箱离开了学校，去了傅家大庄园。
<br/>
<br/>        有好东西，她自然不会忘记疼爱她的祖爷爷和奶奶。
<br/>
<br/>        因为晚上还有选修课要上，所以，连星儿送完特产后，就直接走了，没留在傅家大庄园里陪祖爷爷和奶奶一起吃晚饭。
<br/>
<br/>        返校后，连星儿和三个室友一起去阶梯教室上选修课。
<br/>
<br/>        因为去晚了，她们四个女孩子只能坐在最后一排，所以前方同学们的小动作，她们可以尽收眼底。
<br/>
<br/>        正因如此，当坐在前面的几个女生拿着手机在桌底下相互看来看去，交头接耳后，还时不时地往后回头看，惹得赵安彤也好奇地探身凑上前去看了看。
<br/>
<br/>        顾千凝和路娅玟也将课本立了起来，无聊地在躲在课本后玩手机。
<br/>
<br/>        连星儿倒是一边认真听课，一边在课本上做笔记。
<br/>
<br/>        对于周遭同学的动静，她似乎是与世隔绝了般，完全不受他们的影响。
<br/>
<br/>        赵安彤看到前方女同学手中手机里的内容后，连忙让顾千凝和路娅玟也进入临大论坛去看帖子。
<br/>
<br/>        帖子的标题是“美女学霸的干货”。
<br/>
<br/>        这帖子第一眼看上去，还以为是学习资料的分享帖。
<br/>
<br/>        当她们点击去后，里面的内容简直辣眼睛。
<br/>
<br/>        女学霸的名字简称是“LXE”，高三那年转入BJ一中，在校学习成绩优异，每次考试都是全年级第一，实则是贿赂了老师，拿了考试试题，提前背了答案。
