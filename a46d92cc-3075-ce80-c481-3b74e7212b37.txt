　　几分钟后，慕清霄几乎将书架底下几层都看完，总共收录二十本封印秘法，同时也获得了二十万掠夺积分。
<br/>　　二十万掠夺积分，对于现在的他来说，依旧是一笔不小的数目。
<br/>　　此时，慕清霄星眸中闪烁着喜悦之色，简直比抢都来的快，如果将任务完成，便能给语嫣她们兑换最顶级的修仙心法了。
<br/>　　平复心中的情绪后，他的目光落在书架最顶层，伸手抽出其中唯一的一本秘籍。
<br/>　　“叮，《空明幻虚剑》收录中，价值100000掠夺积分。”
<br/>　　《空明幻虚剑》紫胤真人剑术集大成者，此技凝集全身真气，幻化出无数剑影飞向对方，使对手遭受万剑穿心之苦。
<br/>　　除非有很高修为，否则即刻毙命！
<br/>　　此剑法需集结全身灵气贯以极强剑气方能发出，只有修为极高，或领悟力极强之人才可修习。
<br/>　　十万掠夺积分，令慕清霄咂舌，但更令他兴奋的是，手中的《空明幻虚剑》。
<br/>　　修炼空明剑与太虚剑五年的岁月，为的可不就是手中的剑诀么，现在如愿以偿，怎能不高兴。
<br/>　　要知道紫胤真人可是剑仙啊，而此剑诀就是他的证道剑诀，威力可见一斑。
<br/>　　喜悦过后，慕清霄视线在第四层扫视一圈，确认并没有遗落后，便转身离开。
<br/>　　离开妙法堂后，他先是去了一趟凝丹堂，确认本月需要炼制的丹药后，拿着一枚装满灵药的空间玉镯就离开。
<br/>　　本月需要炼制的丹药是‘地灵丹’，适合元婴期以下修士服用，药效是直接提升一重修为。
<br/>　　‘地灵丹’在年轻的修士们眼中，就是宝物一样的存在。
<br/>　　而天墉城炼制此丹药，自然就是培养门派血脉，同时也是为了笼络，天赋异凛的年轻弟子们的心。
<br/>　　当然，物以稀为贵，所以本月的任务，就是十炉‘地灵丹’，每一炉十颗，总计一百颗左右。
<br/>　　十炉‘地灵丹’对于慕清霄来说，简直就是小菜一碟。
<br/>　　风风火火的返回住处后，慕清霄就开始炼制‘地灵丹’，两日后的清晨，炼丹室的大门就开启。
<br/>　　伴随着一阵令人精神一振的丹香，慕清霄走了出来。
<br/>　　随后，吩咐小兰将‘地灵丹’送去凝丹堂后，慕清霄就前往后山，开始研究《空明幻虚剑》。
<br/>　　修炼《空明幻虚剑》前，慕清霄首先修炼起一本名为《化剑术》的功法。
<br/>　　《化剑术》类似于《化形术》，施术者可自身化作飞剑，瞬息千里，也是一门非常实用的术。
<br/>　　当然，慕清霄本身的速度并不比化剑术慢，修炼此术，只是因为己身化剑比较潇洒而已。
<br/>　　接下来整整一周的时间，慕清霄都在修炼《化剑术》，成果也非常明显。
<br/>　　心念一动，剑芒一闪即逝，慕清霄就消失不见，取而代之的是一柄，散发着恐怖气息，且霸气无双的长剑。
<br/>　　“咻咻——”
<br/>　　长剑化作一道剑芒，一闪即逝，宛如追星赶月般向山下掠去，眨眼间又回到山巅。
<br/>　　脱离化剑模式后，慕清霄立于山巅，微微一笑，道：“此术也并非一无是处，真是潇洒极了。”
<br/>　　随着时间的推移，一名青年立于山巅，双目紧闭，持剑而立，纹丝不动，长袍随风摆动......
<br/>　　不知过了多久，恐怖的剑意一放即收，慕清霄睁开双眼，一抹苦涩一闪即逝。
<br/>　　不愧是剑仙的证道剑诀，想要修炼《空明幻虚剑》，自身实力必须达到练虚期方能开始修炼。
<br/>　　这是修炼的第一个条件，第二个条件就是必须将空明剑修炼至圆满，因为空明剑是空明幻虚剑初级形态。
<br/>　　他早就将空明剑修炼至圆满，但修炼空明幻虚剑，依旧举步难行，足足一年，勉强能够理解其中意境。
<br/>　　而他的修为也达到了练虚五重，也勉强能够施展而已，但极其耗费灵气，所以不到万不得已，他绝不会用此剑法。
<br/>　　除非，等他有了施展此剑术的能力，或者遇到真正对手的时候。
<br/>　　“哐当...!”
<br/>　　就在这时，一阵阵警钟，将慕清霄思绪拉回现实。
<br/>　　瞥了一眼天墉城东边剑塔方向，慕清霄剑眉一凝，唯有天墉城发生大事的时候，才会敲响警钟，难道发生了什么事情？
<br/>　　想到此处，慕清霄就化作一道飞剑，迅速东边方向的剑塔掠去。
<br/>　　“见过凝丹长老...”
<br/>　　“见过凝丹长老...”
<br/>　　“掌门也过来了，快将道路让开。”
<br/>　　见慕清霄与掌门来此，闻声来此的天墉城弟子，都纷纷让开一条道路。
<br/>　　见到涵素真人后，慕清霄微微行礼，道：“掌门，在下听到警钟后赶来，到底发生了何事？”
<br/>　　看见慕清霄，涵素真人严肃的脸色也缓了缓，视他为天墉城的未来，并没有打算隐瞒。
<br/>　　瞥了一眼周围的弟子，涵素真人道：“我们先进去，这里不是说话的地方。”
<br/>　　闻言，慕清霄跟在涵素真人身后，向着剑塔中走去。
<br/>　　剑塔是执剑堂重地，除了执剑长老真传弟子，或者掌门都不得擅自入内，哪怕是各堂长老都必须经过允许后，才能进入。
<br/>　　当慕清霄与涵素真人进入剑塔后，便看到紫胤真人与一名美艳的红裙女子中间，悬着一柄赤红色长剑。
<br/>　　此时，两人正在施展封印之法，脸色也颇为凝重。
<br/>　　瞳孔微微一缩，慕清霄一眼就认出了剑的来历，正是他的目标，焚寂。
<br/>　　而在紫胤真人与红裙女子的身旁，则有一名脸色苍白，昏迷不醒的孩童。
<br/>　　焚寂为龙渊七凶剑之一，内含足以焚毁一切的邪火之力。
<br/>　　是龙渊工匠以“血涂之阵”取太子长琴命魂四魄所造，先被女娲封印千年，后成为主角的随身佩剑。
<br/>　　看到焚寂剑后，慕清霄就知道，面前昏迷不醒的孩童，就是剧情的主角韩云溪，或者说百里屠苏。
<br/>　　见两人正在施展封印之法，涵素真人脚步一顿，并没有上去打扰。
<br/>　　此刻正是关键时刻，若是现在惊扰两人，后果不堪设想。
<br/>　　慕清霄瞥了一眼焚寂，又看了一眼紫胤真人，见他脸色略有苍白，应该是在幽都拯救韩云溪，镇压焚寂时受的伤。
