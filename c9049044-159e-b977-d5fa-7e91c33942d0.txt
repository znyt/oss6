　　“楚兄辛苦了。”
<br/>　　司空轩看着楚风眠飞回来，直接便是迎了上去，笑着说道。
<br/>　　“这些虚空白虎，可不好对付啊，就让我们一起上，只怕都难以全部斩杀。”
<br/>　　“这有什么，任何妖兽，在我的面前，都是不堪一击。”
<br/>　　楚风眠故意大笑起来，故意做出一副毫不在意的样子。
<br/>　　但是在楚风眠身上的灵力，却是已经极为不稳定，似乎因为消耗过大，在苦苦硬撑。
<br/>　　这幅样子，自然是楚风眠在假装的，就是为了麻痹这司空轩等人。
<br/>　　一旦是楚风眠消耗了大部分的灵力，自然看起来也就是没有什么威胁。
<br/>　　等到了颜古龙帝的洞府中，楚风眠也更好的可以崭露头角。
<br/>　　“楚兄果然厉害，只怕是回到北邙学院中，晋升种子弟子，都是指日可待了。”
<br/>　　司空轩故意奉承一句，随后直接说道。
<br/>　　“看楚兄都没有废多少力气，我们还是直接赶路吧。”
<br/>　　这司空轩如今如此故意开口，正是为了不给楚风眠任何休息恢复灵力的时间。
<br/>　　“你！”
<br/>　　楚风眠脸色装出一副极为难堪的样子来，还是外强中干一般的开口道。
<br/>　　“无妨，我们现在就走。”
<br/>　　“走！”
<br/>　　其他几人，各自暗地里传音几声，似乎是在交谈这什么，目光闪烁，看向楚风眠，便是看向傻子一般。
<br/>　　一群人，继续飞行。
<br/>　　不一会，这周围的虚空中，却是多了不少的东西，一块块巨大无比的岩石出现。
<br/>　　在这些岩石中，有着许许多多的洞窟，但是都已经是极为破旧了，里面已经没有任何的灵力了。
<br/>　　“这里就是乱葬海了，这些岩石，都是曾经被发现的远古洞府，不过都已经被人探索一空了。”
<br/>　　司空轩开口道。
<br/>　　“继续走，颜古龙帝的洞府，就在乱葬海的深处边缘。”
<br/>　　一群人继续飞行这。
<br/>　　白萱观察了一眼楚风眠，看着楚风眠故意装出来的外强中干的样子，悄然之间递过去了一枚丹药，递给了楚风眠。
<br/>　　“这些丹药，足以恢复你一些灵力，颜古龙帝的洞府危险无比，你还是将这些丹药全部吞噬了吧。”
<br/>　　“哦？”
<br/>　　看到白萱的举动，楚风眠还是有些惊讶的。
<br/>　　他跟白萱，也是合作关系，不过白萱却是没有相识司空轩等人一样，来坑楚风眠，倒是给楚风眠了丹药。
<br/>　　看来这白萱，倒是真的想要跟楚风眠一起合作，探索颜古龙帝的洞府的。
<br/>　　“无妨，这些丹药还是由师姐自己留着吧。”
<br/>　　楚风眠也是悄然传音道，同时暗地里凝聚一股灵力，直接悄悄的涌入到了白萱的身躯之中。
<br/>　　白萱只感觉到，楚风眠身上的灵力，雄厚无比，如同群山一般的漫长，简直是浑厚到了极点。
<br/>　　白萱震惊的开口道。
<br/>　　“你，居然还有着如此力量，斩杀那些虚空兽，对付居然没有任何影响？”
<br/>　　斩杀如此之多的虚空兽，就算是让白萱自己来，如今也只怕是不剩多少力量了。
<br/>　　所以她却没有想过楚风眠的样子，只是伪装，可现在楚风眠身上浑厚的力量，却是令她都极为震惊了。
<br/>　　楚风眠悄然之间对着白萱传音道，将白萱递过来的丹药，直接拒绝。
<br/>　　“我有着一些手段罢了，先听从他们的话，等到颜古龙帝的洞府再说。”
<br/>　　关于颜古龙帝洞府的种种，司空轩知道的最多，所以便是以他为首。
<br/>　　楚风眠故意示弱，也是在利用司空轩罢了，等到真正进入到了颜古龙帝的洞府中，司空轩也就没有任何的作用了。
<br/>　　倒时候楚风眠在对付他们也不迟。
<br/>　　“这些人，都要死！”
<br/>　　楚风眠心中杀意浓郁到极点，敢来算计他楚风眠，都要死。
<br/>　　看到楚风眠拒绝了白萱的丹药，那司空轩眼神中的不屑更甚。
<br/>　　“果然是一个死要面子的废物，自己都撑不下去了，还要强撑。”
<br/>　　“反正无所谓，就让此人去引那些妖兽就行了。”
<br/>　　楚风眠的举动，更是符合了外强中干的样子，司空轩故意飞行的极快，令楚风眠跟随起来，都是气喘吁吁，困难无比。
<br/>　　终于在三个时辰之后，在司空轩的带领之下，突然来到了乱葬海的边缘之中。
<br/>　　在这面前，空无一物，只有无尽的虚空。
<br/>　　“这颜古龙帝的洞府，就在这里？”
<br/>　　看着司空轩停了下来，楚风眠心中暗自想到，他打开灵识，扫视四周，却是没有发现任何一点关于燕古龙帝洞府的影子。
<br/>　　楚风眠思考之际，那司空轩，却是一步走了出去。
<br/>　　默默的念着些什么，楚风眠听的出来，这是一种极为古老的语言，但是却无法听懂。
<br/>　　随后，在面前的虚空中，突然一道大门出现，将周围的空间，撕裂开来。
<br/>　　“这颜古龙帝的洞府，真是隐秘到了极点，只怕是没有得到奇遇，根本不可能寻找的到。”
<br/>　　楚风眠看着这大门的出现，心中暗自想到。
<br/>　　他虽然可以一早出手杀了司空轩，但是果然，留他是有用的。
<br/>　　没有司空轩带领，就算是楚风眠，也找不到这颜古龙帝的洞府。
<br/>　　“这就是颜古龙帝的洞府？”
<br/>　　“传闻居然是真的，颜古龙帝的洞府，隐秘之际，没有奇遇，根本无法打开。”
<br/>　　云公子，麟木等人，都是看到这大门的出现，极为吃惊。
<br/>　　“各位走吧，先进去再说。”
<br/>　　司空轩轻笑一声，脚下一动，第一个步入到了这大门之中。
<br/>　　看到司空轩率先进入，其他的几人，也都是纷纷踏入其中。
<br/>　　楚风眠站在大门前，利用灵识，探察其中，感觉没有什么危险之后，楚风眠这才也是最后进入了其中。
<br/>　　刚刚进入到这颜古龙帝的洞府之中，顿时面前，便是一处截然不同的世界。
<br/>　　这其中的种种法则，都跟虚空海中的，截然不同。
<br/>　　这颜古龙帝的洞府，赫然也是一方如同北邙学院所在一样的小世界。
<br/>　　像是颜古龙帝这等人物，的确是九域巅峰，足以在虚空中，开辟这样一方小世界来。
<br/>　　这一方小世界，虽然不如北邙学院，但是其中也是庞大无比，在其中有着茂密的丛林。
