　　　　金光蛇影飞出庙堂，在白镇的上空盘旋了起来，路人看到，不无激动的就地而跪，磕头膜拜。
<br/>　　这是神迹啊，几百年才能出现一次，白镇人民的内心十分激动，像波涛汹涌的大海一样，尤其是上了年纪的老人，立刻热泪盈眶起来。
<br/>　　他们世代信仰的蛇王出世了，真真切切的盘旋在白镇的上空，保一方百姓平平安安。
<br/>　　年轻人也震惊的难以复加，居然真的有蛇王的存在，什么蛇能飞起来那么牛掰呀，也就传说中的蛇王了。
<br/>　　立刻，一个个纷纷拿出手机拍摄下如此神迹的一幕，可很快他们却发现，手机拍下来的画面既然是一片空白。
<br/>　　顿时间，他们醍醐灌顶，神灵不可亵渎，旁边的老人一个巴掌呼啸而来：“混帐东西，还不跪下”
<br/>　　黑压压的人头跪地膜拜，一个个高呼：“蛇王万岁，蛇王万岁，蛇王万岁...“
<br/>　　来自...
<br/>　　来自...
<br/>　　来自...
<br/>　　涂小安的系统之中，无形迎来了一大波恐怖的信仰之力产生。
<br/>　　“叮...恭喜宿主增加320000信仰点数”
<br/>　　金光蛇影在白镇上空一显化，顿时间收获了三十二万的信仰之力。
<br/>　　此时，盘卷在上空的金光蛇影如神龙出海般的掠进了佘山深处，消失的无影无踪。
<br/>　　跪地膜拜的人们都知道，蛇王是去铲除邪魅，击杀吃人的恶魔，还白镇人民一个太平。
<br/>　　神对于信徒来说，你信者存在，你不信就是虚无。
<br/>　　但白镇人民信仰的神灵，光天化日之下显化，无人不臣服。
<br/>　　况且数百年流传，但凡在这片区域要是出现了什么灾难，蛇王便会显化，救黎民于水火。
<br/>　　此传闻，现在无人不信了。
<br/>　　佘山之上....
<br/>　　一处隐秘的林间，一条金银蛇像是凭空出现，悬浮在了空中，只见涂小安蛇眸一凝：“就是这里了”
<br/>　　他顺着佘山一路飞行，统治能力大开，隐隐约约间，察觉到了吸血蛇的强大气息。
<br/>　　根据这股气息，涂小安可以断定，此蛇已经进化到二十级以上了。
<br/>　　气息久了不散，已然达到了蛇精的层次。
<br/>　　飞行的这一路上，涂小安还留意到附近的毒蛇仿佛真空了一样，没有出现一条。
<br/>　　莫非都被吸血蛇给吃了？
<br/>　　这畜生想要佘山无蛇不成。
<br/>　　若佘山无蛇，那么不知道有多少户人家将要失业，白镇的经济将会后退几十年。
<br/>　　引发的严重后果不言而喻。
<br/>　　这时，涂小安看向前方的一个入口地带，不禁诧异：“这里是蛇沟，没想到吸血蛇居然进了蛇沟之中”
<br/>　　不得不说这畜生还真会找地方，蛇沟被白镇人民号称毒蛇的大本营，无人敢进，里面的毒蛇数量恐怕要堪比一个白镇的人口了。
<br/>　　当初蛇博士用罕见毒蛇来提升吸血蛇的血脉，现在吸血蛇来到佘山，简直鲤鱼如海，蛇沟里面，将会有大量的罕见毒蛇供它提升自身的血脉。
<br/>　　“吸血蛇，无论你成长到何种地步，最终还是要被我吞的”
<br/>　　涂小安蛇眸精光四射，心中一语，也不准备浪费直接，直接朝着蛇沟飞去。
<br/>　　唳唳……
<br/>　　可就在这时，忽然上空一道刺耳的鹰唳声，冲破层层气浪，呼啸传来。
<br/>　　天际之上，一道黑色影子带着道道破空声，快速的朝着这片地带极速的扶摇直下。
<br/>　　老鹰？
<br/>　　涂小安听到鹰唳声微微一滞，鹰的声音他可太不陌生了，当初还是小蛇的时候，他就杀了一只鹰。
<br/>　　莫非自己跟鹰那么有缘分，又碰到了一只？
<br/>　　涂小安一个抬首，朝着上空放眼望去，旋即他蹙眉起来，上空黑色影子极速的朝他的方向掠下，仅仅一眼，他就看个一目了然。
<br/>　　确实是鹰，而且还是一头成年的老鹰，扶摇直下，威风凛凛，带着极度骇人的威慑力。
<br/>　　看来是自己悬浮在空中，金银色灿灿生辉的条纹在阳光照耀下更为的瞩目，将上空的老鹰给吸引了过来。
<br/>　　那黑色影子不断的在他的蛇眸放大，一对黑色的翅膀张开，慢慢的将上空的天都笼罩的黑了下来。
<br/>　　涂小安还是人的时候，是比较喜欢老鹰这种体态雄伟，性情凶猛的天空霸主，但他变成了蛇，就对鹰这种生物的好感度极速下降。
<br/>　　原因很简单，鹰是蛇类的致命天敌，几乎天地间所有的鹰都会捕蛇，简直就是天生的捕蛇高手。
<br/>　　而且这种家伙也非常的聪明跟灵性，涂小安一出现在蛇沟，就被它发现，应该是这头老鹰早就埋伏在附近。
<br/>　　这里是蛇沟入口，经常会有毒蛇出入。
<br/>　　鹰在这里埋伏，一蹲一个准，每天都能尝到不同品种毒蛇的美味。
<br/>　　不过常在河边走，哪有不湿鞋。
<br/>　　这头老鹰现在将目标锁定在了涂小安这条金银蛇的身上，那么它的好运气也就到头了。
<br/>　　“给我洞察”
<br/>　　“叮，20级老鹰，普通生物”
<br/>　　很简单的数据，别看这老鹰是二十级，但却不是觉醒生物，也并不是任何动物十级后就成了觉醒生物，不同生物，不同的血统，觉醒的要求跟力度也有所不同。
<br/>　　鹰天生强大，想觉醒可没有那么容易。
<br/>　　已涂小安现在的等级，就算吞噬了这种普通生物，纵然是二十级，对他的帮助也不是很大，不过对方既然都杀下来了。
<br/>　　他也不能当做没看到。
<br/>　　“惊雷”
<br/>　　他含此轻吐，望着扶摇直下的老鹰，蛇眸浑然一眯。
<br/>　　劈...
<br/>　　一道粗大的雷电一闪而过，干脆利落的劈在了老鹰丰厚的羽毛上。
<br/>　　霎时间，这头威风凛凛的老鹰就好像出事故的直升机，冒着烟的狠狠坠落在地，浑身抽搐了没几下，就不见动弹了。
<br/>　　“双头蛇，便宜你了”
<br/>　　一首双头的双头蛇凭空出现，看到老鹰尸体凶光乍现，两颗摇曳的蛇头大快朵颐的吃起了老鹰的尸体，其画面怎是一个残忍了得。
<br/>　　双头蛇还在幼期，发育成长阶段，本身又血统奇高，潜力无限，这头老鹰让它吃了，会有点帮助。
<br/>　　十分钟之后。
<br/>　　一头成年老鹰就这样被双头蛇给吃了进去。
<br/>　　“叮...变异双头蛇的等级提升到13级”
<br/>　　好家伙，等级直接提升三级。
<br/>　　给涂小安吞噬了，顶多增加数万的进化能量而已。
<br/>　　他现在想升级，整整需要五十万进化能量点，想快速提升等级，恐怕只有做系统触发的任务了。
<br/>　　一个小插曲之后，涂小安将双头蛇收回系统，朝着蛇沟快如闪电的飞去。
<br/>　　一入蛇沟，吸血蛇的气息越发的浓烈起来，就在前方。
<br/>　　很快，涂小安悬浮在了一个山洞口，这山洞阴森森，时不时有阴风从里面呼啸吹出。
<br/>　　“吸血蛇，你应该就在里面吧”
<br/>　　涂小安察觉到这山洞里面扑面而来吸血蛇的气息，愈演愈烈。
<br/>　　错不了，就是这里了。
