　　神州东部有一片群山，群山之中，又有一处高耸的山群，这片山区灵气逼人，树木郁郁葱葱，老藤盘绕，鸟儿雀跃，猿猴飞奔，白鹤祥云……乃是一处不可多得的人间圣地。
<br/>　　这里常年云雾缭绕，让人看不清虚实，在群山中央，有一座峰高耸入云，直插九霄，给人一种剑刃即将出鞘的感觉。
<br/>　　云卷云舒，云聚云散，祥云变幻无穷，时而幻化为羊群，时而化为猛虎，时而变为巨人，时而消散成沙粒……
<br/>　　于那山之巅，云雾之中，此时盘坐这一个身穿红色道袍的道人，道人三十来岁的样子，竟然与元始有几分相似。这其实就是元始的执尸，当初得到此界的坐标，执尸元始便一路游历混沌苍茫，历经艰辛，终于来到神秘伏羲所在之地。
<br/>　　初到此界之中，元始并没与引起别人的关注，而是默默地看着这个世界的发展。他发现此界正处于五帝争锋的年代，他并没有多管，对于这些，已经不能撼动元始的心神了。他一直记得自己来此的目的，便是寻得此界元始，或者说是元始的一个“他我”，只有找到他，融合后，他的实力才能突破。
<br/>　　可惜，来到此界已经数千年了，元始并没有发现“他我”的存在，或者说是他来晚了，此界的元始早已死了，他的气息都已经融入到了这片星空之中。
<br/>　　元始虽然没了，却有他的传承道统阐教，而最让人奇怪的是，阐教每一任的掌教都叫元始天尊。
<br/>　　元始找到了此处，自然也就占领了他的地盘，自己也就顺势成了元始天尊。此界中最大的矛盾便是宗教与皇权之间的矛盾，两者纠缠不休，根本没有调和的可能。
<br/>　　元始也不管这些小打小闹，他们爱闹自己闹去，这里实力最强的也就是至尊，相当于洪荒金仙修为，他并没有放在心上。
<br/>　　起初元始是有些失望的，他没有找到自己的目标，但很快就发现了一个有意思的事情。
<br/>　　此界之中，命运的气息太浓了，或者说是构成此界的大道法则中，命运占据绝大部分。这对于他来说，也是一个好消息，因为他可以通过研究此界命运，而学习更多命运法则。
<br/>　　此界中人，随便一个至尊，都可以轻易召唤出命运长河，真是不可思议。要知道，在洪荒之中，只有到了大罗金仙境界，才有资格接触到这些。
<br/>　　对于如此一个奇特的世界，元始自然是不会放过研究的机会。此刻，在元始的眼中，他看到了一般人看不到的场景。
<br/>　　一条长河，浩浩荡荡，贯穿了整片星空，将整片天地都联系起来，又像是一张无形的巨网，将整个世界都包含在里面。
<br/>　　长河看上去是那么的遥远，幽静，深邃，神秘，又带着丝丝玄妙，让人忍不住迷醉在其中。
<br/>　　在那庞大的长河之中，一颗颗如同繁星般的斑点，点缀满了整条大河，星星点点，如同镶嵌在一副画中一般。这些斑点有的璀璨如星光，而有的暗淡如死星，有的却是忽明忽暗……
<br/>　　当然，元始看的并不是这些如繁星般的斑点，他看的是命，是运。
<br/>　　这河名为命运长河，河内的东西，自然不是什么星星了，而是每个人的命。
<br/>　　千千万万，密密麻麻地星点，便是无数人的命，星点在闪烁，那便代表着他在与命抗争，星点暗淡或者熄灭的，便代表着那人即将死去或者已经死去。
<br/>　　一条命运长河，横贯于天地之间，天地内所有人的命都与之牵连在一起，只要是在天地内，他的一生轨迹便会被命记录，过去、现在，未来，一切的一切，逃不脱，挣不断！
<br/>　　河里的人，就如同坠入无边苦海，或者万丈红尘，都在苦苦挣扎，却不知，他们再怎么挣扎，也逃不脱。
<br/>　　修行的本质，便是要跳出这片苦海，走出这万丈红尘！
<br/>　　当然，就算是跳出了这片苦海，可却又要落入另一片更加宽阔的苦海之中，一生挣扎，不断进步。
<br/>　　相传，天地间有这么一条河，它横贯了诸天万界，谁也不知道它从何而来，也不知道它欲前往何处，而诸天万界之中的命运长河，岁月长河等，也只是那条河的一段分支，或者是分支的分支。
<br/>　　一条大河，通过不断分流，贯穿了诸天万界，如同一张大网，将一切都包围起来，众生都难以逃脱它的掌控。
<br/>　　也许，你在此界中超脱了这命运长河，可你却不知，这只是它的一个分支的分支，如这般分支，它有无穷无尽。
<br/>　　有时候自认为超脱了，自己能够跳出棋局，可却丝毫不知，你还在另一个棋局之中，就连你认为的那些下棋者，也不过是别人手中的棋子。
<br/>　　天地就是如此广阔，如此浩瀚，谁也不知道未来会是什么样子，命运长河中的未来投影，也只是其中一种可能罢了。
<br/>　　或许，这就是命，这就是运！
<br/>　　当你自认为看懂它的时候，才发现自己只是知道它的一点皮毛。玄妙，浩瀚，幽深，神秘，无解……
<br/>　　一切皆有可能！
<br/>　　这就是命运。
<br/>　　元始心中忽然升起一种明悟，原来命运并不是想要掌控天下苍生之命，而是给了苍生无限创造的命，命是虚无缥缈的，它需要每个人自己取创造。
<br/>　　原来每个人的命都在自己手中。
<br/>　　逆天改命，改了的命，依旧是你的命，是你自己创造的命，而不是命运给你安排的命，命运从来不会给你安排什么，它是至公无私的，他只是相当于一个记录器，记录下天下苍生的生命轨迹。
<br/>　　命，信则有，不信则无！
<br/>　　“原来，这就是命啊。”
<br/>　　元始喃喃道，他突然站了起来，哈哈一笑道：“我自己就是命。”
<br/>　　明悟之后，并没有给他带来什么战力，他该吃的还是吃，该睡的还是睡，该杀的人依然杀了，这是不是命运在操控，取决于自己。
<br/>　　“围起来，来人，将这片山脉围起来，灭了昆仑！”
<br/>　　“这个世上，宗门是不该存在的，宗门只能为皇权服务。”
<br/>　　“……”
<br/>　　这时，在昆仑山来了好多士兵，这些士兵都有修为在身，他们将昆仑山包围起来，隐隐有想要剿灭昆仑的气势。
<br/>　　高峰之上，元始看着下方的人族军队，笑了笑，他自然是知道此事的原因，西伯候姬昌串联三大宗门，以纣王无道为借口，掀起大战。西伯候想称王，而宗门却又想高高在上，两者一拍即合，便联合在一起，共同对抗纣王。
<br/>　　可惜，经此一役，最后三大宗门损失惨重，商朝败亡，各大宗门也差不多，而得到最大好处的周朝，立刻反目成仇，下令诛杀所有宗门弟子。
<br/>　　大周这一番动作，瞬间便将神州许多宗门毁灭干净，想要一家独大，此刻便是派兵来攻打同样损失惨重的昆仑山。
<br/>　　“你们打闹我不管，但就是别来打扰我。”
<br/>　　元始眼神冷厉，他虽然不是这个世界的人，但好歹也在昆仑山待了几千年了，在他即将要走的时候，还发生这样的事，这就让人恼火了。
<br/>　　对于大周这样背信弃义之人，元始没有什么好感，他一手伸出，变成一只惊天巨掌，狠狠地向着下方拍去。
<br/>　　一掌之下，数万士兵连同西伯候姬昌，全都尸骨无存。
<br/>　　远在大周都城的武王姬发听说此事后，便不再提攻打昆仑山之事，他老子死了，他自然开始继承帝位，并发布消息，将昆仑山列为禁地。
<br/>　　看着这片天地，元始轻声道：“我也该走了，这片天地，终究是被封印了，以我的修为，想要撕开封印，十分容易，但也会让这片天地暴露在星空之中。”
<br/>　　元始可是知道，神话时代，这片天地乃是赫赫有名的洪荒星，仅仅至尊数量便多达数十位，震惊一片星空，但最终引发其他大教的忌惮，且在域外大战中，背后偷袭，致使洪荒星落寞，最后更是逼得道尊等人身化大阵，封印洪荒星，这才躲过了一劫。如今洪荒的敌人还在外面虎视眈眈，若是暴露出来，洪荒星的未来可就堪忧了。
<br/>　　“也罢，好歹也沾上洪荒二字，我便留下点东西，给后来人一份机缘。未来之事，自由未来之人来完成，我便不插手了。”
<br/>　　说着，元始取出两片布帛，上面分别写着《雷帝法》、《三生经》，之后他将之扔了出去，布帛没入虚空消失不见，至于今后会被谁得到，那就要看机缘了。
<br/>　　……
