　　想了半天苏重不由摇头叹息，在体内布置诛仙剑阵，想法虽好可不太现实。
<br/>　　他根本不知道诛仙剑阵，而且要把这种巨型阵法建立在体内，需要庞大改造工作，苏重短时间内也没那能力。
<br/>　　“还是尝试《庚金剑气》内的阵法比较靠谱。”
<br/>　　以前他忙着凝聚剑气符，一直和凶戾气较劲。对《庚金剑气》内剑气成阵概念，只是一扫而过。
<br/>　　就连《庚金剑气》创造者，也只提出一个大概猜想和笼统模型，苏重并未在意。
<br/>　　现在剑气符凝聚不成问题，如何进一步加强剑气威力被提上日程。
<br/>　　“按照《庚金剑气》说法，要么让剑气分化布置成阵，引动天地力量加强威力。要么就借助阵法把剑气符凝聚为一，增加剑气符锋锐。”到底该选择哪个？
<br/>　　实际上选择哪个的工作量都不小，都需要他自己去尝试改进。
<br/>　　“剑气分化在于一剑生万法，用剑气符布置阵法，更像一种法术。剑气聚合提高剑气锋锐，走的是一剑破万法的路子，这才有点儿像剑仙本色。”苏重很快有了想法。
<br/>　　想一剑生万法，他不仅要改进剑气符，还要掌握庞大阵法知识。两者融合才能发挥剑气威力。
<br/>　　后者却只需他提高剑气符威力，集中改进融合剑气这一种阵法，所耗时间精力更少，见效更快。
<br/>　　“张小凡都有了标配烧火棍，时间不等人啊！”苏重决定聚合剑气，提高剑气威力。
<br/>　　一旦拿定主意，苏重就全身心投入其中，时间飞速流逝，直到张小凡再次上门。
<br/>　　和前段时间沉默寡言相比，此刻苏重面前的张小凡少见的露出开心神色。
<br/>　　“二蛋，快跟我走。惊羽来看我们啦！”张小凡拉着苏重就跑。
<br/>　　苏重也不反抗，跟在张小凡身后若有所思：“林惊羽来大竹峰，那说明七脉会武将要开始。”
<br/>　　张小凡要开始自己的旅程。苏重不由自主把目光看向对方。一看之下颇为惊讶，张小凡奔跑速度竟不慢。拉着自己的力量也不小。
<br/>　　相比于之前那个瘦弱山村少年，现在的张小凡力大无穷，即便不用轻功，速度也极快。聚元功已颇具火候。
<br/>　　看来这段时间没少往黑竹阵法里面钻，不然聚元功不会修炼如此程度。
<br/>　　“小凡，慢点儿跑，我教你点儿奔跑腾挪的法门。”苏重开口道。
<br/>　　然后苏重一边跑一边讲解一些运用肌肉力量小技巧。张小凡不断尝试，没多久就完全上手。
<br/>　　“二蛋，你真天才，竟还会这种法门！”张小凡惊喜。
<br/>　　天才？与其说我是天才，能够这么快上手运用，眼前的木讷少年同样也是个天才啊！
<br/>　　如果不是天才，如何能够同修太极玄清道和大梵般若，最后更融合五卷天书，可从容运用诛仙剑！
<br/>　　苏重跟着张小凡冲进大竹峰朱峰大殿，就看到一众师兄都在大殿内。还有两个新面孔，一个是林惊羽，一个大概就是龙首峰大师兄齐昊。
<br/>　　“难得你们三个这么长时间见一次面，出去玩吧，不用陪着我们。”师娘苏茹笑吟吟开口道。
<br/>　　张小凡和林惊羽立刻开心点头，然后冲出大殿。苏重无奈摇头，两个小屁孩，能有什么玩的。苏重行礼告辞，也跟着出来。
<br/>　　这一会儿功夫，张小凡林惊羽两人已经巴拉巴拉的聊上了。苏重走到两人身边，只听不说。心里却在不住思索剑气聚合的事。
<br/>　　两个人聊着聊着，很快就说到各自修为。
<br/>　　林惊羽自小就聪明伶俐，进入龙首峰后，因资质出众，很快得到苍松青睐，带在身边细心教导。
<br/>　　如今已经到达御物境界，更被天赐下九天神兵斩龙剑。
<br/>　　在龙首峰别说新晋弟子，就连一些修炼多年弟子，林惊羽都能战而胜之，心里傲气的很。
<br/>　　“咱们来切磋一下怎么样。”林惊羽开口道。他在草庙村就争强好胜，所有小伙伴都以他为首。唯独张小凡这头倔驴不服气。现在他修炼有成，又起了压过一头想法。
<br/>　　“好啊！”张小凡没心没肺，答应的那叫一个痛快。
<br/>　　苏重也不阻拦。两个刚接触修炼没多久的小子，能打出个什么场面，即便两人都能于是武器，苏重也不在意，在他眼里，就是两只菜鸡互啄！
<br/>　　“小凡，小心了，我来啦！”林惊羽自信满满，他天赋卓绝，修炼太极玄清道有成。身体素质强悍。在龙首峰，所有新晋弟子，在拳脚方面每一个人是他对手。
<br/>　　因为他速度快反应灵敏。别人还没反应过来，他就依仗强大身体素质战胜别人。
<br/>　　现在他也打算如此施展。嘴角露出一抹坏笑，林惊羽猛然窜出，刚才还在十米外，下一刻几乎就和张小凡脸对着脸。
<br/>　　这一招他在龙首峰屡试不爽，不少新晋弟子都被他这招吓的一屁股坐在地上。他几乎都能想先张小凡惊慌失措的模样，脸上不自禁露出笑容。
<br/>　　可不等他脸上笑容扩散，一股巨力都骤然从腹部传来。
<br/>　　只见张小凡扭腰转胯，几乎下意识的就一拳轰出！正式被他练的纯熟的聚元功招法。
<br/>　　他此刻力量何其大，林惊羽根本就没反应过来，就被一拳轰出去数米。
<br/>　　看着捂着肚子弓成虾米状的林惊羽，苏重忍不住笑出声来。
<br/>　　这小子想仗着身体素质欺负人，哪里想到张小凡有自己的聚元功，还有黑竹林辅助。
<br/>　　单论身体素质，整个青云门同龄人，就没几个能和张小凡比！
<br/>　　“不对，刚才是我不小心，再来！”林惊羽从地上爬起，再次冲向张小凡。
<br/>　　砰！
<br/>　　一个乌青眼眶印在林惊羽白皙脸庞上。
<br/>　　“不可能！你怎么可能比我快！”林惊羽满心的不甘。
<br/>　　他在龙首峰可是大变同龄人无敌手，可遇到老乡，一个照面就被放倒，他怎会甘心。
<br/>　　“小凡，看来你的玄清道造诣极高。咱们来比一比御剑斗法吧！”林惊羽眼珠子一转。身体素质不行就比御剑呀。他可是有斩龙剑在手的。
<br/>　　不等张小凡回答，林惊羽直接飞剑出鞘朝张小凡攻来。
<br/>　　他看出来了，张小凡身上可没有飞剑。他也不打算伤害张小凡，但剑气横空而来，被吹个跟头不是很正常的吗？自己还被打了个黑眼圈呢。
<br/>　　苏重面色古怪的，这家伙是要耍赖啊！打不过就用法宝作弊。真要让他把张小凡掀翻在地，田不易的脸往哪里放？
<br/>　　苍松打的什么主意，苏重一看知道。林惊羽天赋奇才，这才几年功夫就能御剑。这次齐昊去各峰通知七脉会武消息，别人谁都不带只带林惊羽，可不就是为了显摆吗？
<br/>　　“好了，到此为止吧。切磋而已，动什么兵器。”苏重上前一步阻止。田不易对他不错，他当然不能让田不易丢脸。
<br/>　　林惊羽丝毫不为所动，看苏重站在张小凡身边。斩龙剑上威力更盛，显然是打算把两人一起掀翻。
<br/>　　苏重暗自摇头，果然还是那个争强好胜的林惊羽。苏重抬手就是两道剑气，一道直击斩龙剑，一道飞向林惊羽。
<br/>　　叮的一声响！斩龙剑直接被剑气击飞。另一道剑气去势不减，直飞林惊羽而去。苏重不打算打伤对方，但教训还是有的。
<br/>　　剑气飞到半途，猛然一下炸开。一股气浪裹挟灰尘轰的一下就飞了过去。
<br/>　　苏重正打算欣赏林惊羽灰头土脸模样，就感觉一股凌厉之气从背后袭来。
<br/>　　想都没想，一道剑气骤然飞出。
<br/>　　回头一看，正是满脸心急的齐昊。
<br/>　　这次带着林惊羽出门显摆，他有保驾护航的责任，要是被人家伤到，他回去可不好交代。
<br/>　　他刚从大殿出来，就看苏重剑气横空。锋锐斩龙剑被剑气一下斩飞，另一道剑气直奔林惊羽。心急之下直接出手，来了个围魏救赵。
