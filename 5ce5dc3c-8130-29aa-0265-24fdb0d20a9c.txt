　　咚！咚！咚！
<br/>　　叶尘一拳打爆了神界大祭司，又缓步朝下方的神王路西法走去。
<br/>　　“叶狂仙！今日是我败了，只要你放过我，我保证，今后神界再也不会与你为敌，我们相安无事，就当什么都没有发生过，如何？”
<br/>　　路西法身受重伤，自知无论如何也逃不掉，而且他的手下，也几乎全军覆没，语气顿时没有了之前的强硬，开始主动求和。
<br/>　　叶尘冷冷一笑，
<br/>　　“就当什么事情都没有发生过？你想的可真简单呐！”
<br/>　　说话间，叶尘身体一晃，下一刻已经来到路西法的头顶上空，紫琼剑高高举起，狠狠斩落！
<br/>　　路西法瞳孔一缩，当即脚下猛然一蹬，迅速暴退而去，可惜那道剑芒却如影随从的追了上来，令他只好再次举起手中权杖，抵挡了上去。
<br/>　　轰隆！
<br/>　　天地再次巨震！
<br/>　　刺啦啦！
<br/>　　路西法狂喷了一口鲜血，再次倒飞而出，在地上留下了一道数十米宽，近万米长的深坑，所过之处，山峦俊峰全部崩塌，诺大的华山，惨不忍睹。
<br/>　　轰！
<br/>　　还没等路西法反应过来，忽然一道人影从天而降，叶尘已经再次追杀至跟前，一脚踏在了神王路西法的胸膛之上。
<br/>　　“噗！”
<br/>　　路西法顿时狂喷了一口鲜血，脸上终于露出了惊恐之色，
<br/>　　“别！别杀我！只要你不杀我，今后你就是神界之主，我神界亿万生灵，都由你主宰，如何？
<br/>　　叶尘咧嘴一笑，脸上挂着一抹嘲弄之色，
<br/>　　“区区一个化外空间，也配称神界？本帝要你们这群废物又有何用？”
<br/>　　说完这话，在路西法无比惊恐的眼神中，叶尘猛然一脚踏落！
<br/>　　“不！！”
<br/>　　“彭！”
<br/>　　可怜堂堂神界神王，竟然被叶尘一脚活活踩爆！
<br/>　　“啊！！叶狂仙！我要诅咒你，诅咒你不得好死！！”
<br/>　　一道凄厉的嘶吼声响彻天地。
<br/>　　却是路西法在肉身被灭的那一刹那，元神自肉身中遁出，已经逃窜到了九天之上。
<br/>　　“哼！还想跑？给我，回来！”
<br/>　　叶尘暴喝一声，虚手一抓，
<br/>　　“啊！！”
<br/>　　又是一道惨叫，路西法连化神境都没有达到，元神力量又能强到哪里去？瞬间便被叶尘擒住，直接从九天之上拉扯了回来。
<br/>　　然而就在这时，天空上忽然出现了一道白衣老者，一把将路西法的元神截获了下来，
<br/>　　“人都已经死了，区区一道残魂，阁下又何必赶尽杀绝呢？”
<br/>　　这出手之人，赫然正是仙界第一强者，无涯老人。
<br/>　　唰！唰！唰！
<br/>　　随着无涯老人赶到，其余仙界众强者，也纷纷赶了过来，将叶尘围在了中央。
<br/>　　“无涯道兄，你可算是来了！快！快杀了这小子！以后我神界，愿以仙界马首是瞻！”
<br/>　　路西法的那道残魂躲在无涯老人的身后，开始大声疾呼起来。
<br/>　　他原本想要向其他三界立威的，却万万没有想到，自己费尽心机，竟然反被叶尘团灭，如今无奈之下，也只好向仙界求助了。
<br/>　　叶尘听到这话，顿时两眼微微一眯，
<br/>　　“原来你就是仙界中，天榜排名第一的无涯老人，怎么？你也来找死？”
<br/>　　“放肆！竟然敢对无涯前辈无礼！”
<br/>　　叶尘这话刚刚落下，无涯老人身后，一名仙界强者顿时便跳了出来，指着叶尘大声训斥起来。
<br/>　　叶尘连看都没看那人一眼，直接随手一挥，
<br/>　　“聒噪！”
<br/>　　唰！
<br/>　　一道诡异莫测的紫色剑芒，瞬间便杀到那名仙界强者跟前。
<br/>　　“啊！”
<br/>　　那人顿时瞳孔一缩，想要躲避显然已经来不及了。
<br/>　　关键时刻，那无涯老人再次出手，只见他一掌拍出，
<br/>　　“乾坤挪移！”
<br/>　　说也奇怪，眼看叶尘那一剑，就要斩落在那仙界强者的身上，竟然发生了细微的偏差，就好像空间扭曲了一般，这快如迅雷的一剑，竟然落空了！
<br/>　　“嗯？”
<br/>　　就连叶尘，也不由得眉头微微一皱，
<br/>　　“好诡异的秘术！恐怕至少也是地级功法吧？”
<br/>　　地级功法！
<br/>　　即便是在修真界之中，也属于中上等。
<br/>　　虽然叶尘的脑袋之中，天级功法乃至神级功法都有一些，可是以他如今的实力修为，能够施展的并不多。
<br/>　　仅仅从这一点来看，这无涯老人的实力，比神王路西法就强的不止一点。
<br/>　　“呵呵，叶道友动不动就取人性命，未免有些太霸道……”
<br/>　　无涯老人出手救下了自己的属下，正暗自得意，然而还没等他的话说完，却见叶尘已经再次遥空一拳，朝路西法的残魂砸去！
<br/>　　“轰！”
<br/>　　路西法只有一道残魂，如何挡得住叶尘这一拳？
<br/>　　“尔敢！”
<br/>　　就连无涯老人一时间也没有反应过来，等到他想要出手搭救，已然来不及了。
<br/>　　“彭！”
<br/>　　路西法的元神，直接被叶尘一拳打爆，这位神界之主，彻底消亡于世间。
<br/>　　无涯老人顿时气得勃然大怒，
<br/>　　他救下路西法的元神，就是希望日后，利用此人来控制神界，没想到却被叶尘彻底坏了好事，这让他如何不怒？
<br/>　　“小子！你找死！”
<br/>　　暴怒之下的无涯老人，顿时彻底撕下了伪善面孔。
<br/>　　呼！
<br/>　　无涯老人长袖一挥，
<br/>　　嗖嗖嗖！
<br/>　　自他那宽大的长袖中，竟然飞射出上百柄灵器飞剑，在空中形成了一个小型剑阵！
<br/>　　“小子！你莫要以为杀了路西法，就真的天下无敌了，来领教一下老夫的手段吧！”
<br/>　　“喝！”
<br/>　　随着无涯老人一声暴喝，漫天的剑雨朝叶尘爆射而来，而且隐隐暗合阴阳五行之道。
<br/>　　“不愧是仙界第一强者，竟然比路西法还要强大几分！”
<br/>　　连叶尘也不由得暗暗感叹了一声，不过却并没有放在心上。
<br/>　　“逆生百转，四转！五转！”
<br/>　　他如今的状态，并不能持续太久，所以必须速战速决！
<br/>　　轰隆！
<br/>　　转眼之间，叶尘身上的气息，再次成倍暴涨！
<br/>　　“什么！这家伙，到底是什么怪物？”
<br/>　　无涯老人顿时顿时两眼一瞪，简直都要怀疑人生了。
<br/>　　他本以为，叶尘施展了某种秘术，他只要支撑一时半刻，就算是耗也能耗赢。
<br/>　　却万万想不到，叶尘的实力居然还能再次暴涨！
<br/>　　哗！
<br/>　　将逆生百转施展到第五转之后的叶尘，与无涯老人的实力再次被大大拉开。
<br/>　　紫琼剑斩落之后，顿时以摧枯拉朽之势，将那上百把灵器飞剑尽数斩飞，而无涯老人也直接喷出一口鲜血，倒飞了出去。
<br/>　　所有人顿时全都傻眼了，
<br/>　　任谁也没有想到，无涯老人竟然连一招都没有撑过，就直接败下阵来！
